　　第732章
<br/>　　一石激起千层浪，程大雷走出凉州的消息以迅雷不及掩耳之势传遍京州。沿途多少势力磨刀霍霍，不可能让程大雷活着走到长安城。
<br/>　　这一幕程大雷事先未必没有想到，在茶楼上他暴露身份，这一切便注定会发生。
<br/>　　不过目前的当务之急，还是如何从汝南城脱身。魏牧冲所思所想如何，的确有些难以琢磨。
<br/>　　名叫乔弄云的女孩此时已经醒转，崔白玉守在她床边。小心翼翼睁开眼睛，似乎仍不敢相信眼前看到的一幕。愣了很久，才试探着唤了一声：“姐姐，是你么？”
<br/>　　崔白玉用手揉了揉她脑袋，这样亲昵的动作令乔弄云有些不适应。在她印象中，自己这位表姐实在不是个好亲近的人。
<br/>　　当初崔白玉有崔白玉的架子，放眼整个崔家，她说话还是做几分数的，而乔弄云不过是个微不足道的外戚。不过，如今不是从前，崔家已经倒了，两个无主之魂也只能抱团取暖。
<br/>　　程大雷大脑袋凑过来：“人没事吧？”
<br/>　　乔弄云吓了一跳，险些又昏过去，她好不容易才稳住砰砰乱跳的心脏。仔仔细细将程大雷打量一遍，看得程大雷毛骨悚然，怀疑自己是不是脸上挂着饭粒。
<br/>　　在世人眼里，程大雷是极恶之人。说来说去，崔家这次倒台，还和程大雷有些关系。如果不是他取宋伯康而代之，将整个凉州握在手中。李乐天投鼠忌器，也未必敢向相府动手。
<br/>　　如果他不曾动手，那如今的帝国怕是另一番田地了。
<br/>　　即便在乔弄云这样小人物眼里，也是高高在上的千金小姐，锦衣玉食，身边仆役成云。但没有如果这回事，现在的她命若幽草，捻一捻就断？
<br/>　　崔白玉冷着一张脸：“吃了许多苦，好在是没有伤及根本。”
<br/>　　“啧，我说什么来着，救下来肯定是一场麻烦。先说好，你有什么爹娘姊妹之类的，我可不管插手，趁早别提。”
<br/>　　乔弄云吓得不敢吭声，崔白玉依旧板着一张脸。等程大雷走出屋子，乔弄云才小心翼翼道：“姐姐，苦了你了。”
<br/>　　崔白玉摇摇头：“好好歇息，我们走的时候带你上路，他说得也未必没有道理，舅舅舅母我们的确管不了。”
<br/>　　崔家死了很多人，崔家还要死更多人。这些人或多或少和崔白玉有些关系，如果她真要一个个救，就是公然和李乐天叫板。幸好，崔白玉本身是个性情冷漠的人，也并非菩萨心肠。
<br/>　　乔弄云轻轻叹口气，有些话她想说不敢说。崔白玉落到程大雷手中的事情，她已经知道。在她想来，表姐已经以身侍贼才换来活命。他肯答应救自己，也定是表姐答应了他极其无耻的条件。
<br/>　　如崔白玉这样骄傲的人，已经做到这个地步，自己怎么能奢望其他呢。
<br/>　　“姐姐，我们要怎么逃？”乔弄云问道。
<br/>　　“逃？”崔白玉一顿，然后摇摇头。天下很大，受过相府恩惠的的确不少，可若说此刻谁肯收留自己？偏偏也就是个程大雷。莫说逃不脱，即便走得脱，落入其他诸侯手中，以自己的姿色，怕也是匹夫无罪，怀璧其罪。
<br/>　　至少，程大雷没有做得太过分。
<br/>　　大厦倾塌，崔家又能活几个。如乔弄云的父母，当初仗着相府的势力作威作福，如今自然也逃不过一个死字。就算是乔弄云自己，那种长街驱马，压榨平民的事恐怕也没少做。
<br/>　　所以她和那死去的魏明一样，有此番经历，都不无辜。
<br/>　　程大雷走出屋子，屋外繁星满天，倒是一个好天气。
<br/>　　刘发财站在屋檐下，朝对面屋顶上指了指，程大雷远远看到几个黑影趴在上面。
<br/>　　“得，咱们被监视起来了。”程大雷。
<br/>　　“要不给他们点颜色看看？”刘发财。
<br/>　　程大雷摇摇头：“随他们去吧。”
<br/>　　这自然是魏牧冲派来，免得程大雷悄悄溜走。不过他们只是暗中看守，程大雷也不好扯破脸皮，他们愿意守就让他们守着吧。
<br/>　　几人回到屋内，将门关了，程大雷在椅子上坐了，道：“聊聊吧，看怎么从此地脱身？”
<br/>　　“魏牧冲这个人，不好琢磨呀。”徐神机叹了一声。
<br/>　　程大雷点点头，最棘手的还是魏牧冲。这个人连自己儿子都敢杀，还有什么事情是他做不出来的。如果脑子一抽风，半夜杀进来也不是没可能的事情。
<br/>　　“喏，事情是因为你惹出来的，你想个法子吧？”程大雷看向崔白玉。
<br/>　　崔白玉皱眉思索片刻，道：“这座宅子曾经是崔家的宅子，魏牧冲刚接手不久，看他的性子，也未必了解过宅子的布局。我却是知道，这宅子修筑时有条暗道，通向城外，是为了破城时留一条后路。”
<br/>　　程大雷白了她一眼：“行不通，就算我们可以摸黑离开，那我的座骑怎么办，总不能死在这里。”
<br/>　　“那不过是一头牛，难道比性命还要紧。”崔白玉说了一半，道：“那你有什么法子？”
<br/>　　程大雷皱眉想了想，道：“明天出城，我倒要看看，魏牧冲究竟敢不敢冲我下手。”
<br/>　　魏牧冲的脾气的确不容易摸透，他是个将信仰看得比一切都重要的人。而程大雷唯一可以确定一件事，那就是李乐天并不希望自己死。
<br/>　　李乐天刚刚掌权，他现在需要的是和平，有足够的时间消化到手的权力。而程大雷一旦出事，蛤蟆兵大军攻来，胜负不好说，乱是一定的。
<br/>　　也就是确认这一点，程大雷才敢出凉州上长安。曾经二人是敌人，但现在却是盟友，而自己曾经也是相府的敌人，如今却收留了两个相府的余孽。
<br/>　　造化弄人，大概如此。
<br/>　　程大雷估摸着魏牧冲未必敢向自己下手，否则不会等到现在。而真正的困难其实也不在汝南城，而在接下来的路途。
<br/>　　再如何说，魏牧冲也是把李乐天当会事的，可这条路上多的是不把李乐天当会事的存在。
<br/>　　他们心里是巴不得长安兵荒马乱，越乱越好。
<br/>(本章完)
