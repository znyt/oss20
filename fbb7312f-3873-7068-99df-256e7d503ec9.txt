　　滚滚熔浆中，一道剑气禁制，漂浮于表面。
<br/>　　禁制内，一团金色火焰，诡异地跳动着。
<br/>　　正是地脉金火。
<br/>　　萧逸先是一喜，随后，又眉头一皱。
<br/>　　他发现，自己似乎无法奈何这地脉金火。
<br/>　　当年，裂天剑魔欲要擒它，它极力反抗。
<br/>　　无数岁月下，它游走于地脉岩浆中。
<br/>　　早已拥有了操控地脉岩浆的本事。
<br/>　　在它的调动下，整个巨大火山口爆发。
<br/>　　连同地底深处的地脉岩浆，也被它悉数调动而来。
<br/>　　那一瞬间爆发的威力，连裂天剑魔都拿它没辙。
<br/>　　甚至于，火焰圣山方圆千里之内，一瞬间化为焦土。
<br/>　　火焰圣山如今被包裹的熊熊火焰，就是当年留下的。
<br/>　　当年此事，可是震惊整个火焰郡。
<br/>　　不过，据历史记载，当年所有人都只以为是火山忽然爆发。
<br/>　　并不知道是裂天剑魔在擒地脉金火。
<br/>　　现今，火焰圣教赖以崛起的地火源脉，也不过是当年遗留的火焰而成。
<br/>　　总之，当年连裂天剑魔也无法奈何得了它。
<br/>　　不过，原因，并非是裂天剑魔实力不足。
<br/>　　而是，他那时已经处于突破边缘了，需要尽快进入闭生死关的状态。
<br/>　　故，他的时间不多。
<br/>　　无法在短时间内彻底擒拿下它，又无法修炼成九阳轮回诀。
<br/>　　这才在闭生死关中，宣告失败，并反噬而死。
<br/>　　萧逸至今也没明白那生死关是什么，裂天剑魔，也没有留下信息。
<br/>　　言归正传。
<br/>　　萧逸看了几眼，感知了一番，已经确定自己奈何不了地脉金火。
<br/>　　只得无奈地摇摇头，准备离开。
<br/>　　他来找地脉金火，更多的，是想看看裂天剑魔当年的禁制如今还是否存在。
<br/>　　又能困住地脉金火多久。
<br/>　　现在看来，自己还有数年时间。
<br/>　　谨慎些，算它两年时间。
<br/>　　而两年，已经足够发生许多事。自己的修为，必将达到另一个高度。
<br/>　　届时，再来收服这地脉金火，便把握极大了。
<br/>　　想罢，萧逸转过身。
<br/>　　但，恰在此时，体内却一阵躁动。
<br/>　　冰鸾剑，忽然自动凝聚而出。
<br/>　　剑身上，陡然散发骇人寒冷，竟是盖过了周遭的炎热气息。
<br/>　　下一秒，冰鸾剑毫无预兆地带着萧逸，往那熔浆而去。
<br/>　　“这…”萧逸一惊，脸色顷刻煞白。
<br/>　　地脉金火，可不是善茬。
<br/>　　自己现在这个实力去惹它，岂不是找死？
<br/>　　“给我停下。”萧逸大喝一声。
<br/>　　然而，冰鸾剑不仅没有停下，反而加快了速度。
<br/>　　剑柄上，有一股强大的吸力，紧紧吸附着萧逸的双手。
<br/>　　嗤的一声。
<br/>　　冰鸾剑带着萧逸，转瞬间便来到剑气禁锢之前。
<br/>　　果不其然。
<br/>　　里头的地脉金火，瞬间暴躁起来。
<br/>　　它拥有着微弱的灵智，立刻对靠近它的萧逸产生了敌意。
<br/>　　霎那间，整个火山口暴走。
<br/>　　岩浆，化作了千丈浪潮。
<br/>　　“嘶。”萧逸倒吸一口凉气，连忙凝聚出漫天紫炎，护于自己身前。
<br/>　　嘭…嘭…嘭…
<br/>　　同一时间，周遭又是数个千丈大小的岩浆浪潮出现。
<br/>　　一切，都发生得极快。
<br/>　　地脉金火，果然厉害。
<br/>　　竟在如此短的时间内，就发动了这般骇人的攻击。
<br/>　　萧逸的心情，瞬间跌到了谷底。
<br/>　　若是这数个千丈岩浆浪潮，悉数压向自己。
<br/>　　自己必将葬身火山之内，而且还是尸骨无存那种。
<br/>　　他的紫炎，自然能烧毁岩浆。
<br/>　　但，绝无法在一瞬间将这所有岩浆烧毁。
<br/>　　“完了。”这是萧逸的第一个念头。
<br/>　　但，他没想到。
<br/>　　地脉金火，完成这一系列攻击，耗时虽极短。
<br/>　　现场，却有一物，比它更快。
<br/>　　正是手中冰鸾剑。
<br/>　　一抹蓝色幻影，一闪而过。
<br/>　　速度快得，如若能跨破长空，甚至是无视空间限制。
<br/>　　当蓝色幻影再次出现时。
<br/>　　冰鸾剑，已刺穿了剑气禁制。
<br/>　　剑尖，直直刺在地脉金火上。
<br/>　　冰鸾剑，无坚不摧的特性。
<br/>　　萧逸还从未见过有什么东西能抵挡它的穿刺。
<br/>　　地脉金火，顿时一阵剧烈抖动。
<br/>　　但仅仅是跳动了几下后，竟是再无动弹。
<br/>　　而后，冰鸾剑上，出现了一股吞噬的力量。
<br/>　　几乎是肉眼可见的速度，地脉金火，在逐渐被吸摄。
<br/>　　当年，裂天剑魔之所以无法奈何它。
<br/>　　全因它能瞬间调动地脉岩浆。
<br/>　　若是单单它自己的力量，怕是裂天剑魔一巴掌就能收了它。
<br/>　　如今，冰鸾剑那仿佛能跨越空间，无视时间的一斩。
<br/>　　直接在它彻底调动地脉岩浆力量前，就将它刺中，将它压制住。
<br/>　　它便成了纸老虎。
<br/>　　地脉金火，本质，不过是一团火焰。
<br/>　　不过半晌。
<br/>　　整团地脉金火，被冰鸾剑吸收殆尽。
<br/>　　萧逸内视一番。
<br/>　　果然，地脉金火被冰鸾剑吞噬后，最终进了控火兽的嘴中。
<br/>　　控火兽一口吞下。
<br/>　　这时，萧逸明显感觉到，自己又掌控了一种火焰，正是地脉金火。
<br/>　　同时，一股澎湃的力量，正快速在气泉中充盈。
<br/>　　那是地脉金火本身的力量。
<br/>　　力量，通过冰火金丹，化作了源源不断的冰火元力。
<br/>　　其数量之庞大，足足填满了一成气泉。
<br/>　　也就是说，现在的气泉，被填满了三成。
<br/>　　萧逸的修为，已达地元三重。
<br/>　　以前，在得到紫晶灵炎，以及十界灭生火时，并没有修为上的提升。
<br/>　　那是因为，紫晶灵炎，只是自己需收了一滴紫炎狮王的妖兽精血。
<br/>　　冰鸾剑，强行掠夺了这种天赋。
<br/>　　吸收十界灭生火时，那也只是十界灭生阵残留下来的一丝火焰。
<br/>　　强行被冰鸾剑掠夺了过来。
<br/>　　这两者，本身并不存在多大的力量，只存在那一种能力。
<br/>　　而现在，冰鸾剑吸收的，可是地脉金火本身。
<br/>　　能力与力量，并存。
<br/>　　地脉金火，本身就蕴含着巨大的力量。
<br/>　　冰鸾剑吞噬了它，自然也把它蕴含的力量给掠夺了。
<br/>　　这才让得萧逸如今有修为上的突破。
<br/>　　言归正传。
<br/>　　“去。”萧逸低喝一声。
<br/>　　数个千丈熔浆浪潮，瞬间退去，缓缓落回火山口。
<br/>　　萧逸惊喜一笑。
<br/>　　随后化出紫炎火翼，一个飞跃，回到了火山口之上。
<br/>　　嘭的一声。
<br/>　　萧逸手中，猛然出现了一股金色火焰。
<br/>　　火焰，散发着暴虐气息，极为狂躁。
<br/>　　如果说紫晶灵炎，是焚毁万物的强大。
<br/>　　十界灭生火，是焚烧生机的诡异。
<br/>　　那么，地脉金火，便是狂猛霸道的爆发。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸的身影，飙升到极致。
<br/>　　不多时，回到了禁制之中。
<br/>　　而后，拿出九阳轮回决，细细参悟起来。
<br/>　　九阳轮回诀，虽玄奥，但以萧逸的资质，要悟透并不难。
<br/>　　真正难的，在于炼成，在于得到九种世间最强悍的火焰。
<br/>　　萧逸现在要做的，便是将三种火焰，融合起来。
