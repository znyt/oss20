        &lt;h3&gt;第三卷第一百一十二章  心燃烧起来了（为小k只爱你加更）&lt;/h3&gt;
<br/>
<br/>        宁雅没有问胡婷是怎么解决的凌辰，她陪着胡婷聊了一晚上，才知道胡婷的老公和凌辰经常聊天，之前几个男同学聚了次，大概是同学之间玩闹怕了照片，胡婷就觉得事情大了，闹着问他们是怎么回事。
<br/>
<br/>        她老公认为她是无理取闹，俩个人便发生了争执，俩个人从来都是恩恩爱爱的，很少吵架，这一吵架胡婷就认为是天塌下来了，来她这里寻求庇护了。
<br/>
<br/>        宁雅给胡婷分析了情况，同学们在一起闹着玩的，哄得胡婷高高兴兴地的和她老公回家。
<br/>
<br/>        宁雅又投入到紧张的工作中，他们公司经过融资，股份也已经发生变化，她平时负责的东西就是很杂，很多，现在就更忙了，每天更在陈子昂的身后忙得停不下来。
<br/>
<br/>        而且公司又进行重组，她要配合人事上做很大的调整。
<br/>
<br/>        每次回到家都很晚了，也累的不想动，偶尔她也会想到凌辰为什么好几天了都没有给她发信息，但是看看时间已经很晚了，也就没有给凌辰发过信息。
<br/>
<br/>        忙碌的生活很充实，也会让人错过有些东西。
<br/>
<br/>        陈子昂在回家之前将她和凤盛夏叫到了办公室，对她们说：“我得回老家一趟，这几天公司的事情还比较重要，周一从外地调回来的人就要各就各位，你们负责将前期工作做好，让每个人来北京之后速度进入工作状态。”
<br/>
<br/>        “宁雅你这个办公室主任就更要辛苦一些了，诺，这些文件都要进行整理，然后上传oa，提醒各位股东及时签字。”
<br/>
<br/>        “好的。”宁雅看着厚厚的资料，他们办公室只有三个人。
<br/>
<br/>        “这些都要生产电子版，进行存档，一定要注意。”陈子昂又叮咛宁雅“周一必须整理完。”
<br/>
<br/>        “周一之前？”
<br/>
<br/>        “是的！有什么问题吗？”陈子昂问宁雅
<br/>
<br/>        “没有问题。”宁雅在心里说：“你是老板，你说了算。虎落平阳被犬欺。”
<br/>
<br/>        陈子昂看着宁雅笑着说：“我不欺负人，这个是你们的工作，既然干了这份工作就一定要接受这份工作带来的殊荣。”陈子昂特意将“殊荣”两个字咬的特别重。
<br/>
<br/>        “我也没有欺负你这只虎妹，你这只虎妹只要想冲出牢笼，我随时恭送，不但恭送，还会十里红妆。怎么样？”陈子昂似乎能看透宁雅的心。
<br/>
<br/>        “我就喜欢在你的这个笼子里呆着，咋地？不服气啊，不服气你咬我啊？”宁雅也笑着说。
<br/>
<br/>        “我又不是小狗，动不动就咬人。”陈子昂笑着说。
<br/>
<br/>        “小雅，谢谢你啊，在公司危机的时候替公司着想。也替我谢谢你老同学。”
<br/>
<br/>        “说那么客气的话干什么，我也是公司里的一员呢。”宁雅微笑着说。
<br/>
<br/>        “说真的，你难道想在我公司干一辈子？”陈子昂问宁雅，按照宁雅的能力，她完全可以自己去做一番事业。
<br/>
<br/>        她有很多的资源，有聪明的头脑，讲真，窝在她公司当一个办公室主任太屈才了。
<br/>
<br/>        “我不在你公司干，我在哪里干啊。”宁雅这两年已经很喜欢陈子昂公司的氛围了，每一个企业都有自己的精髓所在，都有自己的企业文化和文化理念。
<br/>
<br/>        陈子昂的公司凝聚着人们的热情还有一种亲情，对，没错，是亲情，那种浓浓的帮助相互协作的充满着温馨的感觉真好。
<br/>
<br/>        “你可以自己干啊，凭着你的能力你干什么都可以啊。”陈子昂笑着看着她。
<br/>
<br/>        “你这是赶我走呢。我偏不走。”宁雅恨恨的说。
<br/>
<br/>        “小雅，我说的是真的，你本来是宁家的女儿，家世很好，资源也很广，趁着自己现在还能干，为什么不自己闯荡一番呢？你好好想一想，如果你想继续在我公司干下去，我给你一个适合你的职位，如果你还想自己做，你随时提出来，如果需要帮助的话我也是可以出一份力的。”陈子昂衷心的说。
<br/>
<br/>        宁雅看着陈子昂那真诚美丽的丹凤眼，心里充满着感动，陈子昂是真的为她着想。
<br/>
<br/>        陈子昂的话在宁雅的心里激起了涟漪。
<br/>
<br/>        陈子昂走后，宁雅也开始忙碌起来了，她不知道凌辰给她打电话的时候，胡婷接了电话，她也不知道凌辰接电话时心里发生了什么样的动荡和改变。
<br/>
<br/>        她更不知道因为那个电话，凌辰睡了一天一夜，因祸得福，做了很多年的总代理。
<br/>
<br/>        翻开了岁月发黄的书签，留下点点墨的恋，爱情总是在转瞬将我们抛远，雪的白撒满冬天，没有你的日子不再温暖。
<br/>
<br/>        万年青一如既往地的不变，胭脂红修饰着苍白的颜，描的眉缺少生动的弯弯，我和你隔着地隔着天，隔着触摸不到的遥远。
<br/>
<br/>        曾经的习惯消失不见，曾经的问候已经搁浅，忙碌将我忘在了沙滩，留下的沙已经被风干。
<br/>
<br/>        宁雅和部门的三个同事加班加点的江工作干完，在周天凌晨两点，三她们个同时站起来说：“终于完成了。”
<br/>
<br/>        “陈疯子将我们也逼成了拼命三娘。我要回去好好睡一觉。”一个同事伸着懒腰。
<br/>
<br/>        “睡个毛线啊，已经到周一了，明天还要迎接新人。”另外一个同事笑着说。
<br/>
<br/>        “算了，凑合着在值班室睡一觉吧，小雅我们一起凑合着住一晚？”他们工作的地方离宁雅住的地方还比较远，要是大家加班都会有直播室。
<br/>
<br/>        宁雅基本不在值班室睡，她无论多晚都会回家。
<br/>
<br/>        “你们俩个就字值班室休息吧，我回家。”宁雅说。
<br/>
<br/>        “这么晚了，你怎么走？”
<br/>
<br/>        “我开车着呢。”宁雅淡淡的说。
<br/>
<br/>        加班结束了，宁雅也轻松了很多，开着车，陈子昂临走时的话又在她耳边响起。
<br/>
<br/>        她的心动了。
<br/>
<br/>        沉寂了两年，她以为她这一生就这样颓废下去了，找个安稳的工作，等着家里人安排着相亲，嫁一个自己爱不爱都无所谓的男人。
<br/>
<br/>        可是陈子昂的话像一把火，将她的心燃烧起来了。
<br/>
<br/>        她也行的，是吧！
<br/>
<br/>        虽然她现在还没有明确的目标要去干什么，但是她突然就想试试自己的能力，看离开了家族她还能不能好好的活着。
<br/>
<br/>        陈子昂这个女人啊，总是会让人充满激情和希望！
<br/>
<br/>        等凌辰打电话的时候和问问凌辰的意见，看看她能做什么。
<br/>
<br/>        此时，宁雅特别想念凌辰，哪怕是发个信息也好，但是她看了一眼腕表，已经是凌晨三点了，于是她打消了给凌辰发信息的念头。
<br/>
<br/>        这么晚，凌辰早已经休息了，明天再说吧！
