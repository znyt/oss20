陆寒霆没有走，而是待在了总裁办公室里，他坐在黑色真皮椅上开始抽烟，烟雾缭绕模糊了他的俊颜，只隐约可以看到他紧蹙的眉心成了一道“川”字，猩红的烟灰从他指尖掉落下来，溢着一层戾气。
<br/>
<br/>        不知道过了多久，里面的休息室里突然传来了一道轻呼，“啊！”
<br/>
<br/>        陆寒霆一怔，将指尖的烟蒂快速的丢进了烟灰缸里，他拔开长腿就推开了休息室的门，“怎么了？”
<br/>
<br/>        夏夕绾已经哭够了，现在眼眶红红鼻翼红红的，纤长的羽捷上都挂着泪珠，她穿好衣服从盥洗台上下来了，但是因为四肢长时间没活动都已经麻了，刚落地差点就摔了。
<br/>
<br/>        夏夕绾抬眸看了一眼门边的男人，嗓音闷闷的，“你怎么没走？”
<br/>
<br/>        陆寒霆没想到她还愿意跟自己说话，她不是不喜欢他了么，她不是讨厌他了么？
<br/>
<br/>        陆寒霆嗓音沙哑，“你在这里，我不想走，也不敢走，总觉得…守着才踏实，我怕我走了，你也会走，以后…都找不回来了。”
<br/>
<br/>        夏夕绾哼了一声，这时她觉得两眼发黑，纤柔的身体直接往地毯上倒去。
<br/>
<br/>        陆寒霆瞳仁一缩，几乎睚眦欲裂，他伸出长臂一把将她扯进自己的怀里，“夏夕绾！夏夕绾你怎么了？”
<br/>
<br/>        夏夕绾睁开眼，陆寒霆英俊的五官在她视线里放大，现在他的脸上溢满了紧张和担忧，还有深深的自责和愧疚。
<br/>
<br/>        “我都说了今天我身体不舒服，你还不信，”夏夕绾将手指的伤口递到他的眼前，“在研究院里我手指不小心划破了，流了很多血，有点头晕，陆子羡只不过扶了我一下正好被你看到了，陆先生，你能不能信我一次？”
<br/>
<br/>        陆寒霆看着她苍白如纸的小脸，刚才在研究院里他就应该察觉到她脸色的不对劲，但是他被嫉妒冲晕了头脑，他都没有发现她受伤了。
<br/>
<br/>        现在他将她带回来又狠狠的折磨了一番。
<br/>
<br/>        “我信，我信了，”陆寒霆握住她的小手放在唇边凌乱而用力的亲吻了一下，“陆太太，对不起，我也不知道自己为什么总是这样，我很讨厌这样的自己。”
<br/>
<br/>        夏夕绾看着他发红的眼角，伸出小手搂住了他的脖子，“我还有点头晕，你抱我去床上，我休息一下就好了。”
<br/>
<br/>        陆寒霆迅速将她打横抱起，送到了柔软的大床上。
<br/>
<br/>        他拿出了医药箱，单膝跪在床边，开始帮她处理手指上的伤口。
<br/>
<br/>        他垂眸的样子很专注，动作轻柔的很怕弄疼她，处理好了手指的伤口他又往她的手腕上抹了药，那些被他弄红的地方他一一仔细而爱怜的处理了。
<br/>
<br/>        做好这一切，将医药箱放了回去，他起身，拿着扫把开始打扫这满地的狼藉。
<br/>
<br/>        夏夕绾看着他，他大手上的伤口没有处理，不过血已经凝结成块了，很快他蹲下身，捡起了那颗被他丢弃的药丸，沉默的放在嘴里然后吞咽了下去。
<br/>
<br/>        夏夕绾突然觉得有一只大手紧紧的拽住了她的心房，让她疼。
