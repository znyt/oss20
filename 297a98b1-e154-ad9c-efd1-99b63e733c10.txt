　　几大宗师一起点头，都是一脸的大义凛然。
<br/>　　如果叶少阳看到这一幕，只怕是下巴都要掉下去了。
<br/>　　“好。你们做到好，不过，纵然也是白搭，不消失一时半刻，等他死于晓寒手下，届时，我很希望看到各位的表情。”鬼姨说完，纵身离去。
<br/>　　张晓寒操控着五朝元气，继续攻击叶少阳的八门金锁阵，并且报之嗤然一笑。
<br/>　　“叶少阳，你是黔驴技穷了吗，你这八门金锁是死阵，我的五朝元气却是灵动自如，你只守不攻，早晚被我破了法身！我看你怎么死！”
<br/>　　叶少阳也还以一笑，摸出一枚铸母大钱，在口边吹了一下，“一钱追魂渡江去！”
<br/>　　铸母大钱飞进阵中，立刻将“五角星”激活。八枚铜钱一起飞起来，在叶少阳周身环绕。叶少阳手擒铸母大钱，朝张晓寒纵身奔去。
<br/>　　“这是金钱索魂术！”云春生吸着气说道，周围那些能够看懂的宗师，跟他都是一样的目瞪口呆的表情。
<br/>　　金钱索魂术，跟八门金锁阵，是完全不相干的两个阵法，只要是天师以上牌位的道士，单独都可以施展，但是连在一起，让金钱索魂术成为八门金锁阵的变阵，等于把两个法术的威力堆在了一起，难度自然也是乘积上升。
<br/>　　这种法术连用，不是没人用过，但非常考验施法者的实力和技巧，不然人人都可以把自己掌握的法术堆在一起用，人人都无敌了。
<br/>　　叶少阳用的这两种法术，都是非常复杂的法术，换成一般的天师，就算单独施展都要耗费巨大的法力，连用起来，这种难度……
<br/>　　“他是地仙！！”云春生非常肯定地给出这个判断，身边几个宗师也跟着点头。这是显而易见的事实。
<br/>　　“春春，你居然调教出一个地仙，之前却是一点没透底，你这事做的不对。”紫云真人道，“你要是早些时候说，我们都可以帮着指点一二，还是那句话，事关法术界的地位，我等可是不会藏私的。”
<br/>　　云春生跺脚叹道：“跟你们说实话，这不是我传人，是伏明子的徒弟！”
<br/>　　伏明子就站在云春生身后，因为前排都是自己长辈，他这个茅山掌教也不敢插话，但一直在听着，听见云春生这么说，当下一惊：“师伯，我……”
<br/>　　“你并不知情。我知道，我想来想去，他多半是你师父秘传的弟子……你师父几年前将大位传给你，便云游天下，很可能那时候收了这个叶少阳，也许是为了保护他，因此谁也没有告诉……”
<br/>　　伏明子一怔，仔细想来，还真是有这个可能。
<br/>　　“这叶少阳年少，你师父应该不会直接收徒，多半是算在你门下，名义上却是你的弟子。”
<br/>　　伏明子看了一眼自己身边的青云子，喃喃道：“怪不得他对我茅山的人事如此关心。”
<br/>　　再看叶少阳，顿时有了一种自己人的感觉，我的弟子啊……
<br/>　　心中打定主意，今天不管怎么样，也要从张晓寒手上把他救下来。
<br/>　　那边，叶少阳在八枚铜钱的灵光环绕之下，飞身来到张晓寒身边，两人缠斗起来。
<br/>　　张晓寒也收回了他的五朝元气，操控在手中，与叶少阳身上的八道金光不断碰撞，一时间灵光纷飞，两人身影相撞，战斗甚是激烈。
<br/>　　所有人都很吃惊于叶少阳的表现，谁也没想到叶少阳居然有如此强大的能力，与张晓寒战到不分胜负……
<br/>　　被废了的陈晓宇，还有那些个被叶少阳打伤的公会的弟子，一个个面色极为难看。
<br/>　　法术公会这边，只有鬼姨无动于衷。
<br/>　　叶少阳是地仙，她也看出来了，但只有她知道，张晓寒的真实实力，距离灵仙也只有一步之遥，虽然都在同排位，但这一上一下，差距可是巨大的，不过叶少阳毕竟也是地仙，因此不可能那么快就被击败。鬼姨在静静地等待，路遥知马力，她相信继续打下去的话，张晓寒最终必然能战胜叶少阳。
<br/>　　“轰……”
<br/>　　叶少阳操控着金钱形成的灵光，在身前汇聚成印，与张晓寒的五朝元气撞击在一起，形成一次极为强烈的撞击，两人都倒飞出去，叶少阳在空中调整了一下，踉跄落地。张晓寒却一头扎进远处的一棵树上，半天才爬下来。
<br/>　　不约而同地，两人朝着对方走了过去。
<br/>　　两人都是头发散乱，衣服也破了，血污满身，看上去十分狼狈。
<br/>　　张晓寒看向叶少阳的目光中，流露出一丝不可置信。他不敢相信，叶少阳居然有这么强的实力。
<br/>　　“倒是小看你了……”
<br/>　　“你现在仍然在小看我。”叶少阳对他招了招手。
<br/>　　张晓寒的脸色冷下去，岔开双腿，双手不断结印，口中念念有词，双手之间，逐渐有灵光溢出，在头顶上方汇聚成了一方大印，通体翠绿，上圆下方，大印的最上方，立着一只仙鹤的形状，单脚站在玉印上，昂首向天，看上去栩栩如生。
<br/>　　“灵宝大法司印！”
<br/>　　现场有人惊叫起来。
<br/>　　印法，也是法术的一大分类，有实印和虚印之分，这灵宝大法司印就是一种虚印，灵宝大法司印，是印法中十分高深复杂的一种，张晓寒能这么轻松地祭出，足可见他的修为，在人间法师中，也差不多算是登峰造极了。
<br/>　　只可惜，他的对手是叶少阳。
<br/>　　叶少阳见他结印，自己也依照作法，在头顶上空结了一个大印，众人立刻把目光移过去，然后当场都懵了：
<br/>　　叶少阳结的印，居然与张晓寒的一样，也是灵宝大法司印！
<br/>　　“这叶少阳，跟人结一模一样的印，未免太托大了吧！”人群之中，有人感叹起来。
<br/>　　“哼，恐怕他这么做，打的是自己的脸。”
<br/>　　一样的印法，一样的手段，这样也没有了什么相生相克和取巧的成分，比的那就是硬实力，谁强谁弱，一碰而知。
<br/>　　“山羊又开始装逼了。”四宝靠在岩壁上，摇了摇头。
<br/>　　凤兮捂着嘴笑。
<br/>　　吴嘉伟望着张晓寒，冷冷道：“我倒是也很想跟他打一场呢。”
