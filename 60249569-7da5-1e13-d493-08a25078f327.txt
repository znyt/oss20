        在手机屏幕上方弹出来的对话框里冒出“漫雪”这两个字后，瞬间让童筱颖睡意全无。
<br/>
<br/>        或许是因为彼此曾经是最好的闺中密友，如今却莫名其妙地成了最熟悉的陌生人的缘故，童筱颖对苏漫雪的感情，总隔着一层说不清道不明的间隙。
<br/>
<br/>        童筱颖点开了大学室友群，看到苏漫雪在群里各种炫富。
<br/>
<br/>        她在韩国买了各种奢侈品牌，大包小包挂在手臂上，而她那手腕上也是各种亮光闪闪的手链和钻石尾戒。
<br/>
<br/>        群里的另外两个女生，看到苏漫雪的炫富后，也是各种羡慕。
<br/>
<br/>        都说她找了一个好男朋友，让她把男朋友的照片亮出来。
<br/>
<br/>        结果苏漫雪在群里回复一句“男友害羞，不肯拍照”给敷衍了过去。
<br/>
<br/>        童筱颖趴在床上，拿着手机默默地看着她们刷屏，不吭声，不发表任何言论。
<br/>
<br/>        哪怕苏漫雪晒出了天价手提包和昂贵的护肤品，童筱颖也不为所动，甚至一点都不羡慕苏漫雪。
<br/>
<br/>        或许是人与人之间的价值观念不同。
<br/>
<br/>        童筱颖对那些奢侈品牌不太感兴趣，不过，当苏漫雪随后又晒出了当地的美食相片时，倒是勾起了她的羡慕心。
<br/>
<br/>        如果她也有资本和时间，她也想昼景陪着她一起去吃遍世界各地的美食。
<br/>
<br/>        随后，苏漫雪还在群聊里发了大红包，每人都有份，但是童筱颖却只是看着并未去点“领取”。
<br/>
<br/>        远在韩国的苏漫雪，坐在了一间豪华套房的梳妆镜前，拿着手机刷群的时候，发现筱颖未领取她的红包，这让她心里莫名其妙地感到不爽。
<br/>
<br/>        不管她在群里如何炫富，就从未见过筱颖出来冒过泡。
<br/>
<br/>        她知道，她一定有看到，为什么就不愿意出来吭声呢？
<br/>
<br/>        一定也是羡慕嫉妒她了吧！
<br/>
<br/>        苏漫雪红唇微扬，看着镜子里的自己，就像是看着筱颖的脸一样。
<br/>
<br/>        现在，她和筱颖的那张脸还未一模一样，但是要整成一模一样的，还需要时间。
<br/>
<br/>        她得慢慢整，不能操之过急。
<br/>
<br/>        苏漫雪以前一直觉得自己的原装脸才是最美的，可在医院里，听了那个主治医生对筱颖相片的夸赞后，心里莫名其妙地燃起一丝嫉妒的怒火。
<br/>
<br/>        那医生说，筱颖的脸，是纯天然的美，想必筱颖的妈妈一定也是个大美女。
<br/>
<br/>        然而，苏漫雪却觉得这是个讽刺。
<br/>
<br/>        筱颖的妈妈，长什么样，她又不是没见过。
<br/>
<br/>        诺母在他们那儿可是以“尖酸刻薄”出名，性格不好，身形臃肿，连美女的边都挨不着。
<br/>
<br/>        那样丑陋的一个女人，怎么就生出筱颖那样的美人胚子来。
<br/>
<br/>        而且，诺爸长得也不是很帅的那种类型。
<br/>
<br/>        就在苏漫雪心里在嫉妒童筱颖的时候，临海城依山别苑里的刘管家又给她发信息过来了。
<br/>
<br/>        大少奶奶，您什么时候从韩国回来？大少爷已经退役从商了。——刘管家。
<br/>
<br/>        噢！我还未玩够了！等我玩够了，我就回来！顺便帮我问一下大少爷，他喜欢什么？不喜欢什么？我好给大少爷带礼物回来。——苏漫雪。
<br/>
<br/>        只要是您买的礼物，不管是什么，大少爷都会喜欢的。——刘管家。
<br/>
<br/>        苏漫雪看到这条信息后，错愕地嘴角抽了抽。
<br/>
<br/>        现在，她心里倒是越来越肯定，这个大少爷一定是见过筱颖的，只不过，是不知道筱颖的真实姓名和家世背景而已。
<br/>
<br/>        忽然间，苏漫雪的眼中露出了一股冷冽的杀气。
<br/>
<br/>        筱颖是她的绊脚石，心头刺，她必须铲除她才行。
<br/>
<br/>        如果……
<br/>
<br/>        筱颖死了……
<br/>
<br/>        这世界上寻无此人，她从此以后不就可以高枕无忧了吗？
<br/>
<br/>        可杀人毕竟是犯法的事情，苏漫雪也只是心里头想想就过了，并未真的去这么做。
<br/>
<br/>        当刘管家告知傅夜沉，“苏漫雪”有关问过他的喜好时，傅夜沉心里高兴，嘴角也跟着情不自禁地微微上扬。
<br/>
<br/>        傅昼景坐在会议桌前一边听着策划部的部长讲解着策划案，一边留意着会议桌主位上的大哥傅夜沉。
<br/>
<br/>        也不知道大哥低着头是不是在玩手机，总而言之，看他那样子，在傅昼景眼里，就像是在看一个玩世不恭的富二代一样。
<br/>
<br/>        “傅总，您觉得我们的策划案如何？”策划部部长讲解完策划案后，立即询问傅夜沉的意见。
<br/>
<br/>        傅夜沉随后淡定地抬起头来，一脸严肃地提出了关于这个策划案中的几个漏洞性的问题。
<br/>
<br/>        傅昼景听着大哥的提问，也非常认可地点了点头。
<br/>
<br/>        刚刚他听这个策划部部长讲解这套方案的时候，就觉得大哥说提到的那几点实施起来确实存在不妥之处。
<br/>
<br/>        策划部部长被傅夜沉质问得哑口无言后，最后就像碰了一鼻子灰般，低着头坐下，并弱弱地应了一声：“我回头再去改改。”
<br/>
<br/>        “其他部门还有什么要汇报的吗？”傅夜沉抬眸看了大家一眼，声音不急不缓地问。
<br/>
<br/>        大家都沉默时，他便宣布散会，气势就像是皇帝在宣布退朝一样。
<br/>
<br/>        季思妍就坐在傅昼景的身边，听到傅夜沉说散会后，便着手整理文件和会议纪要。
<br/>
<br/>        傅昼景并未理会季思妍，而是起身走过去直接跟大哥傅夜沉打趣：“刚刚开会的时候，大哥是在桌子底下和我未来的准大嫂打情骂俏吗？”
<br/>
<br/>        “算是吧！不过，好像是我单方面的单相思。”傅夜沉苦笑，他思念她，而她却一心只想着玩。
<br/>
<br/>        “我一直以为，大哥你不会恋爱。原来，恋爱中的大哥，也有如此感性的一面。”傅昼景忍俊不禁地说。
<br/>
<br/>        从小到大，大哥拒绝了多少女生？只怕有上百号了吧！
<br/>
<br/>        傅昼景至今印象最深的一个，向大哥表白被拒后，还对大哥恋恋不忘的女生名叫李忆莎。
<br/>
<br/>        这个李忆莎好像是姚明娜的闺蜜，他曾听姚明娜提起过，说李忆莎至今都还喜欢着他的大哥，所以到现在都一直未谈男朋友。
<br/>
<br/>        一个女人痴情起来，真的可以做到一辈子都还爱着那个男人。
<br/>
<br/>        这样的单恋，确实令人心生敬佩。
<br/>
<br/>        只可惜，妾有情郎无意。
<br/>
<br/>        只怕到现在，傅夜沉都记不起她李忆莎是谁了。
