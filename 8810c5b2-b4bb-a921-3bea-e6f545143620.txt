<br/>  在上京城外西侧，有两座御苑，南北相对，北为金明池，南为琼林苑，也是宴请新进士之所。<br/>　　<br/>　　  宴由朝廷官员主持，一般会是负责省考的主官主持，但王康明显不想管，就由左相林相如代替了。<br/>　　<br/>　　  宴请之人，一甲，状元，榜眼，探花，二甲传胪肯定是要在的，其他选拔数位优秀之人。<br/>　　<br/>　　  除新科进士之外，还有内外帘官，即是考场的工作人员……<br/>　　<br/>　　  往日静谧的琼林苑，今日开放，热闹非凡，一个个进士及第者身穿绯罗圆领袍，长至脚，内穿罗裙，至于腰束黑朝带。<br/>　　<br/>　　  笑语不停，争相涌入。<br/>　　<br/>　　  古时范进，屡试不中，五十多岁仍是一介童生，他近乎绝望，眼看科考无门，但意外及第……<br/>　　<br/>　　  欢喜之余，差点疯掉。<br/>　　<br/>　　  这才是中个举人！<br/>　　<br/>　　  现在可是进士，可想而知，激动之心，又能参加琼林宴，这可是极大殊荣。<br/>　　<br/>　　  “杨兄，来了，”<br/>　　<br/>　　  “请，”<br/>　　<br/>　　  “杨兄二甲名列前茅，当真了得，”<br/>　　<br/>　　  “哪里，哪里，跟状元宋节守比起，真的是不足挂齿，”<br/>　　<br/>　　  “那还是个怪物，这么偏难的考题，他竟然无一错漏，明法，明算也是如此，震惊所有！”<br/>　　<br/>　　  “听说他还未参加殿试，就被陛下亲点为状元！”<br/>　　<br/>　　  “不止是宋节守，那个南明湘，也是厉害的紧啊！”<br/>　　<br/>　　  众多及第考生，争相议论，有一人突然提问道：“不知王康会不会来？”<br/>　　<br/>　　  “应该会吧，众多人都请言要他来，”<br/>　　<br/>　　  “听说本来取不了这么多名额，但王康给提话了，才是如此……”<br/>　　<br/>　　  一众人都默不作声，对于王康他们都很是复杂。<br/>　　<br/>　　  本为弱冠之年，比之他们中很多人都年纪要小，出身富阳伯爵府的富家少爷，败家子……<br/>　　<br/>　　  科考主官，出考题，抓舞弊，行改制，如此种种，让人们都迷糊了，不知哪个是真的了。<br/>　　<br/>　　  不多时，有官员也是到来依次jinru，王康来时已经稍晚，门口的人已经不多了。<br/>　　<br/>　　  周青赶着马车，将他送来，只身一人，今天的他也穿上了官服，毕竟是参加琼林宴，比较郑重。<br/>　　<br/>　　  “你就在此等着吧，”<br/>　　<br/>　　  “是，”周青应道。<br/>　　<br/>　　  而后王康背着手，像门口走去。<br/>　　<br/>　　  “站住，不过微末五品也想入这琼林苑？”<br/>　　<br/>　　  正进门时，被两边兵卒拦截。<br/>　　<br/>　　  王康现在的本职还是主客司郎中，从五品上，官服为浅绯色，绯即是红，饰小杂花，径一寸五分，一律戴漆纱制成的展角幞头。<br/>　　<br/>　　  与进士服不同。<br/>　　<br/>　　  而今日参会的官员，四品都是少见，又哪有五品？<br/>　　<br/>　　  王康虽然是名声大，但真正认识面的也没几个，更不用说是普通兵卒，又是天色偏黑。<br/>　　<br/>　　  因此，直接被拦。<br/>　　<br/>　　  王康也是讶然了，他现在进出皇宫都很随意，没人阻拦，在这反而被拦了，真是新鲜。<br/>　　<br/>　　  他淡淡道：“你敢拦我？后果自负啊？”<br/>　　<br/>　　  “哟喝，”<br/>　　<br/>　　  这一句也把守门兵卒刺激，冷笑道：“一个五品官员，也敢大言不惭，你可知这是什么地方？”<br/>　　<br/>　　  “这里是琼林苑，今日要举办琼林宴，受邀者都为及第进士，大官要员，是由左相大人亲自主持，我看你是疯了吧！”<br/>　　<br/>　　  “识相点，就快点滚，不然扒了你这层官服！”<br/>　　<br/>　　  兵卒语气相当不客气，如连珠炮一般。<br/>　　<br/>　　  这般动静也将不少人吸引，原本进门的进士也回头而看。<br/>　　<br/>　　  一人摇头道：“大概又是想混迹进来的。”<br/>　　<br/>　　  每当琼林宴时，便有不少未邀请的官吏想要混迹而进，博些名脸。<br/>　　<br/>　　  “从五品？”<br/>　　<br/>　　  有人呢喃，面色疑惑。<br/>　　<br/>　　  这时又有一辆马车行驶过来，兵卒见至，也不理会王康，慌忙迎了过去。<br/>　　<br/>　　  一个年纪四旬的中年人走下，身穿二品官服，不怒自威，他正是当朝左相，林相如。<br/>　　<br/>　　  “左相大人到了，”<br/>　　<br/>　　  一声高喊，从苑内一众官员，进士走了出来，为首的正是宴立群，纪宁等人。<br/>　　<br/>　　  下官迎上官，自古便是如此。<br/>　　<br/>　　  见得王康还在一旁站着，没有离开。<br/>　　<br/>　　  那兵卒好似是为了故意表现，反而大声喝道：“此乃重地，岂是你来之地，还不速速离开，冲撞了诸位大人，你可担待的起？”<br/>　　<br/>　　  听到此，王康更是觉得有意思，他也没理会这个小卒，对着正出来的宴立群大声道：“宴大人，我可是如约来了，但进不去啊，”<br/>　　<br/>　　  兵卒当即喝斥道：“你大胆，宴大人是你叫的吗？”<br/>　　<br/>　　  未等宴立群说话，倒是在一旁的林相如先注意到，几步走过。<br/>　　<br/>　　  兵卒献媚道：“林大人，此人不过微末五品，妄想混迹，您看如何处……”<br/>　　<br/>　　  林相如当即对他喝道：“我看你是瞎了狗眼！”<br/>　　<br/>　　  “嗯？”<br/>　　<br/>　　  兵卒傻了，这是什么情况，林大人怎么会骂自己呢？<br/>　　<br/>　　  这时宴立群也走了过来，笑着道：“王康，你这谱摆的有点大啊，来这么晚，你是想压轴吗？”<br/>　　<br/>　　  “您说对了，我就是故意的，等着诸位出来迎接呢，”<br/>　　<br/>　　  “哈哈！”<br/>　　<br/>　　  王康这一番话，让所有人都笑了。<br/>　　<br/>　　  林相如笑着道：“本来这场琼林宴就是科举主官主持，是王康之责，我是暂代啊！”<br/>　　<br/>　　  王康，他就是王康？<br/>　　<br/>　　  是了，从五品主客司郎中，也唯有他敢如此了……<br/>　　<br/>　　  糊涂，真是糊涂！<br/>　　<br/>　　  兵卒当即面色惊恐，人的名，树的影，这次舞弊大案，至使多少官员落马，多少家族四散……<br/>　　<br/>　　  朝中都传就是这王康一手造成！<br/>　　<br/>　　  又经流传，更是平添传奇色彩，人们生怕躲不及遭受灾祸，御史台如何？<br/>　　<br/>　　  现在都停止了对王康的弹劾！<br/>　　<br/>　　  可谓是谈之色变！<br/>　　<br/>　　  兵卒越想，越觉得恐怖，他可真是瞎了狗眼，两腿发软，忙的跪下……<br/>　　<br/>　　  但王康却看都没看他一眼，在众人的簇拥下，jinru琼林苑。<br/>　　<br/>　　  这边一众及第进士，面面相觑。<br/>　　<br/>　　  王康，这就是王康！<br/>　　<br/>　　  他的官服在一众三品，二品官员中，是那么的扎眼，论年纪，他又是最轻，论资历，他又是最短……<br/>　　<br/>　　  但在其中，却没有半分不适，左相林相如，参知政事纪宁，礼部尚书宴立群，几位翰林学士。<br/>　　<br/>　　  甚至就连一直被人所知，对他有很深成见得兵部尚书吴之荣，也与之笑谈……<br/>　　<br/>　　  这些种种，不由让人感叹，这分气度，不由震撼众人……<br/>　　<br/>　　  【作者题外话】：第三更，<br/><br/><br/>
