
<br/>　　<br/>　　    李伯把王惊蛰带进正厅的时候，李兰荣和茅清水就用审视的目光，几眼就把他给看透了。<br/>　　<br/>　　    王惊蛰还是那幅打扮，一席长袍下是千层底的布鞋，斜跨着一个老旧的帆布包，走起路来的时候发出“叮当”的动静，除此以外身无长物，脸上表情古井不波的，相貌平淡无奇，看起来就像是个和他年龄不太对称的老学究。<br/>　　<br/>　　    李兰荣和茅清水同时对视了一眼，理所当然的闪过了一抹失望的眼神，果然如此啊，出类拔萃和人中龙凤这种词果然用不到他的身上来。<br/>　　<br/>　　    王惊蛰大大方方的站在两人身前，然后弯腰行了一礼：“朝歌，王氏一门王惊蛰拜见陇西李氏家主”<br/>　　<br/>　　    李兰荣伸手示意了下，然后跟李伯说了一声上茶，片刻后一盏青花瓷茶杯泡着热气腾腾的热茶被送到了王惊蛰面前，他端起杯子抿了一口，一股清香瞬间从喉头一直香到了肚子里，令人回味无穷。<br/>　　<br/>　　    茅清水笑道：“看你的姿势也是常喝茶的人，这茶怎么样？”<br/>　　<br/>　　    王惊蛰砸吧了下嘴，实话实说的说道：“好茶”<br/>　　<br/>　　    “茶是上好的明前洞庭碧螺春，每年春分采摘谷雨前后结束，由一妙龄少女用红唇摘下，都是最嫩的雀舌，从杀青，揉捻、搓团显毫到烘干也是由几十年经验的老师傅操刀置办的，这碧螺春每年大概也就出个百八十斤左右，今年的新茶炒完之后江南有几个大老板托人带话想要个三斤五斤的都没有门路，因为那一片茶园几百年来都是我们李氏种植的，咱们啊只喝不卖，专攻李家待客用，还有那泡茶的茶水是从杭州虎跑泉取出来由专人送回家中的，寻常的水我们都是不用的”茅清水顿了顿，伸出手指又指着王惊蛰手里的杯子说道：“唐宋年间青花瓷一套，是几年前从苏富比拍来的，据说当时欧洲有几个皇室成员也在场都想要这套茶具，不过最后李家侥幸了那么一点，给拍到了手里”<br/>　　<br/>　　    王惊蛰心里叹了口气，这个逼装的我真不想给你点个赞，我王家柜子里放着武夷山母树上摘下来的大红袍有好几斤呢，我和谁吹牛逼了。<br/>　　<br/>　　    王惊蛰嘴唇抽搐了几下才憋出一句话来：“受宠若惊啊”<br/>　　<br/>　　    茅清水拢了拢头发，淡淡的说道：“我和你说这些的意思是，你觉得假如娶了我们李家的闺女，平日里你能给他这些在我们来说看似平常，在别人看来却又高不可及的生活品质么？李家的女儿享受惯了这种品质性的生活，如果换一种生活方式的话，她恐怕会不太习惯呢，所以我就开门见山的和你说，关于我家老太爷和你长辈定下的亲事，我不怎么看好”<br/>　　<br/>　　    王惊蛰没想到初一见面就听到了这样的一番话，一时间有些不知所答了。<br/>　　<br/>　　    茅清水看着不知所措的王惊蛰心里又给他种下了另一个评价：“没见过市面，为人木讷”<br/>　　<br/>　　    “指腹为婚看起来好像是一段佳话？哎，这不过是早几十年旧社会里老掉牙的规矩罢了，现实社会里哪还有这种情节啊”茅清水忽然拔高了声调，说道：“我们家老爷子多年前和你长辈定下的婚事是不是有点儿戏了，你说呢？”<br/>　　<br/>　　    王惊蛰哦了一声，嘴里吐出两个字：“悔婚？”<br/>　　<br/>　　    “呵呵······”茅清水笑了，摇头说道：“虽然我不看好这门婚事，但毕竟是当年老爷子一言九鼎给定下来的，我们大门大户的断然做不出出尔反尔的事来”<br/>　　<br/>　　    王惊蛰愣住了，有点没太反应过来，对方前后话里的意思让他有些发懵了。<br/>　　<br/>　　    “唉，对了，当年老爷子和你家长辈定的是什么婚事？你要娶的是我们家里的哪一位？”<br/>　　<br/>　　    “李家长孙女，李家大小姐·······”<br/>　　<br/>　　    “啊，对，对，对，李家长孙女李家大小姐”茅清水点了点头，跟李伯说道：“李伯，去把李家的大小姐抱过来吧”<br/>　　<br/>　　    王惊蛰：“？？？”<br/>　　<br/>　　    抱过来？<br/>　　<br/>　　    这个词用的让王惊蛰一脸懵逼了。<br/>　　<br/>　　    片刻后，李伯抱着一个六七岁左右，嘴里还舔着棒棒糖脑袋上梳着根朝天辫的小女孩来到了正厅里。<br/>　　<br/>　　    茅清水指着李伯怀里的孩子说道：“这就是李家长孙女，大小姐，李香荷”<br/>　　<br/>　　    王惊蛰表情从惊讶到惊愕最后变成了鄙夷和嘲讽，他看着李兰荣和茅清水说道：“你也说了李家是大门大户，但你们悔婚的手段可真让我吓了一跳，二十几年前我父亲和你家老人定下婚事的时候，说的应该是当时快生了的那个，算算时间的话她现在应该也有二十几岁了吧？人家都是怀胎十月，你莫不会告诉你却是怀胎十几二十年吧？哪吒他妈也没这么离谱啊······”<br/>　　<br/>　　    茅清水表情顿时冷了下来，平淡的说道：“当时定的可是李家的长孙女，应该是姓李吧？但抱歉了，我生下的女儿之后可没有姓李，她是随我姓的，所以她可算不上是李家的长孙女大小姐，这一位才是！”<br/>　　<br/>　　    茅清水在说姓李的时候，这两个字眼咬的非常重。<br/>　　<br/>　　    王惊蛰张着嘴瞠目结舌，半天后才大笑了几声，说道：“我懂了，我懂了，这是一出李家版本的狸猫换太子啊”<br/>　　<br/>　　    “要不你再等个十几年左右，等这李家大小姐成年了，再迎娶她过门？”茅清水淡淡的说道。<br/>　　<br/>　　    王惊蛰“豁”的一下突兀的就站了起来，目视两人说道：“你们打定主意了？”<br/>　　<br/>　　    茅清水皱眉说道：“你配不上李家的姑娘”<br/>　　<br/>　　    “有件事你可能不知道，和我们朝歌王氏一门许下的承诺，只有我们不取，而没有人敢不给的道理······”<br/>　　<br/>　　    “年轻人，你话说的太满了”李兰荣第一次开口，却是呵斥了王惊蛰一句。<br/>　　<br/>　　    “呵呵·······”王惊蛰笑了笑，一甩长袍转身就走。<br/>　　<br/>　　    走到门口的时候，王惊蛰又突然回过了头：“一月之后我亲自来退这门婚事，到时候希望你们能接的住”<br/>　　<br/>　　    茅清水摆了摆手说道：“随你的便吧”<br/>　　<br/>　　    “到时，后果自负！”王惊蛰冷笑着瞅着两人的眉间，李兰荣和茅清水左眉下都略有红润，这是家中有喜的面相：“在我没退婚之前，你家的女儿是嫁不出去的”<br/>　　<br/>　　    父母二人，右眉红润为男娶，左眉为嫁女，也就是俗称的喜上眉梢。<br/>　　<br/>　　    王惊蛰刚见李兰荣和茅清水的时候，还以为他俩喜上眉梢是因为自己的提亲，但现在看来是这李大小姐早就名花有主了？
