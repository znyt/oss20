        五人团队组的这件事情定下来后，明司寒交给了连星儿一本厚厚的IT术语词典。
<br/>
<br/>        “我觉得你英文不错，所以，这个对你来说，应该没难度吧？”明司寒会心一笑。
<br/>
<br/>        连星儿自然不想被他小看，将词典抱在怀里，笑了笑：“当然，完全没问题。”
<br/>
<br/>        经过了那一晚上的思想斗争，她算是想开了。
<br/>
<br/>        虽然自己的人生总被爹地给算计着，但至少爹地是为了她好。
<br/>
<br/>        就目前来说，她觉得明司寒这个男生确实还不错。
<br/>
<br/>        她可以跟爹地过不去，但是她不能跟自己喜欢的男生过不去啊！
<br/>
<br/>        和顾千凝她们说明了“同居”的缘由后，连星儿再次被她们三从寝室里推了出去。
<br/>
<br/>        明司寒来接连星儿的时候，顾千凝、赵安彤和路娅玟就像三个“姨母”一样，满脸笑容地给连星儿“送行”。
<br/>
<br/>        “我还会回来的啊！”连星儿忍不住补充了一句。
<br/>
<br/>        顾千凝看着坐在副驾驶座上的连星儿，一边挥手，一边开玩笑道：“是是是，我们都知道，回来的时候，记得带上我们的小侄女或者是小侄子噢！”
<br/>
<br/>        赵安彤和路娅玟也在一旁随声附和：“去掉‘或’字，带一双回来呀！”
<br/>
<br/>        “……”连星儿无语了。
<br/>
<br/>        这帮损友噢！
<br/>
<br/>        连星儿瘪着嘴，腹语道：“我自己都还是个‘宝宝’了。”
<br/>
<br/>        身旁，坐在驾驶座上的明司寒只笑不语。
<br/>
<br/>        到了公寓后，张子晋、邢乐游和仲正浩已经在公寓里开始忙着安装台式电脑了。
<br/>
<br/>        而明司寒则将连星儿的行李搬到了卧室里。
<br/>
<br/>        既然是单身公寓，卧室自然只有一间。
<br/>
<br/>        连星儿不禁好奇地问道：“我睡房间的话，你们睡哪？”
<br/>
<br/>        “我楼上不是还有一个观星台吗？我们四个人睡上面。”明司寒莞尔一笑，顿了一下，又补充道，“房间里有内卫，如果你不放心，晚上你还可以反锁房门。”
<br/>
<br/>        “噢，好。”连星儿尴尬地笑了笑。
<br/>
<br/>        只不过，接下来，他们的训练倒是让她大开眼界。
<br/>
<br/>        因为涉及到学校名誉，所以，明司寒对这次比赛极其用心。
<br/>
<br/>        连星儿抱着词典坐在一旁，全程听指挥做笔录。
<br/>
<br/>        如果拿这种程序设计比赛当竞技游戏比赛的话，她连星儿的位置就相当于是辅助了。
<br/>
<br/>        “那我们这段时间是不需要去上课了吗？”连星儿好奇地问道。
<br/>
<br/>        明司寒点了点头：“离比赛那天只剩下七天的时间了。所以，我们现在集中在一起是加强训练。”
<br/>
<br/>        “你好忙噢！我记得，你旗下还有战队赛。”连星儿接着说道。
<br/>
<br/>        明司寒自信地笑了笑，抬起手来，宠溺地摸了摸连星儿的头顶：“我这不叫‘忙’，这叫‘强大’。我若是不‘强大’点，怎么配得上我家星儿公主，对不对？”
<br/>
<br/>        “你两别放电了，我们这边已经测试成功了，现在只待激活！”张子晋催促道。
<br/>
<br/>        明司寒收了手，正式进入训练状态。
<br/>
<br/>        连星儿也开始随时待命，替他们翻查一些比较难记的专业术语。
<br/>
<br/>        不过，通常在明司寒这里，她只要替他翻一遍，他便记住了。但是张子晋他们三个就不行，有个时候她还得十分耐心地替他们三再翻个三四遍。
<br/>
<br/>        当他们四个不需要她帮忙翻词典的时候，连星儿杵在一旁，一手托腮，看着明司寒，莫名其妙地发起呆来。
<br/>
<br/>        他的十指修长，敲击键盘的时候，就像是在弹钢琴一样，速度很快又灵活。而他那黑色的瞳仁里，映着电脑屏幕上快速运转的程序代码。
<br/>
<br/>        连星儿看着看着，不知不觉中直接趴桌上睡着了。
<br/>
<br/>        “喂，连大小姐，给我翻……”张子晋的话还未说完，就被明司寒拿词典堵住了嘴。
<br/>
<br/>        “你自己翻，别吵她。”明司寒轻声道。
<br/>
<br/>        邢乐游和仲正浩见状，捂着嘴偷笑，却不敢笑出声。
<br/>
<br/>        这次的大赛，其实每个组都会有导师加入，但是明司寒却拒绝了导师。
<br/>
<br/>        因为，在他看来，他就是主控者，不需要再多一个主控者进来“发号施令”。
<br/>
<br/>        张子晋他们三个也非常配合他，最主要的是，他们三个还百分百地对他“服从指令”。
<br/>
<br/>        明司寒接着示意他们三个敲击键盘时，手下的声音弄小声点，然后起身将连星儿从椅子上抱了起来。
<br/>
<br/>        仲正浩见明司寒抱着连星儿去了卧室后，忍不住跟邢乐游和张子晋调侃道：“寒哥这货，将来绝对妥妥的‘老婆奴’，你们信不信？”
<br/>
<br/>        “用得着将来吗？我看现在就是了。”张子晋忍俊不禁地说。
