　　第208章
<br/>　　“这里是长安，我的祖祖辈辈都生活在这里。酒楼由我祖父传给家父，家父又传给我，如今在我手里已经营二十年。”
<br/>　　昏暗的光线中，隐隐约约能看清男人的脸，他腮部肌肉收紧，慢慢倾诉着。
<br/>　　“半个月前，相府的管家来到我的酒楼，点了一只烧鹅，一坛花雕，四样小菜。花雕都喝了，烧鹅只吃了一半。回到家他就病倒了，说是我酒楼的烧鹅有问题，要我赔他一千两银子。”
<br/>　　“烧鹅的方子是我祖父传下来的，全长安的人都吃过我家的烧鹅，从来没有出过问题。我告到官府，府衙却查封了我的店。我打听了很多人，他们都说你能帮助我。”
<br/>　　“为什么你要找官府，而不是先来找我？”
<br/>　　男人对面的桌子后，坐着一个人，阴影里看不清他的脸，怀里好像还抱着东西，猫或者兔子什么的。
<br/>　　“我现在来找你了，你尽管吩咐，我一定照做，但求你一定要帮我忙，让那些狗官受到教训。”
<br/>　　“抱歉，我帮不到你。”
<br/>　　阴影里的人道。
<br/>　　“你要什么我都可以给你。”男人站了起来，手摁着桌子：“在整个长安城只有你能帮助我。”
<br/>　　“全长安都吃过你家的烧鹅，我就没有吃过，今天你有事情要求我，却不知道给我带两只烧鹅。”
<br/>　　“我太着急了，只希望你帮我主持公道。”
<br/>　　“你从来没有尊重我，把我当成恶霸，杀人犯，带着你的银子来，要我帮你杀人。我知道，你开酒楼挣了钱，觉得律法和官府可以保护你，不想和我这种人扯上关系。直到现在，你到我面前，让我帮你讨回公道，可从你坐下至今，却连一声大哥都没有喊我。”
<br/>　　“大哥。”男人低下头。
<br/>　　阴影里伸出一只手，轻轻拍拍他的肩膀。
<br/>　　“我会帮你主持公道，日后，我或许也会遇到些麻烦，但这一天或许永远也不会到，我只希望当我遇到麻烦的时候，你也可以帮助我。”
<br/>　　……
<br/>　　徐神机、秦蛮、徐灵儿立在墙边，目瞪口呆的看着这一幕。
<br/>　　阴影里的程大雷走出来：“干嘛，人设很重要的好不好。”
<br/>　　三人仍愣在哪里，一时没有反应过来。
<br/>　　“阿喜，阿苦。”程大雷唤了一声。
<br/>　　门外的两个男人走进来，程大雷一摆手：“互相认识一下，大家都是弟兄，阿喜，原先在军中任职，后来出了一些事，入了江湖。”
<br/>　　徐神机只看到这阿喜体壮如牛，面目狰狞，像那庙里的怒目金刚相仿。
<br/>　　“阿苦，原先是做掌柜的，有个好脑子，算得好帐。”
<br/>　　这阿苦和阿喜截然相仿，身材干瘦，一双桃花眼看着不像好人。
<br/>　　“阿喜，这件事交给你去办，莫要杀人。”
<br/>　　“是。”
<br/>　　徐神机愣愣的看着这一幕，此时道：“大当家，您一直做这些事？”
<br/>　　“是啊，我每天很忙的。”程大雷道：“没看到我都瘦了。”
<br/>　　徐神机三人已来了三天，对地下城的事情也有了一些了解。
<br/>　　这地下城看上去乱，乌烟瘴气，妖魔汇聚，但事实上却是乱而有章，内里井井有条。
<br/>　　长安城很大，每天也有无数事情发生，有些事情在阳光下，有些事情却是阳光照不到的。而各种各样的事情，都会汇聚到程大雷这里，由他统筹管理，当然，只是在程大雷势力覆盖范围内，并不包括整座长安城。
<br/>　　比如，某个大人物丢了价值连城的夜明珠，报到衙门，限定必须什么时间破案。捕快找来找去找不到，于是就求到程大雷这里。
<br/>　　『好，让我查查，看最近几天谁偷了夜明珠。喔喔，在这里呢，夜明珠拿去交差。日后我若遇到麻烦，诸位也要帮我一个小忙』
<br/>　　再比如，某个杀人犯躲到这里，也有看看你杀的什么人，如果是恩怨有主，杀了仇人，不仅要安排跑路的路线，还要准备路费。若是子弑父，弟杀兄之类伤天害理的，地下城也是不提供庇护的。
<br/>　　这世道，阳光照不到的地方太多了，有光明的地方必有黑暗，黑暗中便滋生程大雷这种人。用了大半年时间，程大雷织了一张大网，如今很多人都靠这张大网生活。
<br/>　　同时，这张大网也是一道无形的墙，隔绝了很多人。比如某个武者被大户人家雇到家里看家护院，竟然没有和程大雷打招呼，那么好吧，一天八十个贼去偷你，只有千日做贼，没有千日防贼，任你天大本事也扛不住。要么是灰溜溜离开这地盘，要么是到地下城拜山门。
<br/>　　所以说，程大雷每天的确很忙的。各种各样的事情汇聚过来，由阿苦分门别类，派人处理。有些他处理不了的，就送到程大雷这里，而若程大雷还处理不了，那就是真没办法处理。
<br/>　　阿苦捧着账本，向程大雷禀报最近的事情。
<br/>　　“黑狐帮和铁担帮抢地盘，双方已经决定在今夜动手，大哥你看这件事怎么办？”
<br/>　　“不就是两帮要饭的么，学什么人打架，告诉他们，把地盘一分两半，不许动手。”程大雷一边洗脸一边道。
<br/>　　“神武镖局护送的东西被葫芦山劫了，他们派人过来求情，看大哥能不能帮他们要回来？”
<br/>　　“是不是神武镖局又忘了给葫芦山上供，我说什么来着，仗着手下有几个猛人，就什么规矩都不守，不管他们，让他们长点教训。”
<br/>　　“上次来得那个戏班。台柱子被当官的看上了，戏班的人都被丢进了大佬。不是大哥把他们救出来的么，现在那个台柱子想当面感谢大哥。”
<br/>　　“好看么？”程大雷一顿。
<br/>　　“好看是挺好看的，不过……是个男的。”
<br/>　　“欸……”程大雷一身鸡皮疙瘩：“让他快滚，别过来烦我。”
<br/>　　“好啦，其他事情你看着办吧，今天我带我的弟兄出去转转。”程大雷面向徐神机三人：“你们等我化完妆，带你去长安城里逛逛，长安城我很熟的。”
<br/>　　“大当家，你现在出门还化妆呢？”徐灵儿满脸嫌弃。
<br/>　　“唉，别提了。”程大雷长叹口气：“上通缉令了。”
<br/>
