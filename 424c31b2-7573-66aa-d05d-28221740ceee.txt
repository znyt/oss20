　　第174章
<br/>　　青牛山，蛤蟆寨。
<br/>　　程大雷行走在山寨中，检查着防御及后勤。
<br/>　　真正的战争无法和想象中一样，与北蛮部初次交锋其实已暴露出很多问题。单兵体力、战斗意志等根本无法和北蛮部的战士相提并论，毕竟，程大雷掌握的只是一伙山贼，说他们是乌合之众好像也没什么不对。
<br/>　　再给程大雷一点时间，程大雷未必不能将他们训练成一支令行禁止的精兵。在这个士兵素质普遍不高的时代，即便只有两千人，也足以横扫大半疆域。
<br/>　　可惜，没有时间。
<br/>　　所谓战争，怎么可能让你一切准备好了才开始。如果真的一切都准备好了，你就去打别人了。
<br/>　　程大雷身穿幽州王赏赐的官服，身后背大斧，腰间佩宝剑，三枚印章在他腰间叮当作响。
<br/>　　他以如此装扮行走在山寨中，言必称『戎族的渣渣们』，语必说『要将畜生们赶尽杀绝』。
<br/>　　山寨的战士看到程大雷如此模样，纵然觉得他有些浮夸，但心里也难免会想：或许大当家真有必胜把握。
<br/>　　大家已经和戎族交过手，怯意稍减，在战斗中大家知道，戎族并非不可战胜，他们也是血肉之躯，一枚弩箭就可以将他们放倒。
<br/>　　士气，便是在如此状态下累积。
<br/>　　而至于究竟能不能打败北蛮部，程大雷心里其实也在打鼓。他夙兴夜寐，或与徐神机、刘悲、黄三元商议接下来的战斗、或巡视城防，所为的不过就是打赢这两个字而已。
<br/>　　咚咚咚
<br/>　　急促的战鼓响，从城门方向传来，瞬间响遍山谷。
<br/>　　“大当家，戎族又来攻城了。”
<br/>　　“又来！”程大雷也有些无语，距离上次交锋还没过两天，马上又来，戎族要不要搞得这么敬业。
<br/>　　“上城墙，上城墙。”程大雷大吼，也别无他法，无非是兵来将挡，水来土掩而已。
<br/>　　而且程大雷知道，上次北蛮部没讨到好处，这么急的又杀过来，肯定是已有了必胜的把握，或者是，他们已觉得自己可以打下蛤蟆寨。
<br/>　　蛤蟆寨的战士上了城头，只见从北蛮部大营方向，一队人涌了过来。
<br/>　　“大当家，你快看！”
<br/>　　徐神机手遥指着远方。
<br/>　　程大雷已经看见了，在队伍最前方的，并非是戎族的战士，而是……平民。
<br/>　　是帝国的平民，他们有老有少，有男有女，程大雷即看见了面容枯槁的老者，也看到了七八岁的黄口小儿。
<br/>　　他们被麻绳串在一起，粗略看有几千人众，戎族骑兵在他们后面挥着辫子驱赶，但凡有敢回头者，便是穿心一箭。
<br/>　　痛苦者，哀求者，跪地不起者，都难逃被驱赶的命运。
<br/>　　他们自然知道戎族要他们干什么，却又明白今日自己的命运与猪狗并无差别。
<br/>　　后面戎族的弯刀不可能赐宽恕于他们，所以他们只有向前，希望前方帝国的子民能赏赐慈悲。
<br/>　　“救救我们啊，救救我们。”
<br/>　　“让我们进去吧，求求你们了。”
<br/>　　各种各样的声音交织在一起，在城头下响成一片。
<br/>　　“这些畜生，竟然用难民冲城！”
<br/>　　“大当家，怎么办？”
<br/>　　程大雷手摁剑柄，眼角青筋暴跳。
<br/>　　事实上，程大雷有三板斧，戎族打仗也有三板斧。第一板斧就是屠城：赶紧投降吧，不投降我们就屠城，很多城池就是被他们这样打破的。问题是就算真的投降，他们也会屠城。
<br/>　　第二板斧就是用难民冲城，以难民的血肉之躯阻挡帝国的强弓劲弩，造成守军慌乱，趁机破城。
<br/>　　第三板斧就是强攻了，仗着士兵不畏死，很少有城池能敌得过他们的强攻。
<br/>　　这些事，程大雷已提前打听过，并不是不知道。但知道是一回事，真正面对是另外一回事。
<br/>　　此刻数千难民冲过来，已接近城头弩箭的射杀范围，程大雷清晰的能看到他们的脸。
<br/>　　看着他们的脸，看着他们的眼睛，你有足够的勇气杀死他们？杀死与你同样的生命？
<br/>　　程大雷没有这个勇气。
<br/>　　但有些事并不是没有勇气就不做。
<br/>　　“放箭。”程大雷拔出佩剑，重重挥下。
<br/>　　他扳开机括，亲自射出了第一枚重箭，他看着箭射出去的方向，看着被弩箭穿过的身体。临死前惊恐的表情，睁大的双眼，从身体中涌出的血水。
<br/>　　程大雷要看清这些，要记住这些，虽然他并不想记住这些。
<br/>　　“放！”
<br/>　　“放！”
<br/>　　“放！”
<br/>　　弩箭如雨，三百步能穿马匹，城头下挤满密密麻麻的人，一枚弩箭便能射穿好几人的身体。
<br/>　　嘟，收到恐惧值999
<br/>　　嘟，收到恐惧值999
<br/>　　嘟，收到恐惧值999
<br/>　　……
<br/>　　程大雷的恐惧值在飙升，城头射出的弩箭毁灭了难民最后一点希望，那是濒死的恐惧，有人吓得一言不发，有人却是放声大吼，有人将手中的婴儿举起，高喊着：救救孩子，救救我的孩子！
<br/>　　恐惧和憎恨在程大雷身上累积，他一言不发，咬紧牙关，指挥放箭。
<br/>　　北蛮部发起冲锋。
<br/>　　上次是试探，这次是强攻，北蛮部是觉得他们能打下蛤蟆寨的，于是攻城的势头远超过上次。
<br/>　　城头几次被突破，城墙被投石机砸出缺口，蛤蟆寨像一艘风暴中的渔村，危在旦夕。
<br/>　　但或许是因为射杀难民但缘故，蛤蟆寨的山寨爆发了从未有过的战斗意志。借云梯爬上城头的戎族被扑下去，徐神机带人抢修城墙，有一伙戎族冲到城门，正当他们用重木撞门时，招待他们的是从城头落下的滚木和火油。
<br/>　　“赤眉，咱们死的人太多了，让战士们撤下来吧。”托蛮。
<br/>　　赤眉眉头拧起，他不想撤，毕竟付出的伤亡已经太大，可他也看得出来，战士们挤在城头下，迎接他们的是大面积大面积的死亡。
<br/>　　“撤……退。”赤眉的手无力挥下。
<br/>　　第二场战斗，赤眉没有打下蛤蟆寨。
<br/>　　沐浴过献血和战火的城墙，空气中漂浮着浑浊的味道。城头下堆积着成山的尸体，血汇成小溪，向低洼处流去。
<br/>　　这一战，让程大雷真切感受到了战争的残酷性，也借着这一战，他的恐惧值达到百万。
<br/>　　
<br/>(本章完)
