
<br/>　　<br/>　　    贺三刀还想追着再问，王惊蛰就指着屋子里的藏品说道：“看看你这些东西，有多少件是从坟坑里挖出来的？这些东西常年伴着死人堆，早就沾染了不知道多少的死气，你全都给搜罗来了时间一久不就损你的阴德了么？你年岁颇大已经无所谓了，这损的阴德自然就落在你的子孙头上了”<br/>　　<br/>　　    贺三刀有些不信的说道：“像我这样的藏家国内可不少，难不成他们个个都损了阴德后代必定遭殃？”<br/>　　<br/>　　    “他们可能比你好点”王惊蛰指了指脚下，说道：“你这每年流通的老货交易太多了······”<br/>　　<br/>　　    贺三刀听到王惊蛰的话后就叹了口气，也没接着再问了，不过往后不到一年的时间里，宝瑞福就渐渐的不在做古董交易的中转了，逐渐变成了一栋真正的茶楼，贺三刀直接公告江湖，宝瑞福从此退出古董鉴赏评价界，喝茶可以聊风月也行，古董交易请免开尊口。<br/>　　<br/>　　    王惊蛰也不算危言耸听，他一进入到宝瑞福就发现这茶楼笼罩着一层淡淡的阴气了，说白了就是这楼跟个活死人墓似的，常年都有出土的老物件在茶楼里进进出出的，身为宝瑞福的老板贺三刀必然深受其害，他一把年纪了最多就是折寿而已，少活几年罢了，但损耗他的阳寿可能还不太够用，自然就贻害到他的子孙头上了。<br/>　　<br/>　　    果然一年之后，贺三刀的儿子就出了变故，差一点小命不保，而在这个时候王惊蛰如约赶到，依照着当初赊刀时留下的承诺，给他儿子又救了回来。<br/>　　<br/>　　    王惊蛰从宝瑞福出来后大概才十点左右，他想看看这据说是杨贵妃戴过的簪子到底有什么猫腻，于是从潘家园出来后就近找了个住的地方住了进去。<br/>　　<br/>　　    “哗啦”窗帘拉上，屋子里顿时漆黑一片，将那个红绸子放在了桌子上后拿掉符纸把簪子摆了出来，漆黑一点光亮都没有的空间里，碧绿的翡翠簪子上透着一点微光，光晕将整个簪子都包裹住了。<br/>　　<br/>　　    这一块玻璃种翡翠的等级绝对是最拔尖的，就算不考虑它的历史年代，光是簪子本身如果拿出去拍卖的话，恐怕少千万都拍不下来，而且要说做工的话现代人已经很难打造的这么完美了，这东西拿出去当聘礼，面子肯定是够了。<br/>　　<br/>　　    “如果你真能冒出个杨玉环的话，我那把刀可真就没白送了·······”王惊蛰眼巴巴的盯着簪子说道。<br/>　　<br/>　　    时间很快过去，午夜子时钟声敲响，王惊蛰的眼珠子就顶在了翡翠玉簪上，瞬间一股阴凉的气息就从簪子里涌出，平地刮起一股阴风，吹的窗帘都猎猎作响。<br/>　　<br/>　　    王惊蛰眯着眼睛“咦”了一声，按照正常的出场状况来推断的话，这簪子应该是被一道阴魂给寄身了，但要说是杨贵妃的魂魄他肯定是不信的。<br/>　　<br/>　　    贵妃可是被皇朝气运加身的，如果不是特殊的原因，是断然不会成为什么孤魂野鬼的。<br/>　　<br/>　　    玉簪上的光晕突然暗淡下去，王惊蛰的眼神慢慢的朝着镜子那边飘了过去，然后一阵极其让人头皮发麻和惊恐的一幕就出现了，贴在墙上的长条镜子里，一个穿着金色宫装的身影正在翩翩起舞，身影较为模糊，勉强能看出是个女人身来。<br/>　　<br/>　　    “天阙沉沉夜未央，碧云仙曲舞霓裳，一声玉笛向空尽，月满骊山宫漏长······”房间里顿时传来一阵空灵的曲调声，那音调听起来非常的晦涩难懂，有点类似于老陕腔发出的声调。<br/>　　<br/>　　    古长安一代人说话，差不多就是这么文绉绉的，那个时期的官方用语，也就是我们现在所说的普通话，可并不是靠近北方的口音。<br/>　　<br/>　　    虽然看不清镜子里人的模样，但却能看清她起舞的风姿，和那颇有些丰腴的身材。<br/>　　<br/>　　    古人都说环肥燕瘦，杨玉环以胖为美，粗略的估算了下镜子里的人从体型上判断的话差不多能有一百二十来斤左右了，谈不上有多胖吧但肯定算是丰满的，这镜子里人影在跳舞的时候就显得有些风韵犹存的感觉了，属实挺诱惑人的。<br/>　　<br/>　　    王惊蛰看了半天，镜子里的人影除了跳舞外，别的什么也没干，这要是孤魂野鬼的话恐怕现在早就扑出来了，这玩意儿看来没有什么害人之心，似乎只是跳了一曲而已。<br/>　　<br/>　　    王惊蛰伸手拿起桌子上的玉簪，这时候上面已经没有那种阴凉的感觉了，稍微有些冷冰冰的，反倒握在手里还感觉挺舒服的。<br/>　　<br/>　　    王惊蛰端详了半天哦了一声后，点头说道：“难怪呢，原来是一块死玉，你可能还真是那杨贵妃的一捋残魂啊”<br/>　　<br/>　　    他看明白了，这玉簪子早先曾经是一块死玉，然后一直被杨玉环戴在身上，久而久之时间一长了后，她的生气就被抽出来不少融入到了簪子里，等杨玉环身死之后，这簪子天长日久就渐渐的生出了一道残魂出来。<br/>　　<br/>　　    玉这东西除了质地和等级之分以外也分活玉和死玉，俗话说人养玉三年，玉养人一生。<br/>　　<br/>　　    玉都是有灵性的，一块上好的活玉被人戴久了以后，渐渐的就人玉相融了，你润养活玉几年后，这玉也能同样的滋润你，至少也会让人感觉心旷神怡头脑清醒，也有驱邪避凶的作用。<br/>　　<br/>　　    如果是死玉的话就正好相反了。<br/>　　<br/>　　    以前曾经有人捡到过一块玉佩，看其挺光鲜亮丽的质地似乎不错就戴在了身上，但没想到几年之后这人的身体越来越差，大病没有小病不断，多年后人就渐渐的瘦成了皮包骨头，后来碰到有懂行的人告诉他，是你戴的这块玉不对，它把你的精血和生气都给抽走了。<br/>　　<br/>　　    人的精气神没了，那玉反倒是愈发的光亮和耀眼了。<br/>　　<br/>　　    果然，那人把玉给扔了以后又养了一段时间，渐渐的身体又重新恢复了过来，如果这玉他还一直戴着的话，恐怕早晚有一天都得被祸害死了。<br/>　　<br/>　　    所以，玉这东西选好了是不错，没选对的话戴上一块死玉，那可就完了。<br/>　　<br/>　　    王惊蛰拿起那道符纸放在了玉簪上，镜中跳舞的人突然瞬间就消失了，这玉簪瞬间又变得有些冰凉刺骨了起来。<br/>　　<br/>　　    “这东西当聘礼的话，档次应该够了·······”<br/>　　<br/>　　    于此同时，距离王惊蛰所住的地方不到百米左右，有道人影正快速的朝这边走来，眼睛一直都在盯着一扇窗户。<br/>　　<br/>　　    “好重的阴气！”
