        听着她把一肚子的苦水都吐完后，他才算得上是真正意义上的静下心来去思考自己和她之间的事情。
<br/>
<br/>        这段日子，他一直在封闭自己的感情，同时也封闭了对筱颖的爱。
<br/>
<br/>        他总是装作漠不关心她，心里却总是在一遍又一遍地思念着她。
<br/>
<br/>        明知道她不爱他，哪怕她被阿景抛弃了一次又一次，却又因为他爱她而无法自拔，无法对她彻底放手。
<br/>
<br/>        所以，他才觉得是他自己在犯贱。
<br/>
<br/>        但此刻，他豁然开朗了。
<br/>
<br/>        原来他爱的筱颖，也是爱他的，对不对？
<br/>
<br/>        傅夜沉平复了心情之后，温柔地轻哄：“筱颖，我的宝贝筱颖……”
<br/>
<br/>        “呜——我才不是你的宝贝！你对我多狠的心呐！”童筱颖依旧抽泣着，泪水沾湿了鬓角的发丝，像黑色的丝带黏在她的脸颊上，凌乱又楚楚可怜。
<br/>
<br/>        傅夜沉轻轻地抓住她那只放在门柄上的小手，温柔地往自己刚毅的胸膛前拉近，另一只手则情不自禁地捧住了她的脸颊。
<br/>
<br/>        他心疼地蹙着眉头，指腹爱怜地抹去她眼角的泪水，扬嗓低沉地说：“筱颖，对不起，我跟你道歉。”
<br/>
<br/>        童筱颖吸了吸鼻子，定定地瞅着傅夜沉的俊脸。
<br/>
<br/>        她那双因为充盈了泪水而变得格外水灵的眸子，仿佛正在十分倔强地告诉他，她现在很伤心、很生气！不想接受他的任何道歉！！
<br/>
<br/>        她在跟他使小孩子气！
<br/>
<br/>        嗯，到底是被他宠过的小女人。
<br/>
<br/>        傅夜沉忍不住轻笑出声，刚刚还眉心紧蹙的他，此刻脸部的线条要柔和了许多。
<br/>
<br/>        “你还笑！一点诚意也没有！”童筱颖瘪了瘪唇，眨了一下，有点儿哭肿了的眼睛。
<br/>
<br/>        “嗯，可你刚刚还打了我一巴掌来着……”
<br/>
<br/>        “那是因为你做得太过分了！”她跟他较真。
<br/>
<br/>        说真的，他长这么大，打他脸的女人，还真就只有她童筱颖一个人。
<br/>
<br/>        “筱颖，对不起。”他忽然敛了笑容，一本正经地说。
<br/>
<br/>        随之他又拉起她的双手，移到唇畔，微微垂帘，在她的十指上落下一个又一个轻柔的吻。
<br/>
<br/>        童筱颖心头震颤，别开眼，下意识想要抽回自己的手。
<br/>
<br/>        他有所察觉，手掌略微施力，不给她任何逃脱的机会。
<br/>
<br/>        “十指连心，你的心已经被我吃了，肯定是要不回去了，所以，我不会放你走。”傅夜沉抬眸凝视着她的眼，嘴角微扬。
<br/>
<br/>        “大哥……”童筱颖对上傅夜沉的黑眸，面对他的反常，心里因为有点儿狐疑，于是试探性地问道，“你……这是相信我了的话吗？”
<br/>
<br/>        “从今以后，只要是我的筱颖所说的话，我都相信。”他落落大方地给出了允诺。
<br/>
<br/>        “你不怕我骗你？”她双眼直勾勾地看着他，不想放过他俊脸上意思的表情变化。
<br/>
<br/>        她像是在屏气凝神，等待他接下来的答案。
<br/>
<br/>        “当然怕！”傅夜沉抬眼，深邃而幽静地望向她眼底，嗓音略哑地问，“那你爱我吗？”
<br/>
<br/>        他只问这最后一次。
<br/>
<br/>        认定了，那就是一辈子。
<br/>
<br/>        “爱！”童筱颖是哭着笑，却是发自内心的告白。
<br/>
<br/>        单凭这一个字，他瞬间有种被她一箭命中红心的怦然悸动。
<br/>
<br/>        好像身边所有的一切，都逐渐褪色成无关紧要的背景，世界仿佛小得只剩下她和他，而他们又大得仿佛已经是全世界。
<br/>
<br/>        傅夜沉双掌捧起童筱颖的脸，垂眼，温热的嘴唇逼近，停在她软嫩的双唇前，欲吻未吻，声音透磁，充满诱惑：“筱颖，嫁给我……”
<br/>
<br/>        童筱颖未回答，双眼微慌地微微瞠大，呼吸短而急促，双颊像有火在烧。
<br/>
<br/>        她是愿意的，可是滚到舌尖上的话却迟迟说不出口。
<br/>
<br/>        因为……
<br/>
<br/>        她有些激动得说不出话来……
<br/>
<br/>        这一次，她没有一丝犹豫。
<br/>
<br/>        他凝望着眼前她那张越来越红的小脸，果决地消弭最后那一点距离，不等她答复，他便已经得到了他想要的答案。
<br/>
<br/>        再次轻柔地覆上她的唇，他对她吻得格外怜惜，如获珍宝般小心翼翼地呵护着……
<br/>
<br/>        跟以前充满情欲的吻不一样，这个吻，没有欲望，只有美好的感情，绵绵密密地绕上他的心头。
<br/>
<br/>        过了许久，他缓缓撤离，深深地凝视着她，说出了最长情地告白：“筱颖，我爱你。”
<br/>
<br/>        童筱颖扬起粉唇，突然从副驾驶座上起身，往傅夜沉的身上扑了过去。
<br/>
<br/>        她的身体很软，也很香，此刻的性子，就像个快乐得无忧无虑的孩子一般，抱着他的脖子，赖在他怀里，声音软糯软糯的，像是在跟他撒娇，也像是在跟他表白：“我也爱大哥，超爱超爱大哥！”
