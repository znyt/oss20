
<br/>　　<br/>　　    黄昏临近，天目山上游人渐少，大部分的游客都在天黑前下了山，只有一小部分打算留在山上过夜，有不少开始从附近的商家租用帐篷和野炊用具，据说天还没亮，大概两三点左右的时候，在天目山上看日出，景色是挺不错的。<br/>　　<br/>　　    长得像权志龙的青年，背着帐篷眼睛放光地盯着李红拂，兴致冲冲地说道：“我听说从仙人顶的背山下去，那里有一条河，河床上都是平坦的鹅卵石，咱们可以去那里扎营，然后来个野外烧烤，路程也就半个小时左右，明天早上三点咱们再起来重新回到山顶，正好看日出，红拂你觉得我这建议怎么样？”<br/>　　<br/>　　    高挑冷淡女李红拂俏脸平淡地点了点头，另外几人都雀跃着说道：“建议不错，快点走吧，我们找个好点扎营的地方，免得被人给占了”<br/>　　<br/>　　    权志龙摆手说道：“那个地方很少有人知道，我也是听一个同学说的，除了本地的村民外面来的游客都不晓得，搞不好就会只有我们一伙人的”<br/>　　<br/>　　    王惊蛰哀叹了一声，这哥们为了泡妞脑子已经都被驴给踢偏了，他真想问问大哥你是什么胆子啊，一行人就七八个，居然敢去深山老林里夜宿露营，你但凡有点野外常识就该明白，夜间的山林轻易是没人敢进去的，就不说什么飞禽走兽的了，草丛里就是趴着条蛇咬你一口，你都未必能防得住啊，真要是碰见五步倒青竹蛇这一种的，估计都等不到救援队把你带下山，就得死在天目山上了。<br/>　　<br/>　　    仙人顶山顶后山，是一片人迹罕至的山林地带，只有一条狭小的山路，两边是都没人踏足的林子，鸟语花香人迹罕至地漫步在林间，确实别有一番情调，徐闯和王清雪手拉着手蹦蹦跳跳，权志龙跟在李红拂表现欲十分强势地展现着自己的见闻，那个赵文亮则是在另外一个女生面前大献殷勤，至于剩下的两个女人虽然闲着呢，但眼神也一点都没有望向王惊蛰的欲望。<br/>　　<br/>　　    “拿着这个，走路的时候敲着点”王惊蛰从地上捡起一根树枝子，递给徐闯。<br/>　　<br/>　　    “干什么的啊？”<br/>　　<br/>　　    王惊蛰淡淡地说道：“打草惊蛇用的……”<br/>　　<br/>　　    半个多小时后，仙人顶山下的一处山坳里，一条清澈的小河蜿蜒而过，河床两边平坦开阔，单从景色上来讲，这里确实挺诗情画意的，就连那个冷淡李红拂的脸上都忍不住地露出了一抹欣喜的味道，暗中观察的权志龙一看，当即就澎湃了，知道自己这个建议，好像把女神给哄开心了。<br/>　　<br/>　　    这帮人在安营扎寨摆弄烧烤架子的时候，王惊蛰背着手溜达在小河边，顺着河岸来回地走了好几趟，经常都是仰着脑袋打量着四周。<br/>　　<br/>　　    王清雪收拾着东西，一边皱眉跟徐闯说道：“你那个朋友也太不懂事了吧，路上什么也不做，吃喝都是捡现成的，晚上了大家露营都在干活，他却跑一边放风去了，怎么的？害怕有土匪来把我们抢走当压寨夫人啊？”<br/>　　<br/>　　    “呃……”徐闯尴尬地咳嗽了一声，努力地整出了一句蹩脚的借口：“他有点特性，不太合群”<br/>　　<br/>　　    权志龙撇嘴说道：“不合群他倒是也别来啊，凑个什么热闹呢”<br/>　　<br/>　　    人吧其实都这样，在哪都有排外的心思，王惊蛰本来就不属于他们这一波人里的，然后从头到尾不与他们交流也不出力，自然谁看着他都烦了，就连徐闯都觉得带他来有点后悔了，这哥们太不给力了。<br/>　　<br/>　　    王惊蛰顺着河岸溜达着往回走，掐着手指不解地嘀咕道：“这地方看起来也不像是什么凶邪之地啊……”<br/>　　<br/>　　    晚间，天黑以后，一伙人围着篝火烤着烧烤，王惊蛰独自一人坐在一块石头上抽着烟，那边欢声笑语的他也不太感兴趣。<br/>　　<br/>　　    李红拂扭头朝他这里望了几眼，寻思了下后拿起一把肉串还有两瓶啤酒，就往他这边走了过来，权志龙和徐闯还有赵文亮等人看见这一幕，都挺诧异的。<br/>　　<br/>　　    李红拂这人的性子就跟冰川似的，跟她相熟的人说话，说三句她可能就回几个字，轻易不和人交谈，就更别提她主动跟谁说话了。<br/>　　<br/>　　    权志龙抽搐着嘴角，说道：“李大美女是不是都看不过去了，打算去损他几句啊？”<br/>　　<br/>　　    “晚上不吃一点东西么？”李红拂走过去后递给王惊蛰一瓶啤酒还有手里的肉串，然后挑了他对面一块干净的石头歪着腿坐了下来。<br/>　　<br/>　　    “你怎么也会和这些人一起胡闹呢，晚间夜宿山林得是多大的胆子啊”王惊蛰接过她手里的东西，挺粗鲁的用牙直接把啤酒给起开了，然后仰头喝了一大口。<br/>　　<br/>　　    李红拂略微一愣，然后就笑了：“你好像把我和他们区别对待了？”<br/>　　<br/>　　    “不是么？”<br/>　　<br/>　　    “你是怎么看出来的”<br/>　　<br/>　　    “你是怎么看出我来的，我就是怎么看出你来的”王惊蛰撸着肉串，含糊着嚼着说道：“晚上精神点吧，别睡得太死了”<br/>　　<br/>　　    李红拂捋了下耳边的长发，说道：“他们不是那些不知深浅的驴友，自然也不会满山乱窜的”<br/>　　<br/>　　    这些年，经常都有新闻说有驴友进山迷路的，遇险的，甚至还有把命给丢了的，一般人都以为这些驴友是在穿山过林的途中出现了意外，但这只是一部分的状况，至少还有一部分是因为在山里撞邪了，山间多精怪，邪门的地方也不少，就是碰到鬼打墙走不出去了，也能把人给活活地耗死在山里，就别提如果是撞邪了的，一脚都有可能踩空掉下山坡去。<br/>　　<br/>　　    晚间的山林，充满了太多未知的因素，成群结队进山的人多火气旺可能还好点，要是就三五个人话，你等于是把自己的小命当成儿戏了。<br/>　　<br/>　　    李红拂和王惊蛰聊了几句，两人基本就是在互相试探着，但彼此都是猴精的人，谁也没探出什么底细，聊了一会后话题就结束了。<br/>　　<br/>　　    此时天色已然很晚了。<br/>　　<br/>　　    徐闯收拾完后，贱嗖嗖的跟王清雪挤咕着眼睛，说道：“亲爱的，咱俩手牵着手去林子里逛一会啊？”<br/>　　<br/>　　    王清雪看着黑漆漆的树林，握着小手说道：“我害怕……”<br/>　　<br/>　　    “啪”徐闯一把抓着她的手腕，拉着她说道：“哎呀，你怕什么啊，咱们也不走远，就找个没人的地方安静的呆一会，畅谈下心扉就行了”<br/>　　<br/>　　    “我是怕你啊”王清雪咬着嘴唇小声说道。<br/>　　<br/>　　    不远处，赵文亮跟另一个女生说道：“小雯啊，饭后没什么事了，也不能睡这么早，咱俩去散散步消化消化呗，我跟你说女人吃完饭不能坐着不动，容易发胖”<br/>　　<br/>　　    小雯眨着大眼睛笑道：“我为什么要跟你去啊？”<br/>　　<br/>　　    赵文亮瞥了眼王惊蛰那边，说道：“那你就只有跟他去了，你觉得跟他走在一起，能迈得开步子么？”<br/>　　<br/>　　    “算了，还真就是得和你了，没办法，狼多肉少，就三个男的我们的可选择性太低了……”<br/>　　<br/>　　    赵文亮和小雯，徐闯跟王清雪，两个泛着滚滚骚气的男生，牵着姑娘的手迈着小碎步就朝着小树林里钻了进去，权志龙看的眼神直冒火，舔着发干的嘴唇臊眉耷眼的斜了着李红拂十分想和她说一句，老妹咱俩也去小树林里逛逛呗，但话到嘴边他绝对不敢出口，身为理工学院的校花，这两年谁也没能把李红拂给摘走，反倒是被她给撅折了不少打她注意的男人。<br/>　　<br/>　　    王惊蛰看见两对男女要钻树林，他皱着眉头快步赶了过去，轻声说道：“徐闯，别往深了去，在边缘呆着就行了，如果看见什么你就招呼一声”<br/>　　<br/>　　    “哎呀，知道知道了，就散散步我还能跑到深山老林里去啊？”徐闯不耐烦地摆了摆手，拽着王清雪的胳膊就往里走。<br/>　　<br/>　　    这时候别说是王惊蛰了，就是他爹妈过来拦着，徐闯可能都得急眼，箭在裆上不得不发，花前月下，肯定不如花前日下。
