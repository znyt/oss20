李玉兰太猛了，在孔真儿还没有任何准备的时候直接将她拖拽到了回廊里，很快回廊里就挤满了人。

<br/>        孔真儿现在能求助的只有夏振国了。
<br/>

<br/>
<br/>        夏振国伸手去抱孔真儿。

<br/>
<br/>        说着李玉兰又冲上前拽着孔真儿就“啪啪”扇了两巴掌，“小贱人，我打死你！”

<br/>        “够了李玉兰，你给我滚！”

<br/>        夏振国伸手一挥，直接将李玉兰挥到了墙壁上。

<br/>        砰一声，李玉兰是硬生生撞上去的，她疼的抽吸，一下子跌坐在了地毯上。

<br/>        痛。
<br/>
<br/>        好痛啊。

<br/>        李玉兰惊恐万分，她现在是大龄产妇，都需要靠安胎药来养胎的，现在身下热血滚滚，她知道自己流产了。

<br/>        完了，她的儿子没有了！

<br/>        什么？

<br/>        她怀孕了？
<br/>
<br/>        怀了他的儿子？

<br/>
<br/>        夏振国抱着李玉兰消失了，围观的吃瓜群众也慢慢消失了，孔真儿却僵在了原地，她万万没有想到李玉兰竟然怀孕了，现在还流产了。
