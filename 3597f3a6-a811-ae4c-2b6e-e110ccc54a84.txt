　　　　首阳山，人族不畏生死，誓死灭杀妖族，震撼了所有人。
<br/>　　高空之上，帝俊与东皇太一的脸色变得越来越阴沉了，他们妖族的妖兵被人族武者屠戮，妖族陨落的强者甚至是人族陨落的数倍，那岂不是在说，妖族不如人族吗？
<br/>　　虽然这些陨落的修士他们并没有看在眼中，但被人族武者屠杀，那是在打他们的脸，他们脸色能好看才怪。
<br/>　　在他们的预料之中，人族应该毫无反抗之力，被妖族无情屠杀才对，可是，眼前的情况好像不太对劲啊，反而倒像是妖族再被人族屠杀。
<br/>　　一旁的白泽偷偷看见帝俊与太一的脸色渐渐的变得越来越不对心中“咯噔”一声，顿时怒火中烧，对着身后的众多妖王大声道，
<br/>　　“你们还在等什么，还不跟我杀上去！”
<br/>　　“吼！”
<br/>　　白泽取出自己的兵器，冲了上去，其他妖王亦紧紧跟随而上。
<br/>　　“人族完了！连妖神白泽都出手了。”
<br/>　　“人族在劫难逃了。”
<br/>　　无数关注此战的修士见到这一幕，纷纷心中暗道。
<br/>　　眼看人族就快要被妖族强者屠杀，人族真的要被灭族，真的无力回天吗？
<br/>　　“谁敢杀我人族！欺我人族者，杀！无！赦！”就在此时，一声大喝，响彻云霄，轰动苍穹。
<br/>　　“轰隆！”
<br/>　　一股强横至极的威压从人族祖地压了过来，紧接着，一个身材魁梧，满脸胡渣的壮汉提着一柄战刀从人祖殿中走了出来。他全身气势锋芒，眼神锐利，此刻充满杀气的看着妖族众妖。
<br/>　　“敢动我人族者，死！”
<br/>　　说着，他一挥手中战刀，向着一众妖王砍去，刀光闪烁，划破天际，斩破虚空，带着一股锐利之势斩在妖王身上，妖王就如同一张纸一般，瞬间破碎。
<br/>　　仅仅一刀，就有十个妖王死亡。
<br/>　　恐怖如斯！
<br/>　　“什么？！大罗金仙高手，人族竟然有大罗金仙高手？”
<br/>　　“这还是后天种族吗，仅仅发展万年，就有堪比大罗金仙的高手，真是恐怖！”
<br/>　　“是啊，他还不是一般的大罗金仙。”
<br/>　　一些人看到这一幕，眼珠子都快要掉出来了，可见他们是有多么惊讶。看那人的气势，就知道不是一般的大罗高手，能够一刀斩杀数个妖王，岂能简单？
<br/>　　高空上的帝俊、太一眼中闪过一丝惊讶，看来的确是小看人族了，随即就又恢复了淡然的姿态，大罗金仙又如何，在他们眼中一样不够看。
<br/>　　只不过，他们现在不会再以普通的眼光看待人族了，人族的发展潜力的确十分巨大，仅仅发展万年，就有堪比大罗的高手，这是在其他种族见不到的。
<br/>　　两人对视了一眼，在这一刻，他们心中下定决心，一定要将人族灭族，要不然，将来人族一定会是一个巨大的威胁。
<br/>　　“还在愣着干什么，给我杀！”
<br/>　　妖神白泽眼神阴沉，对着众妖王大喝一声。
<br/>　　众妖王这才反应过来，连忙挥起手中兵器，杀向人族高手。
<br/>　　“妖族小儿，安敢如此放肆！”
<br/>　　就在此时，又是一声大喝，紧接着一阵恐怖威压出现，从人祖殿中再次走出一位大罗高手。
<br/>　　“想要灭我人族，你妖族还不行！”
<br/>　　紧接着，又是一道人影从人祖殿中走出，气势一样是大罗金仙。
<br/>　　“踏踏踏……”
<br/>　　一阵脚步声响起，之后，在众人目瞪口呆之下，从人祖殿中接连走出数百个人族武者，他们的修为都在天地法相境，也就是相当于大罗金仙。
<br/>　　这群人族走出人祖殿后，将目光紧盯着妖族众妖王，眼中闪过仇恨之色，那是一种刻骨铭心的仇恨，人族与妖族，注定不能两存。
<br/>　　“嘶！怎么可能，人族竟然有如此多大罗高手。”
<br/>　　“这……这怎么可能？我一定是眼花了，看错了！”
<br/>　　“人族，竟有如此多大罗高手？！”
<br/>　　“看来，我们所有人都小看了这人族，谁知道人族竟然恐怖如斯！谁能想到，一个后天种族，仅仅发展近万年时间，就有如此多大罗金仙，这还是我认识的那种后天种族吗？”
<br/>　　“这个世界一定是疯了……”
<br/>　　“……”
<br/>　　这一刻，天地间没人能够平静下来，看到眼前这一幕，简直打破了他们对后天种族的看法，他们对“人”的定义，需要用新的目光去看待。
<br/>　　人族怎么可能有如此多大罗高手？
<br/>　　恐怖啊，短短时间内，人族竟然积累出如此多高手，真是恐怖如斯！
<br/>　　怎能不令人震惊？
<br/>　　混沌天外，娲皇宫中，女娲娘娘亦是一脸吃惊的看着人族涌现出来的高手，就连她这个人族圣母，都不知道人族竟会有如此多高手。这还是我亲手创造的种族吗？
<br/>　　这一刻，女娲娘娘心中更加迷茫了，人族高手多，她固然高兴，但这也就意味着，人族与妖族更加不能共存了，人族与妖族今后要死去更多，面对自己最亲的两方势力，不论哪一方势力败亡，她心中都不好受。
<br/>　　此刻，她的心乱了，她不知道自己该怎么办？
<br/>　　同时，她也隐隐有些后悔，后悔当初不该那般对待人族，不该将他们抛弃。可是，她是圣人，又怎能后悔呢？有些事情，没有后悔，既然做了，就该去独自面对。
<br/>　　西方灵山，接引与准提圣人看着出现的人族高手，亦是大吃一惊。
<br/>　　“师兄，东方真是人杰地灵啊，谁能想到，女娲当初随手创造出来小小人族，竟会有如此多高手。唉！我西方何时能够出现如此盛状？”准提圣人愁眉苦脸道。
<br/>　　“阿弥陀佛！师弟不必担忧，我西方迟早会大兴的。如今东方正处于大战之中，待大劫已过，便是我西方大兴之机。”接引淡淡道。
<br/>　　“善哉！”
<br/>　　昆仑山，元始一脸淡然，好似早知如此，脸上并没有露出任何吃惊的状态。太清老子亦是一脸早知如此的表情，他在人族传道立教千年，最了解人族的状况，如今这种情况，他早就有所预料。通天则是在哈哈大笑。
<br/>　　“哈哈！看到妖族小儿吃瘪，我就高兴！不过，人族还真了不起！”
<br/>　　且不去管其他人的看法，却是首阳山上，一众人族高手一出现，便立刻盯住妖族众妖王，二话不说，直接抄起兵器便杀了上去。人族与妖族之间的血恨，可不是说说就能解决的。
<br/>　　妖族将人族当成食物，肆无忌惮的捕杀人族，无数人族死在妖族口下，这是血与泪的仇恨，这是灭族之仇恨，不是那般轻易就能够说清楚的。
<br/>　　妖族一日不死，人族便一日不会罢手。
<br/>　　此刻看着众妖王竟然想屠杀人族高手，他们怎会如妖族所愿？同等级的战斗，他们怕过谁？之前之所以不出来，便是想让人族法相境以下的对付妖族大罗以下的强者，每个等级有每个等级的打法，如今既然妖族妖王已经出现了，那么他们亦不便躲藏，现身与之一战。
<br/>　　一众人族高手开始与妖族妖王大战起来。人族天地法相境高手，能够施展出自己的法相，法相高达数万丈，有的是熊熊天火，有的是通天巨木，有的是大江大河，有的是风雨雷电，还有的是山峰巨石，更有各种神兽、天妖的形态……
<br/>　　法相一出，谁与争锋？
