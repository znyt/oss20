        少年走到男子的面前后，男子畏畏缩缩地将怀里的黑色皮箱呈上。
<br/>
<br/>        “求、求你……饶、饶我一命，求求你……我、我还不想死！”男子嘴唇打颤，支支吾吾地求饶。
<br/>
<br/>        少年伸手接过黑色皮箱后，持枪的手缓缓地抬起，黑色的枪口，直接对准了这个求饶的男子的脑门。
<br/>
<br/>        就在这时，不远处响起了警车的鸣笛声。
<br/>
<br/>        “嘭”地一声，枪声响起。
<br/>
<br/>        连星儿吓得闭上了眼睛。
<br/>
<br/>        当连星儿再睁眼时，男子双目呆滞地瘫坐在地上，一只腹部上血肉模糊的麻雀和刚刚那只黑色的手枪掉在了他身前的地面上。
<br/>
<br/>        而刚刚那个少年，却不知所踪。
<br/>
<br/>        “我、我还没死……”男子自言自语地说。
<br/>
<br/>        连星儿看着男子，对于刚刚所发生的事情，彻底傻眼了。
<br/>
<br/>        那个少年是——郎、安？！
<br/>
<br/>        明司寒接到电话后，赶到了警局里。
<br/>
<br/>        男子被警察拘留后，连星儿录了口供，跟着明司寒走出了警察局。
<br/>
<br/>        她不知道自己为什么会对那个少年的眼睛那么敏感，明明她只见过郎安一面，却莫名其妙地记住了他的眼睛。
<br/>
<br/>        回去的路上，明司寒见连星儿一言不发，有些担忧地问道：“星儿，你没事吧？”
<br/>
<br/>        “没事。”连星儿微微摇了摇头。
<br/>
<br/>        警局里，警察忙得不可开交，正在核对他们记录的口供。
<br/>
<br/>        对于那个少年的事情，那个持枪发疯的男子没提，连星儿也破天荒地没有主动说出口。
<br/>
<br/>        至于男子臂弯上的伤，警察问及时，男子只说是自己开枪弄的，还说自己有双重人格，一个劲地在那胡言乱语。
<br/>
<br/>        检验科的检查报告显示，那只手枪上只有男子一个人的指纹，枪里的子弹已经全部用完，最后一发子弹，在那只麻雀的尸体里。
<br/>
<br/>        而连星儿的口供里，只说她吓傻了，闭上了眼睛，什么也没看到，什么也不知道。
<br/>
<br/>        更巧的是，那段路边的监控全都坏了，而行人都光顾着逃命去了，没有人去留意过他们俩当时是什么情形。
<br/>
<br/>        具体的真实场面是什么，警察们一无所知。
<br/>
<br/>        谁都不知道，那个发疯的男子竟然有意替那少年作掩护。
<br/>
<br/>        连星儿并不知道警局里是个什么情况，也不知道那个发疯了的男子是否有把那少年供出来。
<br/>
<br/>        只知道自己现在的心情有些慌乱。
<br/>
<br/>        “我送你回去吧！”明司寒的声音再次在耳边响起。
<br/>
<br/>        连星儿这才彻底回过神来，微微仰头看向明司寒：“要不，你先回去吧！我想去我外公家。”
<br/>
<br/>        “我送你去。”明司寒皱了下眉头。
<br/>
<br/>        连星儿强颜欢笑道：“不用了！我叫了家里的司机过来。”顿了一下，又向明司寒挥了挥手，“那我先走了。”
<br/>
<br/>        “星儿！”明司寒唤了一声，连星儿却疾步跑开了。
<br/>
<br/>        看着连星儿离去的背影，明司寒没有去追，反倒是不安地皱起了眉头。
<br/>
<br/>        不知何时，裴萱妍走到了明司寒的身后，顺着他的目光望向前方连星儿的背影。
<br/>
<br/>        “奇怪，那边的人怎么会救连星儿。”裴萱妍不可思议地说道。
<br/>
<br/>        明司寒微微偏头，不解地看着裴萱妍：“你怎么在这？”
<br/>
<br/>        “我接到通知了啊！说今天街上不安全，而不安全的地方又在你的活动范围内，所以我就来了。”裴萱妍淡然道。
<br/>
<br/>        明司寒皱了下眉头，反问：“起初在星儿身上到底发生了什么事？”
<br/>
<br/>        “这得怪你的那个小未婚妻多管闲事啊！”裴萱妍不禁双手抱臂，接着说道，“你的小未婚妻撂倒那个男人后，因此而激怒了那个男人，那男人想掐死你的小未婚妻，还未等我们出手相救，他们自己的人反倒是把你的小未婚妻给救了。”
<br/>
<br/>        “你们到底在做什么啊？”明司寒不禁皱起了眉头。
<br/>
<br/>        裴萱妍微微耸了耸肩：“这个，真的无可奉告噢！”
<br/>
<br/>        “星儿是不是也被牵连在其中？”明司寒质问道。
<br/>
<br/>        裴萱妍瘪了瘪唇，双手一摊，解释道：“傅夜沉不会让自己的女儿深陷其中的，只不过，刚刚那个面具男不仅救了连星儿，而且也没开枪杀那个男人。还真有些奇怪了！”
<br/>
<br/>        “面具男？什么面具男？”明司寒追问。
<br/>
<br/>        裴萱妍却会心一笑：“你若是加入我们，我就告诉你更详细的。”
<br/>
<br/>        “算了，我没兴趣了。”明司寒没好气地说。
<br/>
<br/>        裴萱妍勾了勾唇，只笑不语。
<br/>
<br/>        明司寒仍旧一脸不悦，他心里很担心连星儿，而自己又无能为力。
<br/>
<br/>        另一边，连星儿在大马路上下了车，刚走到通往实验室大门的那片树林的入口时，头顶忽然传来一声口哨。
