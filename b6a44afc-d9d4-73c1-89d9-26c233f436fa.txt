　　“父皇说过水能载舟的话，难道你忘了？”李承乾微微一笑，转身面对长乐说道：“水能载舟亦能覆舟，百姓为水，君为舟，所以你觉得大唐是谁的？天下百姓的还是李家的？”
<br/>　　“可是没有区别啊，我李家就是代表了大唐。”
<br/>　　“是的，眼下是我李家代表了大唐，可是我李家不等于大唐。”李承乾突然叹了口气，望着正在训练场上训练的球队说道：“就好像这支球队代表了皇室，但他们并不是皇室一样，如果他们表现的不好，那么我们就可以抄了他们。而我们呢，你想过没有？如果我们李家表现的不好，天下百姓会如何？会不会也抄了我们？”
<br/>　　“哥，你……你没事吧？为什么会突然想到这些？”长乐似乎有些被李承乾给吓到了，神情显的有些惊慌。
<br/>　　“我的意思是，有些事情我们可以做，有些事情我们不可以做，胜利固然重要，但千万不要作弊，我李家既然代表了大唐，那就拿出自己该有的气度来，明白么？”李承乾目不转睛的看着长乐，几乎是一字一顿的说道。
<br/>　　“哥，你，你都知道了？”
<br/>　　“朕是皇帝啊，你们的小动作如何能瞒得过朕？！”李承乾在长乐的头上拍了一下，就像小时候一般，有一种说不出的宠溺。
<br/>　　这丫头可能是求胜也切，也可能是想要表现自己，竟然消无声息的抓了其它几个球队的家里人作为威胁，虽然那些人都被关在一起地方每天好吃好喝，可这依旧是李承乾无法接受的。
<br/>　　或许放在以前他也会干出同样的事情，可是现在随着地位的改变，他的想法也在变。
<br/>　　球赛的输赢在他看来已经不是那么重要，重要的是精神和态度，敢于面对失败，敢于正视失败，只有这样才会被大众所接受。
<br/>　　而且这种事情还反应出一个人的品格，或许有人觉得犯一次错没有什么，但是不要忘记，有一就会有二，有二就会有三，尝试过不劳而获之后，人就会形成一种惯性，慢慢的一点点堕落下去。
<br/>　　……
<br/>　　时间飞快，转眼间上元节便已经过去，铺天盖地宣传了数月之久的长安杯终于拉开帷幕，整个长安几乎成了欢乐的海洋。
<br/>　　城外的四个马球场彩旗招展，锣鼓喧天，喧嚣的加油助威声直冲霄汉。
<br/>　　一支支来自全国各地的队伍举着属于自己的旗帜在引导员的引导下纷纷入场，在球场中间站立，同时也向四周前来参加开幕式的群众招手致意。
<br/>　　参加比赛的球队太多，一个球场根本就装不下这么些人，所以不得不四个球场同时打开，在约定的时间里，同时招开开幕式。
<br/>　　这样的场合李承乾自然是严格被限制外出的，甚至不但是他，就连他的几个妃子也都被手下劝回皇宫。
<br/>　　人实在是太多了，不说开幕式会场一票难求，就是长安城里面那也是人山人海。
<br/>　　旅店爆满，酒楼爆满，大街上人潮汹涌，接踵摩肩，左、右武候卫采用轮换制全员上街巡逻，长安县、万年县、京兆府派出所有衙役，在街上执勤。
<br/>　　可就算是这样，也依旧不断有全国各地的游客不断涌进长安，一度将宽敞的朱雀大街堵的水泄不通。
<br/>　　在这里不得不提的是宵禁，自从除夕那一天开始，长安城的宵禁就已经解除，事实上就算是不解除也没办法，进入长安的人太多了，想禁都禁不了。
<br/>　　李承乾正好也不喜欢宵禁，索性便干脆不提这件事情，爱咋咋地，谁提出来就特么让人去顶雷搞宵禁。结果满朝文武谁都不傻，李承乾不提，也就没有人主动站出来提，一个个就像没有这回事儿一样。
<br/>　　所以几乎是一夜之间，长安宵禁名存实亡，大牌子虽然立在那里，但是却再也没有人来驱赶半夜在街上蹓跶的人们。
<br/>　　没了宵禁之后，长安更是一片欢腾，不管白天还是晚上，大街上一概全是人，不管你什么时候出去上街都不用担心因为人少而发生什么抢劫之类的事情。
<br/>　　……
<br/>　　开慕式进行的很快，专项负责此事的官员草草说了几句之后直奔主题，在四个分会场各自展开一场表演赛。
<br/>　　其中第一分会场上，出场的是由张启元带队的皇室队伍，而与他对战的则是程家的队伍。
<br/>　　两只球队穿着红黄球衣，战马的身上挂着属于自己的球员号码，高举着战刀一样的球棍进入赛场。
<br/>　　此战必须要胜，必须要胜，身披黄色球衣的张启元不断给自己打气，因为他代表着皇帝，哪怕是拼了这条命，也绝对不可以输。
<br/>　　对面程咬金家的队伍和他们的那个家主一个德性，个个生的膀大腰圆，一脸横肉，看模样就知道一个个都是军队里的刺头。
<br/>　　裁判是从刑部那边调过来的一个书吏，这个没办法，李承乾想遍了大唐官方的各个机构，发现也就刑部能和裁判能贴点边，所以就只能用他们了。
<br/>　　……
<br/>　　随着两支队伍入场，四周看台上的观众很快安静了下来，一个个全都聚精会神的盯着赛场，看着两人支距离越来越近的队伍。
<br/>　　这原本是一场表演性质的比赛，但是因为张启元所代表的势力身份不同，比赛还没有开始气氛便显的紧张起来。
<br/>　　裁判似乎也感觉到了张启元的紧张，不由看了他一眼，以目光询问他是否有什么问题。
<br/>　　张启元摇了摇头，向对面的程家的队长抱了抱拳。
<br/>　　对面程家的愣货这个时候还记得来时家里那位姑奶奶说的话，让他们看情况打球，不要打的太狠，如果到时候让皇室丢了面子，回去非扒了他们的皮不可。
<br/>　　所以见到李启元向他抱拳，便也拱了拱手，看了一眼裁判大咧咧的说道：“老头儿，可以开始比赛了不？”
<br/>　　裁判没搭理程家的愣货，看了一眼张启元，又看看了四周，随后将手里手涂成红色的马球放于场地中间：“你们两支队伍谁先开球？”
