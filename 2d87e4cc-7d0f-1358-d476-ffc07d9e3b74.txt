陆寒霆伸手就拿过了手机，还一手搂着奶奶的老腰，将她往外推，然后将房门给关上了。
<br/>
<br/>        陆寒霆看到了视频里的夏夕绾，夏夕绾没有戴面纱，露出了那张让他心动不已的绝丽小脸，刚洗过澡她的小脸上还蒸蕴出了两抹浅红，那肌肤水当当的，一戳就能出水。
<br/>
<br/>        夏夕绾一看是陆寒霆来了，她将一张清丽绝色的小脸冷了下来，“奶奶呢？”
<br/>
<br/>        陆寒霆看着她那双黑漉泛着澄亮水光的眸子，声线低哑，“奶奶自己出去了。”
<br/>
<br/>        “既然奶奶走了，那我挂了。”夏夕绾伸出纤白的手指去挂视频。
<br/>
<br/>        陆寒霆看到了她新做的指甲，无比清纯轻甜的少女粉，是他喜欢的那一挂。
<br/>
<br/>        “你做了美甲？”他出声。
<br/>
<br/>        夏夕绾迅速收回了自己的小手，这时她就发现他的目光落在了她的身上。
<br/>
<br/>        夏夕绾穿着裸色睡裙，这种裸色衬着她白腻的肌肤，就像是镀了一层羊脂般的光泽，细细的肩带吊在她莹润的香肩上，还沾着湿漉水汽的长发飘散，黑的黑白的白，让人看了就忍不住心旌摇曳。
<br/>
<br/>        夏夕绾发现他的目光过于炙烈和肆意，她迅速凶巴巴道，“看什么看，再看就挖了你的眼睛！”
<br/>
<br/>        陆寒霆滚动了两下喉结，勾起薄唇，“我花钱的都不给我看？”
<br/>
<br/>        “不给！”
<br/>
<br/>        陆寒霆还想说话的，但这时传来了叶翎的声音，“绾绾，过来一下。”
<br/>
<br/>        “来了。”
<br/>
<br/>        夏夕绾伸手就将视频给挂了。
<br/>
<br/>        “…”
<br/>
<br/>        前后才说了几句话的陆寒霆眉心阴郁，他迅速给顾夜瑾发了一条叶翎是什么意思？
<br/>
<br/>        顾夜瑾哦，刚才忘了告诉你，按照叶翎的性格，你花了几百万买的东西肯定会给你看一眼，但是只是一眼，不能再多了。
<br/>
<br/>        陆寒霆…
<br/>
<br/>        顾夜瑾知足吧，你不是看了一眼么？
<br/>
<br/>        陆寒霆直接将手机给扔了，倒在了卧室的大床上。
<br/>
<br/>        他知道夏夕绾还在为那十二个亿的事情生气，夏妍妍曾经救过他，他心里有一分柔软给了那夜那个叫他大哥哥的女孩，所以他没有办法在夏夕绾的面前去坦述那个夜晚。
<br/>
<br/>        而且他知道不管他怎么说，她都会生气的。
<br/>
<br/>        当年他回去找那个女孩了，当夏妍妍拿着那块玉佩出现在他面前时，他竟然觉得浓浓的失望和厌恶。
<br/>
<br/>        夏妍妍是精心打扮去的，身上一股香水味，虽然她表面装得娇柔甜美，但她那双虚荣算计雀跃的双眼出卖了她。
<br/>
<br/>        夏妍妍完全不是他要的那个女孩。
<br/>
<br/>        有时候陆寒霆觉得那个夜晚就是一场梦，那个温软勇敢坚韧，激烈他活下去的女孩只是他的一场幻觉。
<br/>
<br/>        他想象里的女孩不是夏妍妍那样的。
<br/>
<br/>        他想要的女孩应该…是夏夕绾这样的。
<br/>
<br/>        那个女孩应该有着夏夕绾那双黑漉澄亮的翦瞳，有着夏夕绾那样清丽绝色的容颜，有着夏夕绾身上…那股清甜的少女体香。
