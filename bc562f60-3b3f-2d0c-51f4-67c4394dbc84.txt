　　第481章
<br/>　　李行哉的声音并算不上大，众人耳边却同时嗡地一声，久久不肯散去。随之，一股寒意从心底泛起，瞬时间传遍了全身。
<br/>　　如崔相、尉迟离、李乐天等脑子快的人，已经咂摸出李行哉话中的意思。他们目光环视四周，黄沙城的城墙浮现在眼前，四面城墙像同时围了过来，压得人喘不过气来。
<br/>　　黄沙城太小了，因为连年战乱的缘故，这里已经处于半废弃状态。也就是程大雷的蛤蟆军抵达并州后，才从李星手里夺过了这座城池，把它当做自己的大本营。
<br/>　　如今，黄沙城涌入了七八万帝国军，城内所有房屋都已经被涌进的士兵占据，更多人只能在大街上扎营。而因为想打程大雷一个措手不及，又要瞒着李星，所以，目前许多粮草物资还未运来。
<br/>　　“不，不可能，凭程大雷那点兵力，怎么能打得下黄沙城。”尉迟离坚定道。
<br/>　　“你……你们…………”
<br/>　　李行哉躺在担架上，因为失血表情有些扭曲，脸色越发苍白。
<br/>　　“快，快抬殿下去休息。”崔相道。
<br/>　　“你们呐……”
<br/>　　李行哉没来得及说什么，最后只来得及发出一句长长的叹息。
<br/>　　他虽然被抬走了，但诸人心里还在琢磨他的话。
<br/>　　帝国的大人物商议了这一战，既占据了程大雷的大本营，又狠狠得削弱了李星的力量，堪称一石二鸟。所以大家都觉得，这条计策是成功的，是完美的，是螳螂捕蝉，黄雀在后。
<br/>　　可是，黄雀之后，未必没有举起弹弓的顽童。
<br/>　　“报！”
<br/>　　一匹快马跑到这里，斥候从马上跳下来，焦急道：“贼兵在十里外扎营，目前目的不明。”
<br/>　　“哼！”李乐天狠狠的哼了一声：“贼人若敢来，我就会一他，看他怎么将我们赶尽杀绝。”
<br/>　　诸人登上了城头，在李乐天的命令下，黄沙城进入防御状态。只要程大雷敢打进来，必然会遭遇帝国军的猛烈攻击。
<br/>　　程大雷没有打。
<br/>　　蛤蟆军在城北十里处扎营，却选择按兵不同，站立在城头，能看到敌军正埋锅造饭，炊烟袅袅升起。但是，大家感受不到任何敌军要攻击的意思。
<br/>　　众人看着这一幕，心里浮现一丝困惑：这程大雷究竟要干什么。
<br/>　　这一日，就如此过去，没有任何事情发生。夜幕降临，李乐天悬着的心并没有放下去，黄沙城依旧保持警戒状态，提防程大雷有可能发动夜袭。
<br/>　　程大雷没有夜袭。
<br/>　　这一夜静悄悄过去。第二天天亮，各路诸侯聚在黄沙城城主府的大厅，会议依旧由李乐天主持。在会议开始前，首先询问了李行哉的伤情。李行哉已经经过治疗，自然是军中最好的大夫。目前他还下不了床，但根据现在的情况判断，他一条伤腿是可以保住了，未来不会成为一个瘸子。
<br/>　　这让尉迟离，以及尉迟离一派的人，心中暗道可惜。
<br/>　　然后，李乐天轻咳一声，将会议拉入正题。今天会议的主要目的就是询问大家对目前战局的判断或看法。
<br/>　　“我看那程大雷，已经是黔驴技穷，他知道打不下来，便不敢打了。”
<br/>　　一个白面中年人呵呵笑着说，他是青州牧，名唤周唤鲸，以人缘好出名。
<br/>　　李乐天听罢之后，没有太大反应，只是问：“诸位呢，诸位的看法怎样？”
<br/>　　“我觉得周大人言之有理，程大雷现在是被架在上面，打又打不了，如此退兵却又太没面子。”
<br/>　　豫州牧林涛附和道，他刚上任不久，便赶上了天下大乱，按派系来说，算是相府一派。
<br/>　　“山贼么，都好个面子。”周唤鲸扶手道。
<br/>　　“大家都是这么看么？”李乐天目光环视一圈，从他脸上的表情，看不出他心中任何想法。没想到他却突然发怒，手重重的拍在桌子上：“一派胡言！”
<br/>　　“你们动动你们的脑子好好想想，应该送到的粮草，为何现在还没任何消息。负责打探消息的斥候，已经一天一夜没传消息来了，他们，现在都去了哪里？”
<br/>　　李乐天怒不可遏，忽地，有种有心无力的感觉。原来父皇治下，就是这样一群蠢货。他们似乎生下来，就不存在脑子这种东西，就算到了现在，还是自己那三瓜俩枣。
<br/>　　李乐天的话如闷雷一般大厅内炸响，许多人之前就有了预测，然而，更多人直到此刻才明白。
<br/>　　有时候，什么事情都没发生，就是最好的事情。但有时候，什么事情也不发生，却恰恰是最糟糕的消息。
<br/>　　黄沙城涌进了七八万帝国军，但现在，该来的粮草没有来，该传进来的消息，也没有传来。黄沙城的诸位，成了瞎子、聋子、
<br/>　　八万人的队伍，没人哪怕只是一两粮食，加起来也是一个天文数字。人不吃饭，究竟可以撑几天，三天，五天，七天……程大雷的确不需要打，因为过不了几天，城里的人都会饿死。
<br/>　　不管是高高在上如崔相、尉迟离，或者卑贱只是一个普通兵卒，不吃饭都是要死的。
<br/>　　虽然现在外面没有消息传来，但李乐天等人大概也能猜出程大雷正在做什么。他的手下，定是封锁了外界通往黄沙城的道路。只要占领交通要塞，在利用蛤蟆军超强的机动性，这并不是一件太难做到的事情。
<br/>　　其实，之前攻入黄沙城时，未必没有粮草方面的考虑。只不过大家觉得，程大雷抢了很多粮草，都存在黄沙城，只要占了黄沙城，粮草就不是太大问题。
<br/>　　然而，谁知道程大雷用一场爆炸，毁掉了一切。
<br/>　　罪大莫过造反，计狠莫过绝粮。现在，程大雷就用一招绝粮计，把大家圈在了黄沙城。
<br/>　　面对眼前这种状况，诸人不由得又想起昨日李行哉的话：牢笼，坟墓……如此说来，李行哉还真是程大雷的知己呐。
<br/>　　“现在城中的粮草还可以坚持多久？”李乐天问。
<br/>　　“目前最多还能坚持两天。”尉迟离。
<br/>　　“那么如今只有一条路。”李乐天顿了顿：“打出去。”
<br/>(本章完)
