　　“哥，帮我报仇，废了他。”萧若寒虚弱地靠在萧若狂身上，满带怨恨地说了一句后，彻底晕厥了过去。
<br/>　　“若寒。”萧若狂见自己的弟弟模样如此凄惨，顿时心生怒意，看向萧逸的目光，充满了杀意。
<br/>　　“放心吧，我会宰了他。”
<br/>　　萧若狂人如其名，性格极为自大狂妄，从忽然出现，一拳揍飞萧逸，到现在更是扬言宰了萧逸。
<br/>　　当然，他也有狂的资格。
<br/>　　萧若狂，今年不过二十出头，已是玄火门门主坐下六大亲传弟子之一。天赋过人，修为有成。
<br/>　　加上其父乃是萧家五长老，可以说，他的这两重身份，让他成为了紫云城中最负盛名的风云人物。
<br/>　　至于玄火门，乃是坐落于紫云城外不远处的一个门派。
<br/>　　论势力与实力，与紫云城三大家族齐名。
<br/>　　门中上至门主、长老，下至普通弟子，皆不逊色于萧家。
<br/>　　而且萧若狂贵为门主弟子，在门中亦是最强的天才之一，日后极有可能成为下一任门主。
<br/>　　这时，长老席上的三长老出言问道，“萧若狂，你久未回萧家，此番回来，所谓何事？”
<br/>　　萧若狂名声在外，而三长老乃萧家代理家主，自然要问个清楚。
<br/>　　萧若狂负手而立，高声回答道，“禀三长老，听闻家族里出了一个了不得的天才，若狂此番回来，就是想看看这个天才到底有几斤几两。”
<br/>　　“你的意思是？”三长老萧重皱眉问道。
<br/>　　“自然是挑战萧逸，希望他不要让我失望才好。若是在我手上几招都接不住的话，这种废物，有何资格当萧家少家主？”萧若狂说着，眼角瞥了萧逸一眼，目光中尽是不屑。
<br/>　　“挑战萧逸？”族人中，立刻引起了轩然大波。
<br/>　　萧若狂的身份有些特殊，虽为萧家子弟，但多年前已经离开萧家，加入玄火门。所以，无论萧若狂日后有何成就，都只能挂着萧家子弟的名头，却不可能担任萧家任何重要的职务。
<br/>　　毕竟紫云城内外的势力，彼此间都存在着一定的争斗。
<br/>　　此时，五长老见状，当即站了起来，对族人们说道，“诸位勿要多想，我儿若狂此番回来，没有其他意思，只不过是挂念我这父亲，同时也是想为家族出一分力罢了，此番事了，他就会回去玄火门。”
<br/>　　一旁的三长老当即冷笑一声，什么为家族出一分力，嘴上说得好听，实则不过是想在比武台上打败萧逸罢了。
<br/>　　难怪五长老之前要将检验的日子定在三天后，原来就是想拖延时间，等这萧若狂回来为萧若寒造势。
<br/>　　无论萧若寒能否在挑战中打败萧逸，萧若狂都将起着极大的作用。
<br/>　　“逸儿，你是否愿意接受挑战？”三长老对比武台上的萧逸问了一句，同时补充道，“萧若狂如今乃是玄火门弟子，无权过问家族重大事宜，你有权拒绝。”
<br/>　　三长老的意思很清楚，想让萧逸拒绝。
<br/>　　五长老当然不会让这样的事情发生，当即道，“萧逸，无论你有没有那个资格当少家主，但你身为萧家子弟，面对挑战，难道如此没种，退缩拒战？”
<br/>　　比武台上，萧逸暗暗冷笑。
<br/>　　说起来，五长老虽然为人狡诈阴险，但确实命好，两个儿子，皆是天才武者。小儿子是萧家年轻一辈第一人，大儿子是玄火门门主弟子，可谓一门双杰。
<br/>　　只可惜，两个儿子都不是好东西，都是阴险狡诈之辈。
<br/>　　不过，管你什么双杰，敢惹自己，照样一顿揍。
<br/>　　萧逸高声道，“我应战。”
<br/>　　边说着，萧逸冷冷地看向萧若狂，道，“刚才那一拳，总不能让你白打了。”
<br/>　　长老席上，三长老气急，更是重重地拍了下大腿，“哎呀，逸儿太冲动了。”
<br/>　　五长老则阴笑一声，暗道，“果然是年少气盛，小废物，你死定了。”
<br/>　　.....
<br/>　　比武台上，裁判见状，道，“既然萧逸应战，那么比武随时可以开始，两位准备好了吗？”
<br/>　　“可以了。”萧逸点了点头。
<br/>　　“对付一个废物，我需要准备吗？”萧若狂不屑地哼了一声。
<br/>　　下方族人们，无不纷纷叹气。
<br/>　　“唉，萧逸是败定的了。”
<br/>　　“可不是嘛，听说这萧若狂乃是玄火门门主六大亲传弟子之一，尽得玄火门真传，厉害得很呢。”
<br/>　　“不仅如此，听说他早些天已经突破到后天一重，玄火门还大肆宣扬了他这个天才一番呢。”
<br/>　　族人们其实对萧若狂很不感冒，一来不喜欢他的狂妄，二来他不是萧家的核心弟子。
<br/>　　所以他们更希望萧逸能赢，只不过，萧逸已经‘冲动’地应战，一个凡境九重，无论再怎么厉害，都不可能打得过后天境的。
<br/>　　凡境九重与后天一重，虽然仅仅差了一级，但这是大境界上的差距，彼此间的实力，更是天差地远，如云泥之别。
<br/>　　比武台上，当裁判宣布比武开始时，两人都没有动。
<br/>　　萧若狂狂妄惯了，伸出一只手指对着萧逸，挑衅道，“你很弱，没资格让我先出手。”
<br/>　　萧逸冷笑一声，道，“你很狂，但在我面前，你还没那个资格狂。一分钟，若一分钟内我无法击败你，当我输。”
<br/>　　“猖狂。”
<br/>　　“大言不惭。”
<br/>　　萧若狂和五长老同时喝了一声。
<br/>　　连一直对萧逸颇有好感的二长老也摇了摇头，自语道，“这小子，之前我还夸他心智过人，没想到也是冲动自大之辈，唉，可惜了....”
<br/>　　他的话还未说完，‘可惜了’三个字之后的话语，忽然咔在了喉咙中。
<br/>　　因为，比武台上，萧逸的气势竟然在急速飞涨，而且是一种恐怖的气势。
<br/>　　“怎...怎么回事？”二长老愣了愣，自语道，“这小子，怎么忽然间气势大增，修为飙升了？”
<br/>　　比武台上，萧逸气势大涨，冷冷地盯着萧若狂，道，“听好了，你在玄火门狂也就罢了。但，萧家，还轮不到你撒野。刚才你打我那一拳，我要你十倍奉还。”
<br/>　　“升龙。”萧逸大喝一声，随即脚步一动，朝萧若狂攻去。
<br/>　　不错，萧逸使用了形意五绝中最强的一式，升龙。此乃形意拳中的精粹，更是深奥的武学。
<br/>　　这不是什么攻击的手段，而是一种以自身内力强行打通体内全身经脉，达到短时间修为大涨的方法。
<br/>　　华夏武学，博大精深。传闻，升龙一式，炼至巅峰，甚至能做到如龙腾四海，举手间，翻江倒海，威力无匹。
<br/>　　当然，现在的萧逸还没有那个功力，无法做到那个地步。但对付区区萧若狂，升龙一式哪怕只是发挥出百分之一的威力，也足以轻松战胜。
<br/>　　嗖的一声，萧逸消失在原地。
<br/>　　“好快的速度！”萧若狂瞳孔一缩，满脸不可置信，“怎么可能，小废物不过是凡境九重，论速度怎么可能比我这个后天一重还快，甚至超越了我眼睛所能捕捉的速度。”
<br/>　　嘭的一声，当萧若狂反应过来时，已经被萧逸一拳轰中了后背脊梁。
<br/>　　“噗”萧若狂猛地吐出一口鲜血。
<br/>　　后天一重的强悍肉身，竟然挡不住萧逸一拳。
<br/>　　又是嘭的一声，萧逸忽然一脚将萧若狂踹飞。
<br/>　　“额。”萧若狂暗暗吃疼，却毫无犯病，硬是被轰到了半空中。
<br/>　　嗖，萧逸再次消失。当他出现时，已经在半空中的萧若狂身旁。
<br/>　　“给我滚下去。”萧逸冷喝一声，重重地一拳轰在他的胸膛上。
<br/>　　“噗。”萧若狂再次口吐腥血，如同一道断线风筝，直接被轰落地面，将地面砸出一个巨坑，扬起漫天灰尘。
<br/>　　嗖，萧逸再次消失，落到地面。
<br/>　　漫天灰尘阻挡了所有人的视线，但里面却传出了连绵不绝的嘭、嘭、嘭轰击声。
<br/>　　族人们甚至不用多看，已经能想象到里面发生了什么事情，绝对是萧若狂在被虐。
<br/>　　数秒后，当灰尘落下，所有人都被眼前的一幕吓呆了。
<br/>　　萧若狂满脸狼狈地被卡在坑中，浑身是血，而萧逸则一拳又一拳地打着。
<br/>　　“父亲，救我。”萧若狂无力反抗，只得气若游丝地呼救着。
<br/>　　照这个势头下去，恐怕都不用一分钟，萧若狂就得被活活打死。
<br/>　　族人们不可置信地咽了口口水，谁能想象得到，一个后天境武者，竟然被一个才刚刚16岁的少年轻松暴揍。
<br/>　　刚刚还猖狂无比，气焰嚣张的萧若狂，此刻竟被打得如此凄惨。
<br/>　　当然了，族人们心中是暗暗叫好的。
<br/>　　长老席上的五长老也感到不可思议，他本想让萧若狂回来教训萧逸，有可能的话，暗中下死手，废了萧逸。
<br/>　　谁能想到，他的毒计不成，反倒是萧若狂差点被萧逸废了。
<br/>　　萧若狂这个当事人心中也是一阵哀嚎痛哭，万万想不到，自己突破到后天境，意气风发地回到家族，还未嚣张一会儿，立马就被收拾得这么惨。
<br/>　　“若狂。”五长老已经忍不住了，当即高声对裁判道，“立刻结束比武。”
<br/>　　一旁的三长老自豪地笑道，“现在距离一分钟还远着呢，五长老这么沉不住气？”
