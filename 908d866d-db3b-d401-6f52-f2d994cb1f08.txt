        就连逢年过节，那个时候，大哥傅夜沉的压岁钱总是他傅昼景的压岁钱的四五倍。
<br/>
<br/>        哪怕大哥傅夜沉事后会把他的压岁钱分一半给他，但在他傅昼景的心里，那只是大哥傅夜沉同情他，施舍给他的罢了。
<br/>
<br/>        小时候，就连邻家的一群小女生，都知道戴上有色眼镜看人，前前后后簇拥的、追捧的，都是大哥傅夜沉。
<br/>
<br/>        哪怕大哥傅夜沉对她们视而不见，甚至冷落她们，她们仍旧一个个不要脸地继续主动追求。
<br/>
<br/>        除了大哥傅夜沉是嫡出，他傅昼景是庶出以外，傅昼景并不觉得自己比大哥差劲，可为什么，大家的心仍旧偏向于大哥？
<br/>
<br/>        一路上，童筱颖和傅昼景两人各怀心思。
<br/>
<br/>        其实，他和她一样出身不好，但她从未因自己的出身不好而自卑，而他却一直活在了自卑的世界里无法自拔。
<br/>
<br/>        车子抵达临海城的五星级酒店地下车库的停车场里，傅昼景下车后给童筱颖开了车门，她一下车，他便拉着她的手。
<br/>
<br/>        两人就像一对热恋中的情侣，十指相扣，进入了电梯。
<br/>
<br/>        在电梯抵达一楼后，傅昼景和童筱颖遇到了正和他们一样来参加小叔公六十岁生日寿宴的傅夜沉和苏漫雪。
<br/>
<br/>        傅夜沉黑眸冷淡，看到傅昼景紧紧地牵着童筱颖的手，两人十分亲密的样子，眸色瞬间黯淡了几分。
<br/>
<br/>        他俩恩爱的样子，忽然间，让他心里十分的不悦，但他却不能有任何破绽把自己的心思露出来。
<br/>
<br/>        而童筱颖也发现，傅夜沉和苏漫雪之间，一直都是苏漫雪挽着傅夜沉的胳膊，傅夜沉就像一尊肃穆的军人雕塑一样，双手很自然地捶在裤腿两边。
<br/>
<br/>        “大哥、大嫂。”傅昼景温文儒雅地唤了声。
<br/>
<br/>        童筱颖回过神来，也跟着唤道：“大哥、大嫂。”
<br/>
<br/>        苏漫雪见状，故意将自己的身子往傅夜沉身上贴，然后矫揉造作地问道：“筱颖啊！我见你，两眼有眼袋和黑眼圈了，是不是昨晚小别胜新婚，咱们这小叔子把你折腾得彻夜未眠？”
<br/>
<br/>        “大嫂真会说笑，我岂会舍得彻夜折腾筱颍。”傅昼景淡淡地笑了笑。
<br/>
<br/>        “……”听傅昼景这么一说，童筱颖下意识地看了傅昼景一眼，瞬间脸红地低下头去。其实，她跟他，牵手拥抱，哪怕睡在了同一张床上，他都未对她逾越雷池。
<br/>
<br/>        傅夜沉心里听着当然不是滋味，却默不吭声，任由苏漫雪在一旁八卦。
<br/>
<br/>        “这种事情，还是你大哥厉害，一夜能七次。对吧？阿沉！”苏漫雪柳眉挑了挑，故作娇羞地摇了摇傅夜沉的手臂。
<br/>
<br/>        傅夜沉冷冷地瞥了苏漫雪一眼，苏漫雪无意间地抬眸，目光与他在半空中不期而遇。
<br/>
<br/>        他冷冽的眼神里透着戾气，让苏漫雪不寒而栗，下一秒，她便识趣地闭上了嘴。
<br/>
<br/>        他俩的小动作，童筱颖和傅昼景并未注意到。
<br/>
<br/>        童筱颖被傅昼景紧紧地牵着，她小鸟依人般整个身子都被傅昼景搂入了怀中。
<br/>
<br/>        看得出，他俩是彼此相爱的。
<br/>
<br/>        可他们的相爱，却让他傅夜沉的内心在滴血。
<br/>
<br/>        电梯抵达了设宴大厅所在的楼层。
<br/>
<br/>        电梯门一开，一群西装革履的男女，直奔傅夜沉和苏漫雪的跟前欠身问候。
<br/>
<br/>        傅昼景牵着童筱颖从傅夜沉和苏漫雪的身后绕过，几乎没有人去关注他们两个人。
<br/>
<br/>        就连他俩的位子，也被安排到了离主席台最远，离大门口最近，而且又是最偏的角落里。
<br/>
<br/>        傅昼景先去上了红包，然后才和童筱颖一起入席就坐。
<br/>
<br/>        而傅夜沉和苏漫雪，则被请到了主席台前第一个最好的酒桌前，和已经到位的傅老太爷傅炎廷以及傅夜沉的母亲冷夫人坐在一桌。
<br/>
<br/>        苏漫雪今天可谓是打扮得花枝招展，哪怕她和童筱颖有着一模一样的脸，但童筱颖向来素颜出场，所以她苏漫雪这化了妆的，自然是美得更胜一筹。
<br/>
<br/>        寿宴开始后，一个个长相标致的女服务员开始排着长队上菜。
<br/>
<br/>        餐桌上的西餐偏多，童筱颖只觉自己吃不饱，索性多喝了几口果汁，中途不得不起身去了一趟洗手间。
<br/>
<br/>        童筱颖上完厕所，刚从洗手间里出来，站在洗脸盆前洗手的时候，身边一个看上去只有十六七岁，扎着马尾辫，身着天蓝色连衣裙的女生，突然对她说道：“大嫂，你这不化妆比化妆要好看多了！你皮肤好白好嫩噢！是怎么保养的呀？还有，大嫂你这是随身带衣服的吗？才一眨眼的功夫，你就在卫生间里换了一套衣服还把妆给卸了！”
