        &lt;h3&gt;第三卷第二百零八章  看着你们幸福就好&lt;/h3&gt;
<br/>
<br/>        短暂而快乐的假期转眼就过去了，人们都恢复到日常生活中。
<br/>
<br/>        陈子昂离开杭州的时候又去了一趟郭力的家里，这一次是正儿八经的谈工作，郭力不愿意去北京，因为老婆孩子都在杭州，而且杭州适合居住，他的工作性质也没有必要天天跑到公司上班，只要有网络就好。
<br/>
<br/>        他的工作量每天只有四个小时，四个小时之内跟着大盘走势分析行情，给予公司正确的判断和抉择就好，当然整个公司的经济师也不止是郭力一个人。
<br/>
<br/>        郭力的工作室里有八个大屏显示器，每个显示器上都显示着股市大盘走势，而他的那个23寸的显示屏上就是他的分析报告。
<br/>
<br/>        在和陈子昂接触之后，陈子昂就开始邀请他来做她们公司的经济师，开始他还有些犹豫，他喜欢自由，而且孩子还小，雇保姆他也不放心，老婆想出去工作，他是不能去北京的，怕耽误陈子昂的事情，可是他没有想到陈子昂亲自来找他，让他心里还是很感动的。
<br/>
<br/>        陈子昂到他家里的时候，孩子已经睡了，俩个人昨天聊了怎么养小孩的事情，一下就拉近了俩个人的距离感，似乎他们已经成了朋友。
<br/>
<br/>        “小栖桐睡了？”陈子昂没见到孩子。
<br/>
<br/>        “不巧，你今天不能抱他了，这一觉大概得三个小时。”郭力笑着说。
<br/>
<br/>        “嗯，昨天是来找他的，今天是来找你的。”陈子昂和宁雅一起。
<br/>
<br/>        郭力将陈子昂和宁雅请进客厅。
<br/>
<br/>        陈子昂开门见山：“我今天是来请你加入我们的团队的，江总给我推荐过你，宁雅也给我推荐过你，而且我和你也聊了这么久，你的条件就是不想出门，要在家里带宝宝，据我所知，你的工作大概每天就是四个小时，你是不用去北京的。况且现在的信息如此发达，你也没有必要去北京。”
<br/>
<br/>        “这个。”郭力心里还是有点犹豫。
<br/>
<br/>        “我会让人给你准备一套会议系统，你和你的团队每天可以进行适时沟通。”陈子昂看着郭力，让郭力无法拒绝。
<br/>
<br/>        “行，你来看看我的工作室。”郭力不再推辞。
<br/>
<br/>        “你的这个工作室也很牛逼啊。一机多屏？”陈子昂笑着问。
<br/>
<br/>        “不是，好几个电脑呢。”郭力指着电脑对陈子昂说。
<br/>
<br/>        “不用那么麻烦，我让人给你重新安装一套系统，比你这个方便很多。”陈子昂是搞信息化出身的，一眼就看出了郭力的这几个显示屏放在一起会很麻烦，一个人要跑着几个电脑操作。
<br/>
<br/>        “行，你有更合理的就用你的，我这手头上还有几个客户，一时半会也不能推掉。不过他们都是个人，不是企业的。”
<br/>
<br/>        “没问题的，你自己做主就好，谁还没有个小生意。那就这样说定了，我安排这边的人和你对接，准备好之后你就可以接手工作了。”
<br/>
<br/>        “行，你说了算。”
<br/>
<br/>        宁雅：“你们俩个就这样谈妥了？”
<br/>
<br/>        陈子昂：“那还要怎么谈？”
<br/>
<br/>        宁雅：“你们俩个连待遇都没有说。”
<br/>
<br/>        陈子昂：“啊，还有待遇没有说啊？你想要什么样的待遇？”
<br/>
<br/>        郭力哈哈哈大笑：“我想要的你都知道，不说也罢。”
<br/>
<br/>        陈子昂双手一摊：“他不说。”
<br/>
<br/>        郭力：“调皮！”
<br/>
<br/>        宁雅：“调皮！”
<br/>
<br/>        从郭力家里出来，宁雅看着陈子昂：“你给了郭力什么好处？他会不要任何报酬给你卖命？”
<br/>
<br/>        “他有给我卖命吗？他在家里舒舒服服的带孩子上班，你看看他讨论孩子的时候，那个兴奋劲。”
<br/>
<br/>        “你就告诉我是怎么回事嘛。”
<br/>
<br/>        “秘密，不告诉你。”陈子昂笑呵呵的就是不给宁雅说是怎么回事。
<br/>
<br/>        “哼，矫情，小心嫁不出去。”
<br/>
<br/>        “嫁不出去我不是还有你吗？谁那天说要娶我来着？”陈子昂开着玩笑。
<br/>
<br/>        宁雅：“原来江总早都认识郭力，之前你们都联系好了的。”
<br/>
<br/>        陈子昂：“俞轩是怎么认识郭力我不知道，他曾经给我推荐过郭力，我后来又问他要了郭力的qq，会经常咨询郭力一些事情，一来二往我和郭力也就成了朋友了，有事情了我们在qq里聊聊天，没事情了就各自沉寂了。”
<br/>
<br/>        宁雅：“羡慕你这性格和谁都能相处得来，而且只要是和你交往的人都会被你吸引，就连我老爸都被你吸引了。”
<br/>
<br/>        陈子昂：“你老爸和我都是生意人，有生意合作当然是相互吸引，谁和钱过不去呢？不过你老爸倒是因为你意外的得到了我这样的一个客户，哈哈哈，这一趟回家，你还真没亏呢。”
<br/>
<br/>        “亏死了。”宁雅撅着嘴。
<br/>
<br/>        “嗯，我也觉得有点亏，你得想办法将凌辰拿下，你的那个妹妹可是虎视眈眈呢。”陈子昂半认真半调笑。
<br/>
<br/>        “说什么呢，怎么叫我把凌辰拿下？我和他半毛钱的关系都没有。”
<br/>
<br/>        “语言是可以骗人的，但是眼睛是不会骗人的，凌辰看你的眼神不一样，而你看凌辰的眼神也不一样，凌辰眼里都是你，你的眼里有凌辰但是就是不确定。”
<br/>
<br/>        宁雅沉默不语，她早都知道了她和凌辰之间是误会，但是凌辰有没有女朋友呢？
<br/>
<br/>        “你们之间肯定有误会，我想这一次凌辰到你家估计是为你而来，只是昨天的状况特别，他没有机会说出来。”陈子昂为宁雅分析着。
<br/>
<br/>        “我和他确实中间存在着误会，只是我觉得他不够相信我，所以我有些质疑我们之间的感情能不能长久，能不能经得住考验。”宁雅声音有些低沉。
<br/>
<br/>        “这个世界上最经不住考验的就是感情，爱了就爱了，恨了就恨了，没有必要去考验什么，你爱的时候你要坚信，你就是爱着，不爱了就是不爱了，不要拖泥带水，犹豫不决，伤害别人也伤害你自己，人生在世哪能坚守一段感情呢？总会碰到这样或者是那样的人，总会有那么一两次的动心，有些说不出口的感情，不能说那就不是感情，只是只能埋在心底，看着对方好就已经很满足了。”陈子昂轻轻的叹了口气，似乎是说给宁雅听，也似乎是说给自己听。
<br/>
<br/>        “陈子昂，你有爱过的人吗？”宁雅看着陈子昂那双突然忧郁起来的眼睛问。
<br/>
<br/>        “我不是人吗？只是他们都已经娶了别人，而且还有人给我打电话说：陈子昂，我结婚了你怎么办？”陈子昂笑了，笑得眼睛亮晶晶的，她抬头看天，天很蓝，阳光有些刺眼，差点晃得她的眼泪掉落了。
<br/>
<br/>        从此之后，她只能一个人哭，一个人笑，一个人闹，路很长，看着你们幸福就好！
<br/>
<br/>        风扬起的尘，入了我的眼，
<br/>
<br/>        那一些美好的，翻腾着的记忆，
<br/>
<br/>        放映着我与你的故事，
<br/>
<br/>        那一些简单的真实，那一些真实的甜蜜，
<br/>
<br/>        于我，在岁月里注定，青丝间藏着白发，
<br/>
<br/>        续写着时间的流逝，提醒着岁月的变迁，
<br/>
<br/>        而我，很想，将手放在你的手中，直到我离开这个世界的那一刻，
<br/>
<br/>        风，撒落一地的碎梦，我还是我，你依然是你，
<br/>
<br/>        我们平行着前进，似乎再也无法交集。
