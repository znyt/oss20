　　这一天，
<br/>　　下课之后，叶尘正打算离开，却被一道倩影拦在了面前。
<br/>　　赫然正是生物学院，如今的新晋院花，陶婉清。
<br/>　　“叶尘同学，你到底是怎么回事？为什么班里每次举行集体活动，你都不参加？”
<br/>　　陶婉清将叶尘拦下后，大声质问道，俏脸上浮现出一抹怒意。
<br/>　　生物学院人数不多，整个大一年级只有五个班，叶尘和陶婉清同属于一班，而且陶婉清是一班的班长。
<br/>　　原来，叶尘独来独往，班级和学院举行的集体活动，竟然每次都缺席不到，而且从来不向她这个班长解释原因，这让陶婉清有些忍无可忍，于是便直接当面兴师问罪起来。
<br/>　　“不为什么，我没兴趣！”
<br/>　　叶尘直接淡淡撂下一句话，然后便准备从一旁绕开，然而陶婉清当即再次拦在了叶尘面前。
<br/>　　“不行！今天你不说清楚，就不准走！”
<br/>　　叶尘不由得眉头一皱，
<br/>　　“那你想怎样？”
<br/>　　陶婉清深吸了一口气道：
<br/>　　“今天是刘老师的生日，我们五个班，全体同学，准备晚上在帝豪酒店，给刘老师过生日，这一次你必须到场！”
<br/>　　叶尘不由得一愣，
<br/>　　陶婉清口中的刘老师，是整个生物学院五个班，共同的辅导员，因为生物学院班级少，所以他一人承担五个班的辅导员，而且因为为人十分热情，所以深得同学们的喜爱。
<br/>　　一念及此，叶尘点了点头，
<br/>　　“好，我会参加的！”
<br/>　　陶婉清这才喜形于色，同时暗暗松了一口气。
<br/>　　倘若叶尘执意不参加，她还真没有太好的办法，到那个时候，她这个班长可就丢人丢大了。
<br/>　　好在这家伙还算识趣！
<br/>　　……
<br/>　　晚上，
<br/>　　叶尘和李虎三人，一同来到帝豪大酒店。
<br/>　　“啧啧，这可是五星级酒店嗳！据说这次给刘老师过生日，全都是杨之凡一手操办，不但出钱出力，还请了不少天海市的顶级阔少，还有其他学院的精英，咱们学院学生会的精英也全都到场了，场面可是相当的大啊！”
<br/>　　唐威咋舌道。
<br/>　　李虎却冷冷一笑，
<br/>　　“要我说，他这是别有用心，给刘老师过生日是假，给自己立威倒是真的！”
<br/>　　一旁陈东显然没有明白李虎的意思，当即问道：
<br/>　　“老四，你这话是什么意思？”
<br/>　　李虎一脸得意的笑道：
<br/>　　“你们没听说吗？最近这杨之凡，正在竞选咱们学院的学生会副主席！”
<br/>　　“他毕竟是大一新生，根基还太浅，如果不尽快树立威望，根本不可能竞选成功！”
<br/>　　“所以，他给刘老师过生日，只是一个幌子，真正的目的，说白了，不过就是为了向别人证明，他有多牛逼罢了！”
<br/>　　李虎一番解释，就连叶尘也不由得再次对他刮目相看。
<br/>　　李虎这家伙，虽然为人好色，而且整天吊儿郎当的，可其实心里面却精明的很，看问题也很透彻。
<br/>　　难怪叶尘隐约记得前世，这家伙后来好像成为了一方富豪，果然不是全凭运气！
<br/>　　很快，
<br/>　　三人进入帝豪大酒店之中，找到对应的宴厅。
<br/>　　此时宴厅里面，早已经聚集了不少人。
<br/>　　生物学院整个一年级，一共也就一百多人，整个宴厅里面，有十几张桌子，每张桌子上可以坐十几人，就算是加上杨之凡请来的其他人，应该也绰绰有余了。
<br/>　　而且，不知道是不是杨之凡刻意为之，每一张桌子上，都对应着个人的名字，显然是想要让人对号入座。
<br/>　　叶尘四人找了一圈，发现他们几个人的座位，赫然在最边缘的角落里面。
<br/>　　李虎顿时气得不行，
<br/>　　“妈的！这个杨之凡，明显是看不起我们，要不我们直接走吧，省得在这里受他的气！”
<br/>　　叶尘却淡淡一笑，
<br/>　　“终归要有人坐在这里的，而且角落里安静，挺好！”
<br/>　　说着，叶尘便拉开椅子，坐了下去。
<br/>　　陈东和唐威见状，也跟着坐了下来，李虎见三人都坐下了，也只好一脸恨恨的坐下。
<br/>　　四人才刚刚坐定不久，忽然有人高喊了起来，
<br/>　　“快看！是刘老师和杨之凡他们来了！”
<br/>　　众人顿时纷纷朝门口的的方向望去，
<br/>　　只见，
<br/>　　一名约莫二十七八岁的青年男子，穿着一件不太合身的西服，有些拘谨的走了进来。
<br/>　　原来，
<br/>　　这人便是他们的辅导员刘传明。
<br/>　　也难怪刘传明有些紧张，他研究生刚毕业才两年，如今也只是生物学院的辅导员而已，何曾被人如此隆重对待过？
<br/>　　上百人为他过生日，而且还是在五星级的大酒店！
<br/>　　这要是在以前，他连想都不敢想！
<br/>　　即便是现在，刘传明也有点懵，感觉就像是在做梦一般。
<br/>　　而跟在刘传明身旁的杨之凡、陶婉清等人，则显得就从容多了，显然这种场面，对于他们来说，早已经司空见惯。
<br/>　　“刘老师好！”
<br/>　　“刘老师生日快乐！”
<br/>　　众人纷纷站起，高声呼喊了起来。
<br/>　　刘传明顿时觉得整个人都有些飘飘然，脸上洋溢着满意的笑容，冲众人挥手示意，
<br/>　　“同学们好！”
<br/>　　很快，
<br/>　　刘传明在主桌主座上坐了下来，而杨之凡和陶婉清，也理所当然的在一旁坐下。
<br/>　　“刘老师，我请的那几位朋友，还有学生会的几位学长、学姐，应该也都快到了，咱们就再等等他们吧？”
<br/>　　杨之凡冲着刘传明微微一笑道。
<br/>　　刘传明自然是无有不允，
<br/>　　“好好好！”
<br/>　　……
<br/>　　过了片刻，
<br/>　　生物学院的学生会主席张连鹏，带着几名学生会的骨干也来了，也被杨之凡安排在了主桌。
<br/>　　紧接着，
<br/>　　又有几名青年男女走了进来，而且个个气质不凡，一看就知道非富即贵。
<br/>　　看到这几个人出现，杨之凡的脸上，顿时露出一抹喜色，当即快步迎了上去，
<br/>　　“王少！江少！方少！欢迎欢迎！快快里面请！”
<br/>　　杨之凡向这些人一一问好之后，将他们也引到了主桌之上。
<br/>　　而与此同时，整个宴厅的开始躁动起来。
<br/>　　“如果我没有看错的，那为首的三人，好像分别是王家、江家和方家的三位大少，人称天海三少！”
<br/>　　“不错！就是他们三位！王家、江家和方家，如今可是天海市新崛起的三大家族！”
<br/>　　“天海市是天南省的省会，天海三大家族，在整个天南省，也是最牛的了吧？”
<br/>　　“那倒不是！天南省如今最牛的家族，是云州的唐家和曹家，不过天海三大家族，也算是仅次于唐家和曹家的势力了！”
<br/>　　……
