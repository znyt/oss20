　　又一次的猛烈冲撞，让那深坑再次扩大，爆破更是轰轰烈烈的炸起一连串炸响。
<br/>　　大·片黄土冲上天空，本就龟裂的土地裂出了半米宽的条条裂缝，蜘蛛网一下子扩大至两倍，‘咔咔咔’的震裂声不断响起。
<br/>　　四周的防守队成员只能在震荡中不断后退，只有极少数的高战力者能抗住这种可怕的波动，可这一爆破刚一发生后不久，他们便听到‘碰碰碰’的几声。
<br/>　　只见那些死士一个个倒飞而出，摔在地上吐血不止！
<br/>　　个别精英还留在那爆破中心，不断发出阵阵闷·哼声。
<br/>　　只见他们站在丧尸王的半米之外，原本攻击的姿态全部改为了防守，而丧尸王的身上则不断爆发着恐怖气息，能量并不能肉·眼可见，但却能清晰的感受到那股波动在一圈圈的荡漾而开。
<br/>　　站在对面的白允儿，则是身前一道诡异的蓝紫色屏障，其上‘噼里啪啦’的带着电闪雷鸣，将刚才的那阵爆破及丧尸王释放的能量波动，集体挡了回去！
<br/>　　尚九娣早已在开打的瞬间用最快速度后退，此时她站在能量波及的范围内，但却聪明的躲到了白允儿身后侧，站在了那道屏障的后方。
<br/>　　白允儿面色带着苍白，显然重伤状态下的她再次抗住这种爆破威能，已经筋疲力竭。
<br/>　　前方白允儿的阻拦，后方白家死士的纠缠，让丧尸王木天的耐心到达极限，他不再浪费时间和精力，在众人都还处于刚刚那一爆破的惊吓中时，猛然转身！
<br/>　　粗·壮的手臂伸出，一拳轰向了一名白家死士的心脏！
<br/>　　实质化的攻击带来的恐怖威力，让此人的心脏部位穿出一个洞，同时身躯在这股震压之下扭曲，骨骼发出‘磕磕’的挤压声，皮肤缓缓变得红紫相间。
<br/>　　紧接着——
<br/>　　嘭！
<br/>　　直接爆裂而开，炸成了一地的碎肉！
<br/>　　这一幕发生的极快，那名死士的同伴们都来不及反应，但木天却已经将目标放在了下一名死士身上。
<br/>　　在这一过程中白允儿一步未动，冷冷的目光看着前方，她的嘴角有一丝鲜血将要溢出，却被她硬生生咽了回去。
<br/>　　刚才那些白家死士们的集体攻击，一瞬间的威力不下于丧尸王的第一次突袭，虽然持续时间短，但还是需要白允儿用尽全力抵挡。
<br/>　　因为哪怕是一瞬间的威力，也足以让身后的木屋灰飞烟灭！
<br/>　　白允儿很清楚白家这些人的使命是什么，哪怕如此也要尝试杀掉楚涵么？
<br/>　　这时候的尚九娣捂住胸口，平抚着心悸感道：“刚才怎么了？”
<br/>　　白允儿眉头拧成了一个川字，在刚才的爆破中木屋之后的情况被完全掩盖，她哪怕感知力再强也听不到。
<br/>　　“快去后面，有……”白允儿的话还未说完，就猛然一口血涌上来，腥红的颜色染在身上。
<br/>　　尚九娣一惊，但却来不及查看白允儿伤势，举起枪就往后方跑。
<br/>　　只是在她刚迈开脚步的一刹那——
<br/>　　噗噗噗！
<br/>　　一连串的几道声响，伴随着一声声惊恐的叫喊声，响起在木屋的后方地带！
<br/>　　尚九娣的脚步猛然一顿，手中的枪支不停颤抖，紧接着便一个冲刺，发了疯似得拼命跑向木屋。
<br/>　　她没有再去木屋后方的那片地带，因为可能……
<br/>　　已经没有必要了！
<br/>　　顾凉晨及狼牙的防守队成员，这时候也反应了过来，一下子涌上来将木屋从前方开始围住，顾凉晨更是惊慌失措的直接跳到木屋的屋顶。
<br/>　　很快，宽阔的视野便将一切都呈现在他眼前。
<br/>　　木屋前方是大·片的战斗痕迹，白允儿挡住了两次爆破，正口吐鲜血受了重伤。
<br/>　　那只丧尸王在屠杀白家的死士，到处都是带着伤势乱窜逃跑的身影，却也有不肯走留在原地找机会的大胆者。
<br/>　　而木屋的后方……
<br/>　　却是一群罗家死士的尸体！
<br/>　　与木屋前方已知及能猜到的发展相比，木屋后方的场面简直触目惊心！
<br/>　　罗家派来的死士作为暗杀者，目的就是为了杀楚涵，他们不择手段的绕后前来无可厚非，但眼下这东倒西歪的一地尸体，以及血流成河的场面，却让顾凉晨都心惊胆战。
<br/>　　这些人不是驻守在后方的狼牙成员所杀，而是另有他人！
<br/>　　因为他们的死亡，全部都是被掏出了心脏！
<br/>　　这样的手法，这种模式……
<br/>　　不等顾凉晨思考，一个长相极其清秀，肤色白·皙面上带笑的年轻男子，就这么一步步走出，他的手中还拎着一个血淋淋的心脏。
<br/>　　他一点没有要影藏自己的意思，就这么抬眼冲着屋顶上的顾凉晨一笑，一边缓慢优雅的将手中心脏递到嘴边吃着，不留一滴残渣和鲜血在地，一边像是散步一样走向木屋。
<br/>　　眼前的一切，无论是顾凉晨还是狼牙防守队，还是罗家白家的死士，在他看来似乎都不值一提，只是路边的风景……
<br/>　　顾凉晨整个人都开始发凉，脚底更是冰冷打着哆嗦。
<br/>　　丧尸王来了，异种王难道也来了？！
<br/>　　此时的木天也已经解决掉了围住他的那几名死士，还有极个别逃脱，他没有追上去而是站在原地，脚边满是人类的碎肉，暴力的手段仿佛就是为了凸显出他身为丧尸王的身份。
<br/>　　他嘴角挂起一个古怪的笑意，而后忽然咧开嘴冲着白允儿道：“你没机会了。”

<br/>　　尚九娣举着枪，恶狠狠的看着木屋后方越来越近的脱变异种，但越是凶狠，却越是凸显出眼前这个大异种漫不经心的强大。
<br/>　　狼牙成员的心头，一阵阴霾和绝望，心头只环绕了两个字：
<br/>　　完了！
<br/>　　木屋前白允儿看着丧尸王，她的表情从未如此冰冷，瞳孔在此时开始一阵阵收缩，一道细微的炫蓝色流光在她眼中闪烁，而后一点点的扩散至瞳孔大小。
<br/>　　于此同时的天空，忽然毫无预兆的乌云密布，狂风四起间将她的长发吹的飘扬，
<br/>　　以她为中心的土地开始出现一圈圈的波浪状风形，一层层外围扩散，无数的尘埃开始莫名往上浮，像是身处一个诡异的磁场地带。
