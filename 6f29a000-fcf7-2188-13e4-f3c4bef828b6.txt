        但因为傅夜沉的母亲是个混血儿，所以，傅夜沉的五官比傅昼景的五官更加深邃立体。
<br/>
<br/>        童筱颖不否认，傅夜沉长得确实比傅昼景更帅，但是，他太过阳刚霸道，让她心里更偏向于傅昼景的温柔与体贴。
<br/>
<br/>        此时此刻，傅夜沉的脸上没有一丝表情，他深邃俊美的五官冷冰冰的，一双黑色的瞳仁就像打量猎物一般，正打量着他们。
<br/>
<br/>        傅昼景不经意间地对上傅夜沉的目光时，只觉大哥傅夜沉的目光凌厉，似要将他猎杀一般地阴鸷冷血。
<br/>
<br/>        “大哥。”当傅夜沉走到他们跟前的时候，傅昼景微笑着唤了声。
<br/>
<br/>        童筱颖小鸟依人般靠在傅昼景的怀中，礼貌性地向傅夜沉颔首，问候道：“大哥，晚上好！”
<br/>
<br/>        对，她唤他大哥，嫁夫随夫，所以唤他大哥，但是他一点都不想当她劳什子的大哥！
<br/>
<br/>        更何况，阿景还……
<br/>
<br/>        傅夜沉将目光落在童筱颖身上时，心情极其复杂，他怒极反笑，薄唇勾起一抹意味不明地笑意，像是在打趣地说道：“你们俩站在这大门口亲热，还真是有损风雅。”
<br/>
<br/>        刚刚，傅昼景前脚刚到，他后脚便到了。
<br/>
<br/>        只是，令他没想到的是，他就坐在车内，还未下车，就要眼睁睁地看着他俩当众接吻。
<br/>
<br/>        其实，他知道，筱颖在阿景身边的时候，一定被阿景吻过无数回。
<br/>
<br/>        但是，要他亲眼看着她与自己的弟弟接吻，这对他来说，就像一把锋锐的刀刃，在割他的心头肉。
<br/>
<br/>        所以，他才让司机把远光灯打开，直接打到他们身上，阻止了他两的亲热。
<br/>
<br/>        他傅夜沉，是不是很坏？！
<br/>
<br/>        “大哥说的是，我和筱颍，向大哥道歉。”傅昼景也一本正经地开玩笑。
<br/>
<br/>        童筱颖顿时被傅夜沉说得脸红耳热起来，两只手紧紧地挽着傅昼景的胳膊，默不吭声。
<br/>
<br/>        “我听爷爷说，明天你们俩就要移民定居去澳洲了。以后，跟我和爷爷不再相见。那么，我希望你们将来，能在澳洲过得幸福。”傅夜沉一边说着，一边张开手臂，朝傅昼景抱了过去。
<br/>
<br/>        他抱住傅昼景的同时，也将童筱颖一起拥入了怀中。
<br/>
<br/>        傅昼景并未察觉到大哥傅夜沉的异样，只是心底突然对大哥感到愧疚起来。
<br/>
<br/>        从小到大，大哥一直护着他，不管他犯了多大的错，大哥总是会想办法替他解决，而他却一直在暗中偷偷地算计过大哥。
<br/>
<br/>        “阿景，不管以前你发生过什么。今后，我希望，你能一心一意地对筱颖。”傅夜沉深沉地说道，微微偏头颔首之际，他的薄唇，落在了童筱颖的头顶。
<br/>
<br/>        他吻上了她的头顶，轻轻地闻着她发梢间那令他梦回萦绕的柠檬香气。
<br/>
<br/>        这世上，应该没有谁比他爱得更痛苦了吧！
<br/>
<br/>        他只能以这种另类的方式，与自己心爱的女孩道别。
<br/>
<br/>        语毕，吻也毕。
<br/>
<br/>        傅夜沉放开了傅昼景和童筱颖，淡然地笑了笑。
<br/>
<br/>        傅昼景抬起手来，拍了拍傅夜沉的肩头，微微一笑：“大哥，你也不要太过伤心。大嫂，哦，不是。是苏漫雪那种女人，不值得大哥你去爱。”
<br/>
<br/>        “苏漫雪她怎么呢？”童筱颖忍不住好奇地反问。
<br/>
<br/>        “她……”
<br/>
<br/>        “她没事，很好。”傅夜沉立即打断了傅昼景欲要说的话。
<br/>
<br/>        但是，傅夜沉的话，并未打消童筱颖的疑虑。
<br/>
<br/>        苏漫雪不是跟她说，她苏漫雪是大哥傅夜沉的软肋，所以才被爷爷保护起来，困在依山别苑里的。那么，她苏漫雪到底为什么要跟她互换身份，逃离依山别苑？
<br/>
<br/>        这个问题，一直让童筱颖感到很纳闷。
<br/>
<br/>        殊不知，是她苏漫雪自作自受，被季思妍给算计了送到了看守所后，又被爷爷傅炎廷狠心顶包了傅昼景所犯下的错，辗转间去坐了牢。
<br/>
<br/>        季思妍一直以为去坐牢的女人是童筱颖，所以暗中花钱讨好监狱里的那群女牢监犯，让她们在监狱了欺负“童筱颖”。
<br/>
<br/>        “你们叙旧续完了吗？续完了就赶紧进屋吧！你们肚子不饿，我肚子可是饿了！”连华生不知何时，未见其人到先闻其声。
<br/>
<br/>        他的声音，打断了三人的话题。
<br/>
<br/>        三人不约而同地循声望去，只见连华生从大门内走出来。
<br/>
<br/>        “傅老太爷可是叫吃晚饭了，你们还要磨蹭到什么时候去？”连华生笑吟吟地接着说。
<br/>
<br/>        童筱颖这一下午都在傅家大庄园里，却一直未见过连华生的身影，他是什么时候入的园子的，她不得而知。
<br/>
<br/>        傅昼景随之牵着童筱颖的手，对连华生微微一笑，然后拉着童筱颖进了大门。
