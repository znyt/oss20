　　第717章
<br/>　　钱通玉慢慢安静下来，强撑起笑容：“言重了，言重了，程当家当真要去长安？”
<br/>　　“委实是不想去。”程大雷道：“最近偶感风寒，又是一路旅途奔波，奈何陛下已经差人下旨，不去一趟实在说不过去。唉，毕竟是劳碌命，享不了清福呐。”
<br/>　　钱通玉三人都闭上嘴巴不说话，有些话他们并不方便说出口。毕竟不能指着程大雷的鼻子说：你一个山贼去长安不是找死么。
<br/>　　出奇之人，必有出奇之举。程大雷自从出世以来，做的许多事情大家都看不透，但却总有出人意料的收获。他这次赴险去长安，心中该是已经有了把握，至少能保证自己全身而退。
<br/>　　他的把握从何而来？难道说，他暗中真的是李氏皇族在支持，双方已达成某种意义上的同盟。
<br/>　　能成为一城之主，自然不会有人是傻瓜。钱通玉、白空集、孙别山三人概莫能外，此刻他们心底都思量着。程大雷就坐在他们面前，拔出刀就能杀掉的距离，可他却像周身罩着迷雾，令人无法看透。
<br/>　　钱通玉哈哈笑了一声，试图冲淡场子上的严肃：“那我就先祝程当家一路顺风，其实程当家误会了，我们三家对程当家一向都是很佩服的，不然也不会派人去和程当家谈。不过，原谅我们都有些私心，遭逢乱世，心里难免就多想一些。”
<br/>　　“想什么？”程大雷饮尽杯中酒：“我知道，在帝国我的名声不如何好，是真是假，世人自有公断。可我问一句，可有人听过，我对不起自己人？”
<br/>　　三人略想了想，确实在这方面，程大雷并没有什么污点。当初为了林少羽，他不惜与整个帝国为敌，做人做事，纵然读书人对其口诛笔伐，但仁义礼智信中的这个义字，程大雷并没有丢失。
<br/>　　“若说诚意，我今天出现在这里便是诚意。既然想要和三位做朋友，我便先将生死交给三位，可三位所言所行，委实有些差强人意，实在令程某谈不上佩服呐。”
<br/>　　钱通玉三人都垂下头去，心中有几分惭愧。倒是孙别山先开口，道：“不是不信任程当家，实在是世道如此。既然话已说到这个地步，孙某也就把话挑明了说。从今往后，姓孙的跟着程当家做事，这凉州，程当家说了算。”
<br/>　　钱通玉不过略顿了顿，便拱手道：“唯程当家马首是瞻。”
<br/>　　只有白空集一人未表态，程大雷目光看着他，钱通玉与孙别山也同时看着他。
<br/>　　白空集与宋伯康是结义兄弟，交情如何不去说，名声是有的。谁知他会怎样抉择，不过程大雷并不着急，他很有耐心。假若交情当真值钱的话，今日白空集何必出现在这里。
<br/>　　“以后听程当家的。”
<br/>　　程大雷端起酒杯，道：“请。”
<br/>　　三人同时举杯，异口同声道：“敬程当家。”
<br/>　　落日沉入西山，一轮明月升上半空，隔窗看过去，水天一色，耳边有夜鸟鸣声，虫语鸣鸣。
<br/>　　钱通玉喝得面红耳赤，道：“程当家不用担心，今日就在城中歇下，有一品楼的小红鲤，色艺双绝，今夜就由她伺候程当家。”
<br/>　　程大雷摆摆手：“不了，家里还有事要做，今夜就不在这里歇了。”
<br/>　　钱通玉三人面面相觑，莫非是小红鲤满足不了程大雷的胃口，程大雷嫌弃对方是个风尘女子。
<br/>　　也不是没有这种可能，说不得要献出家中的歌姬爱妾，更甚者还得是钱通玉的女儿侄女之类的。
<br/>　　如果程大雷真能摘到凉州王的封号，便是送给他也不算什么，说不得还是好事。
<br/>　　不过目前，程大雷还不值得钱通玉如此下本。现在钱通玉不明白程大雷推三堵四所为那般，倒是不曾听说他是个吃素的主儿。
<br/>　　程大雷去意已决，钱通玉三人再三挽留也是无用。只好一路将程大雷送到渡口，程大雷登上小船，双方互相拱手。
<br/>　　“程大雷，一路顺风。”
<br/>　　“好说，好说，诸位请回吧。”
<br/>　　依旧是银眸摇橹，顺流而下，船行的速度很快。但钱通玉三人立在渡口，遥望程大雷消失的背影，还未离开。
<br/>　　“当真就这样放他走么？”白空集道：“纵虎容易擒虎难呐？”
<br/>　　孙别山道：“真如果动手，有把握留住他的性命么？即便杀了他又如何，蛤蟆寨的精兵强将一拥而上，我们当真挡得住么？”
<br/>　　“也未必挡不住。”白空集说了一声。
<br/>　　“虎……”钱通玉摇了摇头：“此獠已露麟角之象，放眼整个帝国我不敢说，但在凉州，他已是一头化风化雨的龙，说不得有朝一日便乘风而起，龙腾九霄。”
<br/>　　白空集一楞：“我们当真要跟随一个山贼？”
<br/>　　“成者王侯败者寇，谁敢说他是贼？”
<br/>　　浩甲与凉州城相距甚远，但顺流而下总比逆流而上更快些。大概有两个时辰，程大雷已经返回凉州城。
<br/>　　刘悲、徐神机、吴用几人都等在城主府，他们也是担心程大雷。程大雷有胆子铤而走险，谁知钱通玉会不会狗急跳墙。
<br/>　　见到程大雷回来，三人才算放心。小鹿捧来醒酒茶，程大雷漱了漱口，吐在痰盂中。
<br/>　　“大当家，怎样，事情谈妥了没有？”吴用道。
<br/>　　“能活着回来，就算是谈得差不都了。再和他们接触一下，许多事情就应该好谈了。”
<br/>　　不单单刘悲三人，便是崔白玉几人，也惊叹程大雷的作为。想要做凉州的老大，总也是要摆出老大的气度。崔白玉以前接触的人，不缺乏智慧和算计，但勇气这种东西在贵族子弟中，的确是稀罕的品质。
<br/>　　“咱们能信任他们么？”吴用问。
<br/>　　程大雷难免多看了他几眼，这种愚蠢的问题，徐神机都不会问。
<br/>　　世上根本没有绝对信任这种东西，无非是诱惑不够，只要他们承担不起背叛的代价，他们便只能选择信任。
<br/>　　“看来我当真要走一趟长安。”程大雷静坐片刻，饮了杯茶。现在他已感觉，名正言顺当真是个好东西。
<br/>(本章完)
