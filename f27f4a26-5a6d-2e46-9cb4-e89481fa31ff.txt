　　一个小时，匆匆而过，
<br/>　　而在这一个小时的时间里，网上早已经彻底炸开了锅，全球观看直播的人数，已经超过了二十亿，几乎所有人都在议论这个惊天大新闻。
<br/>　　来自华夏国的一位超级英雄，一人横扫米国，踏灭国会山，逼迫米国政府投降！
<br/>　　之前，各国政府都十分默契的在竭力掩盖超自然力量，可是经过这次大事件，便再也掩盖不住了，
<br/>　　“原来这世上真的有神灵存在啊！”
<br/>　　“一人压倒一国！太强大了！”
<br/>　　“据说此人是来自华夏国的东方修仙者！”
<br/>　　……
<br/>　　随着各国网络上，越来越火爆的议论此事，一些黑暗世界的人，也开始纷纷站出来，向世人普及知识。
<br/>　　东方的修仙者，西方的血族、狼人、教廷等等，各种以前只存在于传说中的东西，纷纷被人搬了出来。
<br/>　　一开始，还有人半信半疑，可是随即，很多人拿出了相关的照片、录像等各种石锤，顿时彻底打消了这些人的疑虑。
<br/>　　这一天，对于全球的普通人类来说，绝对是颠覆性的一天！
<br/>　　自今日之后，所有人都知道，这个世界远远没有他们想的那么简单，在他们这个平凡的世界之外，还有着一个光怪陆离的世界。
<br/>　　就连叶尘也没有想到，他的这番举动，会给整个世界，带来多么巨大的影响，甚至整个人类的发展进程，也都因此而发生了变化！
<br/>　　……
<br/>　　在网上众人的议论中，米国总统普朗克和一众议员们，重新出现在众人的视野。
<br/>　　只见此时的普朗克，似乎一下子苍老了十岁，双手捧着投降书，颤颤巍巍的走到，依旧还在闭目养神的叶尘跟前，恭恭敬敬行了一礼，声音无比沉重道：
<br/>　　“我，普朗克，代表米国联邦政府，正式向叶先生阁下，投降！”
<br/>　　当那‘投降’两字，从普朗克的口中说出，身后的国务卿、国防部长，以及所有国会议员们，全都一脸羞愧的低下了高傲的头颅。
<br/>　　这绝对是自建国以来，米国所遭受的最大耻辱！
<br/>　　这个星球上最强大的国度，竟然真的向一个人投降了！
<br/>　　而与此同时，整个全球网络，也彻底爆炸了，已经掀起了一波修仙者的狂潮！
<br/>　　“这就是东方修仙者的能耐吗？我也要修仙！”
<br/>　　“谁能告诉我这位华夏仙人的信息，我要拜他为师！”
<br/>　　“话说这位华夏仙人，不但很年轻，长得也好帅啊，难道也是因为修仙者的缘故吗？”
<br/>　　“你懂什么？既然是仙人，自然具备长生不老的能力，说不定都已经几百岁了呢！”
<br/>　　……
<br/>　　人人都渴望得到强大的力量，人人都想拥有绝世容颜，人人都想要长生不老。
<br/>　　之前，世人以为仙人只是传说，可是现在他们看到了真正的仙人，哪里还按耐的住心头的欲望？
<br/>　　叶尘并不知道，他今日的这番举动，对世界造成了多么巨大的影响，伸手微微一抓，
<br/>　　唰！
<br/>　　普朗克手中的投降书，已经被叶尘吸入掌心。
<br/>　　大致扫了一眼，之前他所提的那两个条件，也已经被写入投降书之中。
<br/>　　“希望你们遵守承诺，否则这国会山，便是你们的榜样！”
<br/>　　说完这话，叶尘猛然抬手一拍，
<br/>　　轰隆！
<br/>　　之前已经被叶尘一剑劈成两半的国会大厦，瞬间化成了一堆齑粉，整个国会山都塌陷了下去。
<br/>　　等到众人站稳身形之后，天空上早已经没有了叶尘的身影。
<br/>　　既然他的目的已经达到，自然也就没有再继续待下去的必要了。
<br/>　　呼！
<br/>　　普朗克和众议员纷纷松了一口长气，不管怎么说，总算是把这个瘟神送走了。
<br/>　　“总统阁下，我们当真要把兵力撤出亚洲？还有那十万亿美金，虽然我们可以承受，可是难保此人以后不会再生事端！”
<br/>　　确定叶尘已经离开之后，国防部长拉姆斯一脸不甘道。
<br/>　　普朗克喟然长叹一声，道：
<br/>　　“不然还能怎样？”
<br/>　　众人一阵默然。
<br/>　　而就在这时，众议员之中，一名头发苍白的老者，忽然开口道：
<br/>　　“或许，我们可以借助那个世界的力量！”
<br/>　　“那个世界？”
<br/>　　众人全都一脸茫然，唯独米国总统、国务卿、国防部长等寥寥数人，瞬间明白了老者的意思。
<br/>　　“参议长大人说的是，阿拉斯加州的那个恶魔谷吗？据说只要将那里的封印打开，就召唤出恶魔为我们战斗！”
<br/>　　“可是那毕竟只是传说而已！没人知道真假？而且我们也不知道如何打开封印！”
<br/>　　“有一人肯定知道！”
<br/>　　“谁？”
<br/>　　“西方教廷的那位老教皇！”
<br/>　　……
<br/>　　叶尘并不知道，米国的这帮老家伙们，前脚才刚刚递交了投降书，紧接着竟然又开始密谋对付他！
<br/>　　当然，其实这也不难理解，
<br/>　　毕竟，谁也不想自己的头顶上，始终悬着一把利剑，随时都有可能要了他们的性命！
<br/>　　叶尘悄无声息的离开华盛顿之后，并没有返回返回华夏，而是先隐藏了自己的行踪，摆脱了卫星监控，然后朝北面赶了过去。
<br/>　　“之前已经复活了叶天歌，接下来也该去复活玫瑰了！”
<br/>　　玫瑰的葬身之地，位于米国最北部的阿拉斯加州，玫瑰的尸身，被冰封在一处冰湖湖底的万年玄冰洞之中。
<br/>　　当初，为了保住玫瑰的残魂，叶尘便没有将她的肉身带走，不过他当时在那个溶洞之中，布下了层层禁制，想来应该不会有什么问题。
<br/>　　……
<br/>　　一天之后，叶尘终于找到了那处冰湖，而与此同时，华夏仙人逼迫米国投降的消息，也已经传遍全球每一个角落。
<br/>　　尽管米国政府，拼命的动用公关力量，想要掩盖这一事实，可根本无济于事。
<br/>　　非但如此，就连叶尘曾经的种种事迹，也被知道内情的好事者，在网上散播开来，顿时再次引起了巨大轰动。
<br/>　　“原来之前岛国首相，公开道歉，竟然也是他的手笔！”
<br/>　　“还有东南六国军队被袭击，天竺国金光寺被毁，狼王岛被核弹轰炸……竟然也全都因为他！”
<br/>　　“简直太强大了！这是真正的神灵！”
<br/>　　“我要修仙，要成为像华夏仙人那样的存在！”
<br/>　　……
<br/>　　就连叶尘也没有想到，因为他，全球已经不知不觉，掀起了一场修仙狂潮！
