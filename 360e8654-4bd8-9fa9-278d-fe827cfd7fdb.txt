        “我要相亲。”
<br/>
<br/>        傅夜沉淡漠地丢下这么一句话，而落以柔当场瞠目结舌了。
<br/>
<br/>        大哥居然要去相亲！！！
<br/>
<br/>        落以柔这一整天的心情，就像蒙上了一层雾霭，阴沉沉又无精打采。
<br/>
<br/>        傍晚时分，连华生是第一次来这种路边摊吃夜宵。
<br/>
<br/>        四方形的小矮桌、木板凳，几瓶啤酒、一些海鲜料理，很多人可以就这样消磨掉一段夜晚时光。
<br/>
<br/>        其实，落以柔也不经常来，只是今晚，她需要来一点啤酒，需要一个比较豪迈放松的场所。
<br/>
<br/>        “我本来以为你要为了报答我的调教之恩，要请我去‘夜惑’喝酒享乐听歌看美女。”连华生啧啧嘴，向落以柔抗议。
<br/>
<br/>        他今晚出门前还特意梳洗了一番，打算去“夜惑”里来一段艳遇，谁知这一会却只能坐在大马路边，陪着她落以柔喝着这低档啤酒，吃着全是调料味的海鲜。
<br/>
<br/>        “这里是哪不好？快炒的海鲜好吃，啤酒又冰凉，而且不必做作……”落以柔心酸地说道，“而我也就跟这路边摊一样，入不了你们这些豪门贵少的眼。”
<br/>
<br/>        连华生不禁露出一个古怪的表情，有些莫可奈何地说道：“我可不是这个意思。”
<br/>
<br/>        顿了顿后，连华生机敏地反问：“阿沉欺负你呢？”
<br/>
<br/>        “你知道这一刻他人在哪里吗？”落以柔一脸落寞的表情问。
<br/>
<br/>        “我如果要查，就一定能知道他现在在哪里。不过了，他此刻一定不在私宅。因为我打电话给刘婶说我不会回家吃饭的时候，刘婶跟我念叨大少爷今晚也不会回家吃饭。”
<br/>
<br/>        话虽如此，但是她落以柔不用查，就知道傅夜沉在哪里。
<br/>
<br/>        因为地点是她订的，时间也是她敲好的，是她亲自安排了这场相亲宴。
<br/>
<br/>        在六星级饭店的法国餐厅里，有由法国聘来的主厨、奥地利进口的精美瓷器、小提琴现场演奏、高雅浪漫的气氛、一流的餐桌服务……
<br/>
<br/>        这一顿大概要花上傅夜沉一笔大钱，但是，成功机率也会相对提升。
<br/>
<br/>        一想到这里，她的心更酸涩了。为自己所喜欢的男人安排他和别的女人相亲……还有什么比这更残忍的事？
<br/>
<br/>        “以柔，到底是怎么一回事？”连华生一本正经的问，“快说吧，你不就是找我出来‘倒垃圾’的？”
<br/>
<br/>        “大哥他……去相亲了。”落以柔叹了一口气之后，悻悻地说道。
<br/>
<br/>        “真的假的？”就连他连华生都震惊了。
<br/>
<br/>        三年了啊！
<br/>
<br/>        傅老太爷给他傅夜沉物色过无数个相亲对象，各个都是绝顶美女，而他傅夜沉却从未出席过，几乎每次把傅老太爷气得火冒三丈。
<br/>
<br/>        “当然是真的！相亲的时间、地点，全是我安排的。”落以柔说着说着，眼眶里浮上了心酸的眼泪。
<br/>
<br/>        “你放心吧！一定不会成功！”连华生回过神来后，安慰道。
<br/>
<br/>        落以柔却摇了摇头：“我觉得，大哥这次是认真的。”
<br/>
<br/>        “话说回来，你和阿沉相处了也有两个月了，他就从来没有想过要染指你吗？”连华生非常纳闷。
<br/>
<br/>        他可是照着童筱颖的言行举止去调教落以柔的，甚至还教会了落以柔许多连童筱颖都不会的事情。
<br/>
<br/>        包括傅夜沉的所有喜好，连华生可是一字不漏地全告知了落以柔。
<br/>
<br/>        落以柔不光是工作上精明能干，就能连生活上也把傅夜沉的事情安排得妥妥帖帖。
<br/>
<br/>        傅夜沉应该也能从落以柔的身上感觉到才对。
<br/>
<br/>        “连医生，你在胡说些什么呀！”落以柔顿时害臊地红了耳根低斥着连华生。
<br/>
<br/>        连华生还是不敢置信：“他真的、真的从来没有把念头动到你身上过？”
<br/>
<br/>        “大哥他不是那种人。”落以柔极力为傅夜沉辩解。
<br/>
<br/>        “那你觉得，他是哪种人？”连华生忽然间不太懂傅夜沉了，他特意“送”了这么一个与童筱颖几乎相近的女人放到他傅夜沉身边，他傅夜沉竟然不为所动？
<br/>
<br/>        “大哥是正人君子。”落以柔感慨道，“虽然只与他相处了两个月，他身边对他有意思的优秀女性也很多，但是他从来不乱搞男女关系，而且很洁身自爱。”
<br/>
<br/>        连华生却唉声叹气地回答道：“那是因为他早已心有所属。”
<br/>
<br/>        “你说什么？”落以柔没大听清楚连华生的话。
<br/>
<br/>        连华生回过神来，咧嘴一笑，敷衍道：“没什么！我还是陪你喝啤酒吧！今晚，我们俩不醉不归！”
<br/>
<br/>        童筱颖的事情，连华生在落以柔面前只字未提。
<br/>
<br/>        她已经是傅夜沉的过去式了，作为朋友，连华生还是希望傅夜沉向前看齐。
