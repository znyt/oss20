        童筱颖以为自己可以原谅傅昼景的，果然，那也只不过是她自己一时不甘心的执念。
<br/>
<br/>        两个人在一起久了，就会有种不愿意轻易放手的执着在心里作祟。
<br/>
<br/>        或许还爱着，彼此不知道，又或许不爱了，彼此还是不知道。
<br/>
<br/>        倘若，她真的为小卫生了孩子，然后抚养小尚轩，那么，她今后真的就能像以前那样，毫无顾忌地对小卫仍旧爱得纯粹吗？
<br/>
<br/>        答案是否定的！
<br/>
<br/>        她一定会爱得不纯粹了，甚至还会拿这件事情大做文章地跟小卫吵架。
<br/>
<br/>        又或许，她不会跟他吵架，只会跟他开冷战。
<br/>
<br/>        因为，她和小卫离婚，是主要的原因还是小卫受不了她一个星期地对他不闻不问。
<br/>
<br/>        其实，她和小卫在澳洲悉尼生活的这三年里，他们并不是没有吵过架。
<br/>
<br/>        吵过，也闹得很凶过，但每一次，都是她先妥协。
<br/>
<br/>        童筱颖记得印象最深的一次，是星期五那天下午，她刚从学校里回来，走到家门口发现地上掉了一张设计图。
<br/>
<br/>        这张设计图她是知道的，是傅昼景上个月就开始设计的一个工程项目设计图中的其中一张。
<br/>
<br/>        今天刚好是交标日期。
<br/>
<br/>        她拾起地上那张设计图，给傅昼景打了通电话，但是电话提示已关机。
<br/>
<br/>        为了保险起见，时间上又还来得及，童筱颖连屋都未进，直接拿着设计图，火急火燎地直奔傅昼景的公司。
<br/>
<br/>        她明明是好心，却把好事给做砸了。
<br/>
<br/>        傅昼景负责设计的项目，本来可以中标的，但是因为她童筱颖的到来，告知了设计图里少了一张，所以造成傅昼景和他的团队辛苦所做的标书，被评标人发现了这个大漏洞而导致废标。
<br/>
<br/>        那天回到家后，傅昼景气急败坏地给了童筱颖一巴掌。
<br/>
<br/>        他怒火冲天到了极限，忍无可忍地怒吼：“你为什么要把我的设计图交到前台？”
<br/>
<br/>        “前台的那个女人说，你们公司在开重大会议，闲杂人等不得入内，她还说可以帮我转交的……”
<br/>
<br/>        “筱颍，你平日里无趣也就算了，为什么到了关键时刻，你还如此无知？”傅昼景凶神恶煞地质问。
<br/>
<br/>        当时，童筱颖哑口无言了。
<br/>
<br/>        她知道，那次的竞标项目，关系到他的升职加薪。
<br/>
<br/>        那是傅昼景第一次动手打她，也是唯一的一次动手。
<br/>
<br/>        平日里那个温文儒雅、风度翩翩、温柔体贴的好老公，第一次因为工作上的不顺，在回家后判若两人，大动肝火地打人、摔东西。
<br/>
<br/>        虽然，他打了她，但是是她先低头认错。
<br/>
<br/>        那件事情，才就此作罢。
<br/>
<br/>        不过，事后，他也很认真、很诚恳地跟她道了歉，允诺从今以后再也不会动手打她。
<br/>
<br/>        所以，童筱颖也很快就不把这事放心上了，但是，这件事却莫名其妙地格外令她印象深刻。
<br/>
<br/>        回忆到这里，童筱颖站起身来，微笑着说道：“外婆，我改天再来看你。我今天买了一些瓜果蔬菜的种子，还买了一些花种，明天我再到山上去采一些治疗感冒的药草回来种着。我会把外婆的园子，重新打理。”
<br/>
<br/>        而这个时候，夏凌焰转了预付工资到童筱颖的手机上，备注是：小筱颖，模型做得不错，先给你预付一半的工资，接下来，要加油噢！
<br/>
<br/>        “呼——”童筱颖心里也终于松了口气，还好这个夏凌焰是个讲信用的主。
<br/>
<br/>        云城酒店，总统套房。
<br/>
<br/>        该死！
<br/>
<br/>        他的头，痛得快要炸了！
<br/>
<br/>        其实，他是千杯不醉的那种，但是昨天晚上为什么会那么轻易的就喝醉呢？
<br/>
<br/>        傅夜沉那英气逼人的剑眉紧紧地拢起，一手揉着太阳穴，一时之间还搞不清楚自己现在身在何处。
<br/>
<br/>        他吃痛地坐起身，长腿屈起，托着手肘，白色的被子只有一个角刚好遮住他的腹部以下。
<br/>
<br/>        “嗯——夜沉，别闹了，我不要了，让我再睡一会儿……”
<br/>
<br/>        忽然，被子下传来一个女人的声音。
<br/>
<br/>        傅夜沉冷不丁地打了个寒颤，浑身像触电了一般，裹着被子迅速从床上跳了下来。
<br/>
<br/>        而被子的另一端里，李忆莎撑了个懒腰，雪白的双臂和肩膀，毫无保留地露在了空气里。
<br/>
<br/>        只见她的脖子上、锁骨上、肩膀上，就连手臂上都有一抹又一抹的暗紫偏红的数不清的吻痕。
<br/>
<br/>        顿时，傅夜沉整个人懵了，更是傻眼了。
<br/>
<br/>        “嗯？夜沉，你怎么不睡呢？”李忆莎揉了揉惺忪的睡颜，打着哈欠，若无其事地坐起身来。
<br/>
<br/>        昨天晚上……
<br/>
<br/>        他……
<br/>
<br/>        好像又梦到了筱颖……
<br/>
<br/>        然后……
<br/>
<br/>        他霸道地要了筱颖……
