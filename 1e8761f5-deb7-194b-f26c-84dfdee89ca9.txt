　　第9章
<br/>　　多年游戏宅的经验告诉程大雷，开箱子是一件赌脸的事情。有可能开到白板道具也有可能直接开出顶级橙武，当然空箱子的可能也是有的。
<br/>　　不过自己今天这么旺，这种事肯定不会发生。
<br/>　　他净手焚香，冲着天空拜了三拜，最后搬过来一口木箱，用力打开。
<br/>　　“大家注意，见证奇迹的时刻！”
<br/>　　所有人都凑过来眼睛，发现箱子空荡荡什么都没有。程大雷讪讪：“千把万把不能赢第一把，这个道理没人告诉你们嘛，好运气都在后面。”
<br/>　　第二口箱子一样是空的，程大雷不信邪，一连开十口箱子，结果一无所获。尤其是最后一口，感觉特别重，程大雷心下激动，这是脱非入欧的绝佳机会，结果打开后才发现，箱子里是一堆石头。
<br/>　　这让程大雷不敢再开，心中郁闷，系统就算被自己刷BUG刷出阴影，也不该这么报复。
<br/>　　“大当家，让我开一口吧。”徐神机。
<br/>　　“你开你开，你脸比我还黑。”
<br/>　　徐神机激动的搬过来一口箱子，虽然不报希望，但程大雷还是递过去一个眼睛。
<br/>　　“有东西，有东西……”徐神机大叫，突然一怔：“这是什么东西？”
<br/>　　里面算是满满一箱子黑色粉末状的东西，程大雷捏了一把，忽然一激灵：“这是火药。”
<br/>　　“什么是火药？”秦蛮，徐神机，徐灵儿都困惑的问道。
<br/>　　程大雷没有回答，他一直不知道这个世界的生产水平处于什么程度，不过以程大雷的判断，几乎已经失去攀科技树的可能。第一自己文科生程度没有这个能力，而这个世界的生产力也没有这个土壤。
<br/>　　看三人都不认得眼前的黑火药，虽然有青牛山信息闭塞的原因，但至少也说明火药还没有被大规模运用。
<br/>　　有这火药助力，再仗着地理之便，并不是没有办法应对黑石城的攻击呐。
<br/>　　程大雷一激灵，道：“快把木箱都打开，看有多少这东西。”
<br/>　　徐灵儿三人开始了愉悦的开箱子游戏，有人开到东西，就发出一声兴奋的大叫。然而程大雷只能在一旁看得眼馋，没办法，非酋附身，程大雷不敢冒险。
<br/>　　最后一共开出十箱火药，其他箱子都空着。程大雷还需要思考，怎样才能发挥这火药的最大价值。
<br/>　　“箱子收好，我还有用。”
<br/>　　“大当家高见，这些都是好木头，我们冬天可以烧柴。”
<br/>　　“我其实很想把你烧了，留好，弄坏一个我拿你是问。”程大雷无奈摇摇头，忽然又想起一件事，问：“军师，咱们山寨的田地在哪里？”
<br/>　　“在后面坡上，都荒好几年了，我春天还撒下些麦种，结果什么都没长出来。大当家问这个干什么？”
<br/>　　“去看看。”
<br/>　　“天都要黑了，走那么远干什么，有好几里路呢。”
<br/>　　徐神机很懒，搬箱子已经搬得他很累，不过程大雷一意孤行，他也只能带路。
<br/>　　而等四人走到后面山坡时，徐神机整个都愣住。十亩连在一起的麦田，现在长满金黄色麦穗，山上那口本已干涸的山泉，此刻流淌着清冽的泉水。
<br/>　　一切都在程大雷的预料之中，现在最好一笔系统奖励自己已经看到了。
<br/>　　“神仙保佑，神仙保佑呐！”
<br/>　　徐神机忽然冲进麦田，脸上露出迷醉的表情，山寨那么多收获，都没令他露出这样的表情。秦蛮也露出笑容，憨憨的笑着。
<br/>　　程大雷惊住了，呆呆的看着这一幕。
<br/>　　“灵儿，你不要乱跑，把麦穗碰掉了。”
<br/>　　“这些都能收割，可以打上千斤粮食，这个冬天能熬过去了。”
<br/>　　“麦秸还能收了烧炕，冬天能少遭点醉。”
<br/>　　“得赶紧收，雨季都快要到了。”
<br/>　　三人站在麦田内，热络的讨论着，得，还都是内行。
<br/>　　“大当家，咱们连夜把麦子收了吧，一下雨可都泡汤了。”徐神机跑过来道。
<br/>　　“好啊，你们抓紧忙，把麦子收好，等黑石城的人来了，正好让他们拉走。”
<br/>　　徐神机无语了，他竟忽然跪下来，口中不停道：“神仙保佑，大恩大德。”
<br/>　　程大雷无语，不过几亩麦子，至于如此嘛。而更令他吃惊的是，连秦蛮和徐灵儿都跪在地上，说着同样的话。
<br/>　　看着他们脸上虔诚的表情，程大雷忽然明白些什么。
<br/>　　山寨，并不是如想象中那样，一伙凶神恶煞的恶人，杀人不眨眼，没事下山抢一遭，大块吃肉大口喝酒。事实并不是如此，山寨有时候更像一个村庄，有老人有小孩，有人开荒种粮，有时候还会下山买些生活必须品。
<br/>　　比如程大雷眼前这三位，说他们是山贼真真是抬举他们。
<br/>　　像徐神机，这两天程大雷也摸清他的来历。徐神机在山下的时候，是一个村子里的私塾先生，说是私塾先生，也就是勉强认字的程度。他最大的娱乐，就是攒点铜板，换一壶酒，盯着村里寡妇的胸脯流口水。
<br/>　　永远是面目可憎，惹人生厌，无论在山下还是在山上。
<br/>　　某天村里一恶霸盯上徐灵儿，恶霸也不是特别厉害角色，不过是七拐八拐的亲戚，是城里某个贵族的仆人。可这在村里就没人敢惹，更不必说浑身上下没有三两肉的徐神机。
<br/>　　有一天，徐神机把徐灵儿打扮得漂漂亮亮的，将恶霸请到家中，赊钱买了酒，宰掉家里下蛋的老母鸡……
<br/>　　“灵儿跟了您可就有福气啦，我以后也能沾点福气。”
<br/>　　“您看我灵儿长得多俊，脸蛋多白，肯定能伺候好你。”
<br/>　　“今天你们就洞房，放心，我给你们把门……哈哈……”
<br/>　　……
<br/>　　……
<br/>　　在将恶霸灌得大醉的时候，徐神机用菜刀砍下对方脑袋，带着徐灵儿一路逃出来，几经波折，沦落到蛤蟆寨。
<br/>　　这也许是徐神机平生第一次用计，也肯定是唯一成功的一次，但程大雷真心觉得……干的漂亮。
<br/>　　最后直到日沉西山，三人才恋恋不舍离开麦田，四人一起回到山寨。
<br/>　　“爹，我忘了摘点麦穗回来，磨成面就可以给你们烙饼。”
<br/>　　“烙饼，要不要这么奢侈！一顿饼加上榆树叶可以吃好几天呢。”
<br/>　　程大雷不得不打断三人兴奋的聊天，他扯出那面大旗，终于把徐神机的注意力吸引过来。
<br/>　　“咦，这不是咱们山寨原先的大旗吗，我以为被谁改裤子了，今天竟然也找到了。”
<br/>　　“我就像问问，这图案是谁设计的，一个蛤蟆，头上顶个蛋，是想表达蛤蟆寨顶个蛋用吗？”
<br/>　　“那不是蛋，那是月亮，这一幅天蟾吞月图，被大当家你这么一说……”
<br/>　　“挂上！”程大雷把大旗丢给秦蛮。
<br/>　　秦蛮矫健如老猿，三两下爬到旗杆上，将大旗挂在顶端。
<br/>　　等他下来的时候，四人都仰头望着大旗，被夜风展开，大旗猎猎而舞，上面的癞蛤蟆似乎动起来，冲着头顶的月亮露出贪婪的目光。似乎想要表达一个意思：
<br/>　　就是要做只癞蛤蟆，就是要吞月亮，吃天鹅肉！
<br/>　　四人心中似乎某种东西被连接，山寨有了粮食，大家可以在这个冬天活下去，只要，只要能撑过眼前的危机，四人眼中恍惚有火光闪烁，不约而同心里想的都是一句话。
<br/>　　『这是我的地方，我要守住他』
<br/>　　“自己这算是揭竿而起嘛？”程大雷心底默默想，就在这个时候，系统给出两条提示。
<br/>　　嘟，正义的少年喔，你的大旗已经挂起，四方英雄将会闻风而来，投到你的麾下。
<br/>　　嘟，接到任务募集人口，将山寨人口提升到100，奖励抽奖次数1 。
<br/>　　
<br/>(本章完)
