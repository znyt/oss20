<br/>        “可是，娱乐圈里不是很复杂吗？你就不能帮我给洛羽保驾护航吗？”连星儿闷闷不乐地问。

<br/>        傅夜沉最终还是答应了：“好，爹地答应你就是。”

<br/>        “嗯，那提前跟爹地说晚安喽！”

<br/>        “晚安，爹地。”

<br/>        连星儿和爹地傅夜沉通完电话后，又拿着英语词典一边背记单词，一边继续做瑜伽。

<br/>        差不多的时候，她拿起睡衣去洗了个澡回来，手机里，后援会的群，大家都在向群主报平安。

<br/>        趴在白色的大床上，连星儿双手托起脖子上挂着的小纽扣，嘴角情不自禁地上扬，脸上的笑容格外甜。

<br/>        另一边的房间里，林苏还是花钱请人查了连星儿的底细。
<br/>
<br/>        毕竟，她有些怀疑这丫头会不会对他们撒谎。
<br/>
<br/>        连星儿的身份证信息是她在后援会的群主那儿拿到的，资料发出两个小时候，对方回给她的邮件里说查不到，还把钱也一并退给了她。
