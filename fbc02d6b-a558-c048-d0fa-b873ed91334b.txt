　　教堂外剩余的六个警卫队员心中打起了退堂鼓，与其就这么送人头，还不如趁着混乱立刻离开，继续留在这里也不能改变什么，有狙击手在教堂钟楼上虎视眈眈的盯着，他们是绝对不可能进去教堂并且带走塔尼特，而且在教堂钟楼上有狙击手出现，就不能排除在教堂内还有其他人保护着塔尼特。
<br/>　　那六个警卫队员心中已经认定出现在教堂中的武装力量应该是比洛多找来保护塔尼特的雇佣兵，所以他们也认为应该把这个情况报告给巴坦伦，让巴坦伦再派出援兵来。
<br/>　　一个藏在吉普车背后的警卫队员从口袋中摸出一个老款诺基亚手机，在上面按了一个号码，满脸是汗，眼睛中还带着对教堂内狙击手的惊恐，向巴坦伦汇报这里的情况，“将军，比洛多找来了雇佣兵保护他的女儿，我们被教堂内的狙击手拦住了进不去教堂，另外我们有两个人已经死了。”
<br/>　　“雇佣兵！？”
<br/>　　巴坦伦从书桌背后站起身来，对着电话那头的警卫队员怒声质问道：“为什么会有雇佣兵，比洛多什么时候找了雇佣兵！？”
<br/>　　这个问题巴坦伦不知道，他手下的警卫队更加不可能知道。他今天特意给比洛多安排了一些工作，拖住了他，方便自己的警卫队去把塔尼特带走，这本来应该是一件很简单的事情，可是却没想到发生了意外。
<br/>　　“是的，我可以肯定他们一定是雇佣兵。他们的装备先进，而且实力很强，应该不是比洛多手下的人。”
<br/>　　说着，那个拿着手机的警卫队员还探头往外看了一眼。警卫队员的脑袋刚从吉普车背后露出只有拳头大小的时候，一颗子弹带着尖锐的破空声就朝着他飞来，直接把他的头盖骨都给掀开，血浆喷射而出，黏哒哒的糊在吉普车的车窗上。
<br/>　　第三个！
<br/>　　头盖骨被掀飞的警卫队员，瞪圆着眼睛栽倒在地上，他手中那部还传来巴坦伦质问声的手机也是滚落到一旁。
<br/>　　巴坦伦拿着电话大声质问了几遍都没有听到回复，知道又出了事情，巴坦伦放下电话，按下桌子上的对话机，叫来了自己的警卫队长，“立刻再派两队人去教堂，那边出事情了，比洛多找来了雇佣兵保护他的女儿。”
<br/>　　“是！”
<br/>　　警卫队长对着巴坦伦敬了一个礼，然后转身走出了房间，打开门，看到手中端着一个盘子的詹弗妮，愣了愣神，随后恭敬地称呼道：“夫人！”
<br/>　　巴坦伦身边的人都知道，詹弗妮上位是早晚的事情，现在是因为大选的事情，巴坦伦需要注意影响才没有把詹弗妮扶正，大选结束，等到巴坦伦成功当选下一届总统，詹弗妮肯定会是总统夫人。
<br/>　　没有人会怀疑巴坦伦会在这次大选中失败，不管是巴坦伦自己，还是巴坦伦身边的人。
<br/>　　詹弗妮对警卫队长对自己的称呼很满意，轻轻点了点头，端着给巴坦伦准备好的水果朝着巴坦伦的书房走去。巴坦伦的书房，在没有巴坦伦允许的情况下，是不允许他人随便进入的，而在这份不许进入的名单中，显然并不包括詹弗妮，她可以随时随地进入这里。
<br/>　　把手中的果盘放在书桌上，顺势坐在巴坦伦的大腿上，看着巴坦伦那张油腻腻的肥脸，还有他嘴里散发出来的带着烟焦油的臭味，詹弗妮眼中闪过一丝厌恶，但这丝厌恶只是瞬闪即逝，巴坦伦根本没有捕捉到。
<br/>　　詹弗妮跟巴坦伦之间是爱情吗？
<br/>　　显然不是！
<br/>　　詹弗妮年轻美丽，绝对不可能看上巴坦伦这个肥的跟个肉球似得的死胖子，每次被自己眼前这个死胖子压在身下的时候，詹弗妮每次都要装得很开心，很满足，而每次巴坦伦都只是满足了自己而没有满足詹弗妮。
<br/>　　如果不是因为巴坦伦的权利和他的钱财，詹弗妮压根就不会多看他一眼。比起巴坦伦这个死胖子，布克尔特更具有男人味，可是詹弗妮暗中给布克尔特递了几次信息，布克尔特却不为所动，这个男人好像对女人根本不感兴趣一样。
<br/>　　“事情很棘手吗？”詹弗妮对巴坦伦问道。
<br/>　　“去教堂的那些人遇到了麻烦。”巴坦伦回答道：“比洛多请了雇佣兵来保护他的女儿，派去的人遭到了狙击，出现了伤亡。”
<br/>　　“雇佣兵？”詹弗妮纤细的眉毛轻挑，说道：“比洛多居然会招来雇佣兵？”
<br/>　　“我也觉得有些不可思议。”巴坦伦的肥手在詹弗妮光滑的皮肤上抚摸着，眼睛中闪烁着令人觉得刺骨和心悸的寒光，说道：“可是就算有雇佣兵保护，也改变不了什么，比洛多这次肯定要向我妥协，他手里的部队和他的女儿只能选择一样！”
<br/>　　詹弗妮不敢公然跟别的男人眉来眼去，还要在巴坦伦面前装作自己很满足的样子，不断地来演戏讨好巴坦伦，还是因为巴坦伦的手段。巴坦伦的手段可不像他在床上的时候那么没用，如果让他察觉到异常，詹弗妮知道自己一定会很惨，到时候可能连一个站在路边的廉价妓女都不如！
<br/>　　……

<br/>　　齐凡通过耳麦告诉教堂内的莱昂和布谷撤离，负责保护塔尼特的莱昂对塔尼特说道：“塔尼特小姐，我们必须要离开这里了，巴坦伦的增援部队应该很快就会抵达，到时候情况就会变得很危急。”
<br/>　　“难道你们就不能留下保护这里的人吗？”塔尼特对莱昂问道。
