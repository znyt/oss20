　　第728章
<br/>　　都是心冷之人，并不会为个本无多少情份的表妹下死力气。但终究有几分属于无可奈何的难过，默不作声的坐下，眉眼低垂，任谁也能看见她眸子里的悲伤。
<br/>　　程大雷悠悠叹了口气：“美人一笑，烽火千里，当真有人宁要美人，不要江山。来来来，掀开面纱与本当家笑一个，烽火诸侯是不敢想了，救个把人还是没问题的。”
<br/>　　崔白玉瞧着程大雷打趣的眼神，也不知他说得是真心还是假意。她呆坐在那里，当真不知该如何是好。
<br/>　　长街上，年轻将军赶着犯人，不时用手中鞭子抽打在犯人身上。犯人畏惧得身体颤抖，发出惨叫，如此换来兵丁的哄笑。
<br/>　　程大雷一动不动盯着崔白玉，道：“美人，来，与本大爷笑一个。”
<br/>　　黑纱下，崔白玉咬紧牙关。她生得花容月貌，倾国倾城，自己心里其实也明白这一点，否则也不会一直以黑纱遮面。但崔白玉偏偏不是个靠脸吃饭的主，她美貌不假，但从来都是以才学换得地位尊重。让她卖笑换来男人为自己做事，第一，这种事她从未做过，第二，这也与她做人的道德不符。
<br/>　　有人可以一双藕臂千人枕，也有人沾衣裸袖便为失节，偏生崔白玉是后者。
<br/>　　这关头，自己的才学，自己的识人之能统统起不到作用。终究，自己要回到一个女人的身份上，用娘胎里带出来的那点东西愉悦男人，以换取某些东西。
<br/>　　虽然黑沙遮盖着，但她的脸色已变得苍白，内心正在进行剧烈的天人交战。一双手紧紧攥起拳头，想要抬起手却感觉双臂有千钧之重。
<br/>　　“呔。”
<br/>　　程大雷忽然一喝，却是吓得崔白玉三人都是一楞。尤其是崔白玉，惊慌抬起头来，心道：莫非自己惹他不满，这尊魔头要发作了。
<br/>　　“你说巧不巧，偏偏本当家还是个怜香惜玉的主儿，最是见不得美人皱眉。”
<br/>　　说着话，程大雷接下来的举动令人大吃一惊，拎起面前的茶壶隔着窗户砸了出去。
<br/>　　那年轻将军刚上任不久，正是乍穿新鞋高抬脚的时候。这帮犯人在他眼里与畜生差不多，随他打骂。鞭子举起，刚要抽在一个女孩身上，冷不防一物从天而降，稳准狠的砸在年轻将军头顶上。
<br/>　　水花四溅，瓷片纷飞，茶水里夹杂着血水，年轻将军直接从马背上被砸了下来。
<br/>　　“刺客，有刺客。”
<br/>　　兵丁大惊，拿着兵器四处张望，想知道他们心中的『刺客』藏在何处。
<br/>　　程大雷冲鱼洞真拱了拱手：“请了，将那女孩带上来。”
<br/>　　一道黑影从窗子里窜出，飘飘落在地面上，兵丁大惊，立刻将她围在当中。
<br/>　　鱼洞真抓住女孩双臂，将其抱在怀里，脚尖轻轻一点地，便跃入二楼茶馆。
<br/>　　崔白玉倒抽了一口凉气，她曾调查过程大雷，但关于黑衣老尼的资料并没有收集到多少。一路上黑衣老尼几乎不说话，崔白玉也想不明白程大雷为何要将她带在身边。直到今日才明白，黑衣老尼竟有如此出神入化的本事。
<br/>　　程大雷麾下，还真是高手如云呐。
<br/>　　那叫乔弄云的姑娘十六七岁年纪，如今已被折磨的形神俱销。双目涣散，神情麻木，只有见到崔白玉眼神中才有一丝光芒。
<br/>　　“姐姐……”
<br/>　　崔白玉抿紧双唇，轻轻抚了抚她的头顶。乔弄云悲吟一声，晕倒在她怀里。
<br/>　　“刺客在茶楼，刺客在茶楼。”
<br/>　　大街上传来兵丁的大叫，然后有急促的脚步声越来越近，一帮兵丁闯进茶楼，直扑二楼。茶楼的掌柜与店小二都吓得窝在柜台里，抱住脑袋身体不住颤抖。
<br/>　　刘发财已握住刀，看向程大雷，以眼神询问他该如何行事。
<br/>　　那脑袋被程大雷开了的年轻将军闯进来，二楼没有他想象的杀机，只有几个人围桌喝茶。
<br/>　　“什么人，瞎了你的狗眼，不知老子是谁么。”
<br/>　　程大雷拿起一根筷子，挥手丢了出去，打在对方膝盖上。对方膝盖一疼，忍不住单膝跪地。
<br/>　　“凭你也想问我名字，叫这座城的城主来，否则有人敢上二楼，来一个跪一个。”
<br/>　　年轻将军恨得咬碎一口钢牙，想要站起来，发现左腿像是灌了铅一般，如何也站不起来。
<br/>　　“与我拿了。”
<br/>　　众兵丁一拥而上，挤过楼梯向程大雷扑来。程大雷抓起一把筷子，挥手打了出去。这些无灵的竹筷仿佛长了眼睛一般，专打对方膝盖。噗通噗通，在二楼地板上跪倒一片，却没有一人能靠近程大雷五步之内。
<br/>　　崔白玉眯起眼睛，她不会功夫，但眼力是有的。收集的资料中，说过程大雷本领奇高，但没有说他高到这种程度。飞花摘叶皆可杀人，怕已到超凡脱俗的境界。
<br/>　　一行五人，程大雷、徐神机、刘发财、鱼洞真包括崔白玉，谁也没有对这一幕露出大惊小怪的表情。
<br/>　　徐神机打了个哈哈，道：“好啦，小孩子闹够了就把家大人找来，这里不是你们撒泼的地方。”
<br/>　　毕竟是吃过见过的主儿，徐神机便是到了朝廷也会被奉为上宾，他眼里自然也不搁一座汝南城。
<br/>　　楼下，掌柜和店小二依旧趴着不敢动弹。掌柜恶狠狠瞪了店小二一眼：“你是将什么样的人引了进来？”
<br/>　　店小二苦着一张脸：“我哪里知道……”
<br/>　　这些兵丁在二人眼里，俨然就是魔王一般的存在，吃人都不吐半根骨头。但在这伙客人眼里，他们就像是小孩子一样，手里的兵器也不过是小孩子的玩具。
<br/>　　楼上的人大概也清楚了这伙人不好对付，急忙忙有人跑出去，召集更多的人来，请高人出手收拾程大雷这伙人。
<br/>　　没有多长时间，便有几波人马赶到，将茶楼围了一个水泄不通。
<br/>　　那年轻将军直到现在也没站起来，也不知道程大雷施了什么魔法，他左腿好像不听自己使唤一样。
<br/>　　这对他是极大的屈辱，抬起头恨不得将程大雷千刀万剐。男人剁碎了去填粪坑，女人一定要好好留着，日日夜夜承受自己的怒火。
<br/>　　等等，面带黑纱……莫非，是她？
<br/>(本章完)
