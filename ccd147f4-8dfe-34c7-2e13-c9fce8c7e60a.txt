　　楚风眠的话中，森森杀意，毫不掩饰。
<br/>　　他已经是真的动了杀意，楚风眠给人机会，向来只有一次。
<br/>　　既然现在他们不准备臣服，那楚风眠，可也就再也不手下留情了。
<br/>　　“想要杀了我们？小子，你真以为我们秦皇国度皇室可以屹立千年，没有底牌吗？”
<br/>　　秦莽圣者看着楚风眠，冷哼一声说道。
<br/>　　“老夫承认你的实力，甚至是比起超无敌，都要恐怖许多，的确是千年难见的绝世天才，但是你太过狂傲了！你的狂傲，会让你今天灭亡的！”
<br/>　　秦莽圣者怒声说道。
<br/>　　“一起出手，唤醒地脉之力加持！今日我们就算是拼劲一切，也要杀了他！”
<br/>　　“好！”
<br/>　　五尊巅峰圣者，猛然出手，唤醒地脉之力，一股股地脉之力，便是加持在了他们五人的身上。
<br/>　　这五人，皆是有着皇室血脉，可以唤醒这秦皇国度之下的地脉，如今地脉之力加持。
<br/>　　如今一个个的实力，丝毫不在当时巅峰的秦皇大帝之下。
<br/>　　“狂傲小辈！受死吧！”
<br/>　　感受到力量暴涨，五尊巅峰圣者，同时出手。
<br/>　　一股股灵力爆发而出，周围的空间，甚至都是要被震的破碎。
<br/>　　这五人的实力，已经是无限接近到了生死境，只怕是再过一会，就会引起天劫注意，触发生死大劫。
<br/>　　“天地霸主！”
<br/>　　王风老祖率先出手，一身霸者之气，爆发的淋漓尽致。
<br/>　　他所修行的，乃是无上霸道，如今地脉力量加持，令他自信心大增，无上霸道的力量，一下子体现了出来。
<br/>　　令他看来，仿佛是如同一方霸者一般的存在。
<br/>　　天地霸主这一招一出，整个空间，仿佛都要臣服，威力绝伦，恐怖到极致，跟那当初秦皇大帝打出的劫雷，力量都在伯仲之间。
<br/>　　加持了地脉，便是让这些人拥有了一个“小神脉”一样的存在。
<br/>　　虽然远远做不到神脉一般的神奇，拥有无穷无尽的灵力，但是却也可以加持地脉的力量，一招出手，引动地脉，力量比起平日里还要强大数倍。
<br/>　　这等实力，便是一尊巅峰圣者，在加持了地脉的力量后，爆发出来的力量。
<br/>　　这一招压迫过来，足以是让许多巅峰圣者，都抵挡不住，赵无敌都是因为忌惮这秦皇国度皇室的地脉之力，在踏入生死境之前，也不敢来找皇室的麻烦。
<br/>　　王风老祖这一招，也是必杀，要将楚风眠，一掌轰杀殆尽。
<br/>　　但是楚风眠的双眼，早已经是看破了这王风老祖的这一招，他也同样是一掌轰出，力量相对，居然是隐隐约约的伯仲之间。
<br/>　　楚风眠跟王风老祖各退一步，这王风老祖的脸上，却是露出了几分得意笑容。
<br/>　　“果然，这小子的力量在强大，也还是在神力境的范畴，我们如今加持的地脉之力，跟他的力量已经是相差不远，现在我们一起出手，将他活活压制住，轰杀了他！”
<br/>　　王风老祖大笑道。
<br/>　　剩下的四尊巅峰圣者，也是同时出手。
<br/>　　猛然之间，整个宫殿中，都是被无数武技所充斥，一招一式的，冲着楚风眠打了过来。
<br/>　　令楚风眠都只能是疲于招架，居然是无法反击，仿佛是被死死的压制住了。
<br/>　　“小子，尝到了被压制住的滋味了吧！今日你必死！”
<br/>　　夏祖长老，疯狂的怒喝道，他一肚子火气，如今终于是得意释放爆发出来。
<br/>　　瞬息之间，他便是数十道武技，同时爆发。
<br/>　　有着地脉的力量加持，他打出武技消耗的灵力，只是寻常的数分之一，所以如今他根本不需要担心灵力耗尽的问题。
<br/>　　无穷无尽的武技，都在向着楚风眠，轰杀了过来。
<br/>　　而那些一旁的圣者们，看到楚风眠被压制住了，也是一个个面露激动，同时出手。
<br/>　　他们的力量，如果单一爆发出来，甚至是就连伤到楚风眠都不可能。
<br/>　　可是在现在楚风眠被压制住的情况下，楚风眠终归会出现防守漏洞，他们的武技，便是一个个极为致命了。
<br/>　　落井下石。
<br/>　　痛打落水狗。
<br/>　　这些圣者，一个个都想要杀了楚风眠，好分楚风眠奇遇的时候，也可以分一杯羹。
<br/>　　楚风眠一直在尽可能的抵挡着无数武技，看起来仿佛是稍显败势，但是他的神识，始终是注意着周围的一举一动。
<br/>　　突然之间，就在皇城之上，一道巨大无比的劫云，突然出现。
<br/>　　这劫云，足足是有着那秦皇大帝的劫云的两三倍大。
<br/>　　这五尊巅峰圣者的生死大劫，终于是降临了。
<br/>　　他们如今动用全力出手，终于是将生死大劫引动了。
<br/>　　这也是楚风眠一直在等待这的。
<br/>　　楚风眠嘴角露出一抹笑意，看着那五人平静开口道。
<br/>　　“生死大劫出现了。”
<br/>　　“那又如何？小子，今日我们五人就算是拼上性命，也一样会杀死你！”
<br/>　　夏祖圣者看着楚风眠，不以为然的说道，语气中还有着几分嘲笑。
<br/>　　他还以为楚风眠是将那生死大劫当做了救命稻草一样。
<br/>　　生死大劫降临，也不是一朝一夕的事情，在这段时间内，他有自信，可以将楚风眠彻底灭杀。
<br/>　　“小子，别以为生死大劫出现就可以救你，今日你是必死无疑！别以为我们没办法杀了你！你现在已经是灯油枯尽了！我们再一次联手，你就是必死无疑！”
<br/>　　秦莽老祖愤怒到极点的声音响起，为了楚风眠，他们已经是引动了生死大劫。
<br/>　　今日他们必须要拉上楚风眠来陪葬。
<br/>　　“是吗？灯油枯尽？”
<br/>　　听到秦莽老祖的话，楚风眠眼角突然露出几分笑意，突然之间，楚风眠的身躯中，本来所剩无几的灵力，一下子充满了，状态又恢复到了最好的一刻。
<br/>　　楚风眠有着神脉，从来不担心灵力耗尽的问题。
<br/>　　刚刚他故意压制住了大部分的灵力，造成了一副灯油枯尽的状态，就是为了引这五尊巅峰圣者全力出手，好引来生死大劫。
<br/>　　这五人的性命，对楚风眠来说不重要，他看重的，是生死大劫。
<br/>　　应该说，是这劫云核心的造化之力。
