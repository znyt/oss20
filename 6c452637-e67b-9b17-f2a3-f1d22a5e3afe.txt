　　“嗖。”
<br/>　　一道黑色的巨型影子飞速掠来。
<br/>　　萧景不为所动，萧壮萧子木和另外两个家族子弟则如临大敌。
<br/>　　“噬火百刃。”萧逸冷喝一声，一道火刃脱手而出，将黑色巨影断成两截。
<br/>　　萧景眼前一亮，道，“那是二级妖兽，铁背巨蛇，实力相当于人类的后天一重武者。”
<br/>　　萧逸一行人已经离开萧家矿脉数十里，陆陆续续遇到了不少妖兽的袭击。
<br/>　　“这已经是遇到的第八次妖兽袭击了吧，陨星山脉内，果然是妖兽横行。”萧逸沉声说道。
<br/>　　妖兽敌视人类武者，但它们拥有灵性，很凶残也很狡猾。
<br/>　　无论是大型矿脉还是小型矿脉，一般人类武者集中的地方，很少能见到妖兽。
<br/>　　它们会在少有武者足迹的地方。
<br/>　　萧逸等人才刚刚离开数十里，更加深入陨星山脉，已经见到了不少妖兽。
<br/>　　当然，这些妖兽有强有弱。
<br/>　　弱的妖兽，见到萧逸一行人人多，会毫不犹豫地远离。只有一些强大的妖兽才会来找麻烦。
<br/>　　同时，路上也遇到了零星一些猎妖师。或单独行走，或成群结队，在山脉中小心翼翼地行走着。
<br/>　　单独行走的猎妖师，必然实力过人。
<br/>　　妖兽内丹，妖兽尸体，都是非常值钱的东西。
<br/>　　人类武者为了这些东西，不惜涉险前来捕杀。
<br/>　　当然了，人为财死，鸟为死亡。这些武者，也有很多葬身妖兽腹中。
<br/>　　总之，在这偌大的陨星山脉内，每天每时每刻都在经历着战斗。
<br/>　　或妖兽与武者斗，或妖兽与妖兽相斗，甚至武者与武者为了利益而争斗。
<br/>　　人命，成了这里极不值钱的东西；血腥和杀戮，成了这里的家常便饭。
<br/>　　危险，充斥着整个山脉。
<br/>　　“少家主。”萧景忽然道，“我观你对噬火百刃的熟练度，应该要不了多久就能凝聚出第二道了吧。”
<br/>　　“萧景执事好眼力。”萧逸道，“我正在尝试凝聚第二道，应该快成功了。”
<br/>　　萧壮等家族子弟面露崇拜之色，“噬火百刃可是我们萧家最强大的武技，我们现在连学习的机会都没有呢，少家主却已经准备凝聚第二道了。”
<br/>　　曾经的家族废物，如今在他们眼中是最耀眼的星芒。
<br/>　　但他们没有不服气，反倒是对萧逸更加地信服。
<br/>　　一行人继续赶路，数个时辰后，萧景停下了脚步，道，“我们到了。”
<br/>　　恰在此时，一道人影从一颗大树上跳下。
<br/>　　“萧景执事，少家主。”人影对着萧景和萧逸行了个礼。
<br/>　　这是萧家探路队的队员，一直在附近放哨。
<br/>　　在这个队员的带领下，一直往密林深入了数百米，沿途陆续有探路队的人从树上跳下，足有八人。
<br/>　　萧逸淡淡道，“不愧是萧家的探路队，个个都很警惕，借助密林藏身的本事也不错。”
<br/>　　萧景饶有兴致地问道，“哦？少家主你也有在密林行走的经验？”
<br/>　　“呵呵，有那么一点儿。”萧逸随口搪塞了一句。
<br/>　　当年在地球时，他什么险地没去过。危险的亚马逊森林、神秘的大雪山、致命的撒哈拉沙漠，他都一一横穿了，自然各种行走经验极为丰富。
<br/>　　忽然，一位队员指向前方，兴高采烈道，“看，那就是妖血炎心果。”
<br/>　　萧逸抬头看去，前方有一颗奇怪的大树。
<br/>　　大树高约数米，枝繁叶茂。但怪就怪在，它浑身火红，连叶子也是火红色的，宛若一株冒火的树苗。
<br/>　　而在交错的叶子中，却有三颗红色的果子。
<br/>　　果子的红色，完全异于树叶。
<br/>　　树叶是火红色，而果子却是血红色。
<br/>　　那浑圆的灵果，仿佛是被无数血液包裹着的一滴巨大血滴。
<br/>　　那就是妖血炎心果。
<br/>　　一眼看过去，三颗妖血炎心果给人的第一感觉是妖异，妖异得让人心惧。
<br/>　　“咕。”萧壮等家族子弟更是咽了口口水，有些发怵。
<br/>　　至于萧逸，则仍旧是那副淡淡的脸色。
<br/>　　萧景看了一眼，暗道，“不愧是少家主，我们萧家的第一天才，竟镇定如斯。我当初第一次看到这灵果都被吓了一跳呢。”
<br/>　　“看这灵果的成熟度，应该要不了几个时辰就能采摘了。”萧逸忽然说道。
<br/>　　“额。”萧景反应过来，笑道“嗯，原本我以为这些果子起码还要两天才能成熟呢。不过，少家主你可是炼药师，对这些天材地宝可比我熟悉多了。”
<br/>　　密林中，人的视线本来就很有限。这棵大树生长在这种地方，若不是萧景等人恰好遇到，恐怕外人永远都发现不了它的存在。
<br/>　　萧逸一行人只能继续在密林中耐心地等着。
<br/>　　大概过了两个时辰，萧逸听到一声异动。
<br/>　　寻声看去，竟是大树下方的泥土发生了异变。
<br/>　　大树周围一大片的泥土，顷刻间化为了腥红色的血土。血土中，逸散出一阵阵血气。
<br/>　　那血气如同气泡，发出‘噼啪’的低沉响声。
<br/>　　血气从泥土中冒出后，自动蜿蜒环绕，最后缠住了树干。
<br/>　　树干顷刻间从火红色变为了腥红色。
<br/>　　再之后，连叶子也成了腥红色。
<br/>　　萧壮等人面露惊慌，讶异道，“泥土怎么冒血了？还有那颗大树，它在吸收这些血气。”
<br/>　　萧逸摇摇头，沉声道，“不，真正在吸收血气的，是那三颗妖血炎心果。它只是在通过树干和树叶吸收罢了，最后还是会凝聚到果子上。”
<br/>　　萧逸继续道，“血气出土，融于树身，三颗妖血炎心果越发妖异血红，这是它们成熟的迹象。”
<br/>　　“可以采摘了吗？”萧景一喜，问道。
<br/>　　“可以了。”萧逸点点头。
<br/>　　然而，正当萧景准备动时，萧逸却一把拉住了他，“先别动。”
<br/>　　“怎么了？”萧景疑惑地问道。
<br/>　　萧逸指了指前方，眼尖的他发现，那里竟有几道薄薄的脚印，“那些脚印是怎么回事？”
<br/>　　萧景放松下神色，道，“我还以为少家主问什么呢，那些脚印或许是其他队员曾经靠近过吧。”
<br/>　　“不可能。”萧逸脸色变得肃然，道，“那些脚印很浅，我都差点没看出来。很显然是有人曾经走到那里，然后又特地掩盖了脚印。”
<br/>　　“所以，那些脚印才会这么浅，甚至只留下一道淡淡的痕迹。”
<br/>　　萧景也看了一眼，对身后的队员问道，“你们曾经靠近过那里吗？”
<br/>　　那些队员连忙摇头，道，“不曾，妖血炎心果事关重大，没有队长你的吩咐，我们岂敢随意靠近。”
<br/>　　闻言，萧逸瞬间一惊，道，“这证明曾经有人比我们更早地发现了这三颗妖血炎心果。”
<br/>　　萧景眉头一皱，道，“不管了，为免夜长梦多，我先把它们采摘下来。”
<br/>　　“别冲动。”萧逸低喝一声，想拉住萧景。
<br/>　　然而，萧景毕竟是后天七重武者，速度太快了。不管萧逸的提醒，一把跃了过去。
<br/>　　“少家主不用谨慎过头，或许你只是看错了，我采摘完马上就回来，不会有事的。”萧景笑着跃了出去。
<br/>　　萧逸一向谨慎，他的洞察力和敏锐直觉都远超萧景，他绝对不会看错。
<br/>　　果然，正当萧景快要来到大树面前，准备采摘时。
<br/>　　大树之后，忽然露出了一声尖锐的嘶吼。随即，一道庞然大物忽然出现。
<br/>　　那是一条巨蟒，浑身银色，气势惊人，显然不是善茬。
<br/>　　“不好，那是三级妖兽，银背妖蟒，实力不亚于先天武者，快退回来。”萧逸大喝一声。
<br/>　　萧景可不是傻瓜，当看到银背妖蟒时，甚至不用萧逸提醒，已经立马后退。
<br/>　　银背妖蟒立刻追击，它那血盆大口，一口就能把萧景吞了。
<br/>　　“该死，你们在这等着，我去救萧景执事。”萧逸吩咐一声，立刻跃出。
<br/>　　“噬火百刃。”萧逸一道火刃打出。
<br/>　　散发着炽热高温的火刃堪堪打中了银背妖蟒，让它顿了顿，减缓了速度。萧景趁机立马加快速度后退。
<br/>　　恰在此时，忽然，大树后方，忽然有一道人影掠过。人影的速度很快，一把采过三颗妖血炎心果，而后立刻逃遁。
<br/>　　“什么人？”萧逸一惊。
<br/>　　“是慕容千军。”萧景愤怒地吼了一声。
<br/>　　“哈哈哈哈。”那人影，正是消失了一段时间的慕容千军，边逃遁边大笑道，“一群傻瓜，要不是你们帮我引开了这条银背妖蟒，我还不知道怎么采摘这三颗妖血炎心果呢。哈哈哈哈。”
<br/>(本章完)
