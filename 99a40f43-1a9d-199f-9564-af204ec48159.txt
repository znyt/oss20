　　第228章
<br/>　　这是枪和刀的交锋。
<br/>　　两马对冲，马头相遇也相撞，刀和枪仿佛要撕裂空气，疾驰，挥砍，直逼要害，两匹马齐声发出长啸，算是一个回合的结束，然后酝酿下一次冲锋。
<br/>　　没人想到，以诗词闻名的屈九江用的竟然是一柄重五十二斤的大刀，而柳轻名用的是一柄黑铁重枪。
<br/>　　程大雷暗暗心惊，当初找人揍他的时候幸亏想到用麻袋套住脑袋，不然让他把实力放出来，一人挡百应该不是太大问题。
<br/>　　铁枪的用法和花枪不同，在于挥砸砍，在于一力降十会。我有一千斤力气，你有一百斤力气，怎么打你也是死。
<br/>　　柳轻名知道大刀架不住铁枪，也就放弃了硬格的架势，只专注斩之一字。
<br/>　　这对身下马是极高的要求，马需通人性，知道何时冲锋，何时发力冲撞。看二人身下马，一黑一白，此时转圈厮杀，就像一副旋转的阴阳图。
<br/>　　观战的诸人都嗅到一丝杀气，一般这样的比试都是用木刀木枪代替。可不知道是朝廷疏忽还是其它，这次二人用的是真刀真枪。而二人已明显杀出火气，枪愈快，刀愈急，这样的交锋，但凡有一个疏忽，死不了也是重伤。
<br/>　　观战的诸人屏住呼吸，谁都明白二人是在刀尖上行走，生与死就在一瞬。
<br/>　　明帝也很高兴，这明显是证明自己手下还是有人才的嘛。正好戎族也在，你们也给我睁大眼睛看看，瞧瞧我帝国的少年英雄。
<br/>　　“戎族也有这样的儿郎么？”明帝笑着问。
<br/>　　“没有。”呼延力淡淡道：“这样的儿郎都被我们打败了。”
<br/>　　“呃……”
<br/>　　妈的，原来蛮夷也会噎人，堵的我不要不要的。
<br/>　　而这个时候，场上的交锋也接近尾声。在双方体力都露出颓势的时候，柳轻名一招蹬里藏身冒险接近屈九江，屈九江的刀施展不开，便被柳轻名得了机会。一招毒蛇出洞，将屈九江捅到马下。
<br/>　　“第二场马战，胜者屈九江。”
<br/>　　一胜一负，最后的武状元还是要在问策中选出，而至于问策谁胜谁负，那就是看明帝心情了。
<br/>　　屈九江和柳轻名被带下去检查伤势，换上新的衣服，二人刚才打得已经很狼狈，总不能以这幅模样去见明帝。
<br/>　　台上，文武官员对刚才的比试赞不绝口，屈九江纵然落败，但也可以说虽败犹荣。他和柳轻名的本事差不了多少，再来一场，胜的就未必是柳轻名。
<br/>　　“连我也不知晓，九江还有如此了得的刀法。”崔相抚须赞道：“配得上文武双全这四个字。”
<br/>　　“毕竟是轻名胜了，在他这个年纪有这样的身手，我看天下并没有几人。”尉迟离也夸奖柳轻名。
<br/>　　柳轻名是将军府的人，屈九江是相府的人，这在帝国几乎已经是公开的秘密。最后决出胜负的环节是问策，由明帝决定，二人自然趁着这个时间拉拉印象分。
<br/>　　下面的文官武将闻弦歌而知雅意，自然纷纷夸赞二人。
<br/>　　明帝不语，手下的斗争他明白也并不在意，如果他们不争了，岂不是要合起伙来对付自己了。
<br/>　　正这个时候，呼延力忽然呵呵笑了一声。
<br/>　　“我看帝国的武状元也不过如此么。”
<br/>　　明帝微闭的双目睁开，其他文官武将也看着开口的呼延力。
<br/>　　呼延力一拱手，道：“陛下，不如让我戎族的勇士和帝国的状元较量较量。”
<br/>　　明帝一顿，人家挑战了，我泱泱大国岂有不应战的道理。可同时明帝心里也清楚，呼延力既然敢说这话，必然有些底气，而在经过刚才一轮大战后，不管是屈九江还是柳轻名体力都有巨大消耗，此时应战其实吃了很大亏。
<br/>　　但什么叫泱泱大国呢，泱泱大国就是牙掉了往肚里咽，胳膊折了往袖子里藏，宁折里子不丢面子。哪怕是被人打断骨头呢，也得笑着说：不疼。
<br/>　　“呼延王子既然有兴致，不如就让他们比试比试，点到为止。”
<br/>　　“好。”呼延力笑了一声，道：“舒雷。”
<br/>　　一个戎族汉子没有走台阶，直接从高台上跃下去，只见他身高八尺开外，肩宽背阔，落在地上后宛若一头直立的大虫。
<br/>　　他健步如飞，飞身跃上一匹马，在场上狂奔，在马背上抽弓搭肩，在几个呼吸内便将箭篓内的羽箭射尽。
<br/>　　已经不用报靶，他射的不是稻草人，而是负责报靶的小校，连续十箭射掉十个小校的头盔。
<br/>　　明帝的脸色变得极其难看。
<br/>　　射人自然要比射草人困难，而十箭一样射中，单在箭法上已经甩屈九江和柳轻名十八条街。
<br/>　　戎族的箭法的确要比帝国强。
<br/>　　舒雷射箭过后，纵马在台下狂奔，口中发出阵阵啸声。
<br/>　　明帝，崔相，尉迟离，以及场上所有的文官武将，观战的士子此刻都觉得脸上火辣辣的疼。
<br/>　　呼延力笑得很开心：“舒雷在草原不过是一般人才。”
<br/>　　一般人才，一般人才就能超过我帝国的武状元，就是说我们的武状元在草原一抓一大把喽。
<br/>　　此时屈九江和柳轻名已换过衣服过来，刘阿吉传递明帝的话：“屈九江，柳轻名，这位戎族的壮士要与你二位较量，你二人不要谦让。”
<br/>　　不要谦让，意思就是给我狠狠揍他。
<br/>　　屈九江和柳轻名对视一眼，虽然二人暗地里斗得你是我活，可一旦面对戎族立刻同仇敌忾起来。同仇敌忾。
<br/>　　在场所有人此刻怕都是同一想法：给我狠狠揍他。
<br/>　　屈九江翻身上马，提着大刀冲了过去，即便现在他也没有要和柳轻名联手以多欺少的意思，泱泱大国么。
<br/>　　舒雷以铁弓为兵器，也照着屈九江冲锋过去。在两马相合的时候，他身形往马肚子里一藏，竟然在屈九江眼里消失不见了。
<br/>　　屈九江有些没回过神，两马相错而过，舒雷的身影突然出现，铁弓砸在屈九江背上。
<br/>　　屈九江被砸得口吐鲜血，身体从马上跌下来。
<br/>　　明帝的脸黑的像黑锅底，只是一个回合呐。
<br/>　　而程大雷对这一幕却不觉得陌生，因为他见识过戎族骑术。
<br/>　　戎族骑术，毕竟天下无双。
