　　九位冰宫长老，率先进入了雪林。
<br/>　　一众武者，立刻跟随。
<br/>　　雪林内，自然也有着妖兽。
<br/>　　但一行人浩浩荡荡，皆是强者，聚集在一起的气息，极其恐怖。
<br/>　　妖兽们，远远就能感知到，自然不敢造次。
<br/>　　一只只妖兽，尽皆躲藏了起来。
<br/>　　一行人，也没有理会这些普通妖兽。
<br/>　　大概半个时辰后，九位冰宫长老，停了下来。
<br/>　　前方，一个散发着强悍气息的巨大无形屏障，赫然出现在众人的感知当中。
<br/>　　冰宫大长老看向一众强者，道，“这屏障里头，是一不知名的洞府遗迹。”
<br/>　　极寒之地，极其辽阔。
<br/>　　没有任何武者敢说将足迹踏遍整个极寒之地。
<br/>　　更没有任何人知道这片地方的尽头在哪里。
<br/>　　但就是这样一个万里冰封，终年飘雪的不毛之地，却隐藏着许多遗迹。
<br/>　　这些遗迹，或自主出世，或被某个幸运的武者发现。
<br/>　　总之，每次出现，必引得所有人疯狂，掀起一片腥风血雨。
<br/>　　历届的冰缘大会，第一轮的冰海之缘，地点都是固定的，就在极寒冰海。
<br/>　　而第二轮的遗迹之缘，则并不固定。
<br/>　　毕竟每个遗迹出现的地点不同。
<br/>　　另外，被用作举行冰缘大会的洞府遗迹，在各方天骄到来前，任何人禁止开启。
<br/>　　连主办方都不行。
<br/>　　因为，这是属于各方天骄，以及参加冰缘大会武者的机缘。
<br/>　　冰宫长老高声道，“这个屏障，原封不动，没有任何人触碰过，也没有任何人入过这个遗迹。”
<br/>　　“诸位可检查一番。”
<br/>　　屏障，是洞府的一道‘门’。
<br/>　　若是尘封多年的洞府遗迹，其本身会带有一种古老气息。
<br/>　　屏障若被人动过手脚，过率先有人进入过，会留下痕迹。
<br/>　　这一点，骗不了人，也很容易就能检查出来。
<br/>　　而由于没有任何人进入过，自然的，这个洞府遗迹，便是不知名的。
<br/>　　不知道其‘主人’是谁。
<br/>　　更不知道里头蕴含着怎样的凶险。
<br/>　　甚至于，里头还可以互相争夺，互相厮杀。
<br/>　　为了重宝，而大的出手，并非什么罕见之事。
<br/>　　如果说第一轮的冰海之缘，是获得机缘。
<br/>　　那么第二轮的遗迹之缘，便是争夺机缘。
<br/>　　冰宫大长老高声道，“规矩，大家也清楚了。”
<br/>　　“而各位天骄，是否要进入其内，寻找、争夺自己的机缘，全凭自己意愿。”
<br/>　　“若是不愿，在外头等候即可。”
<br/>　　“一旦入内，发生什么意外，我不夜冰宫，概不负责。”
<br/>　　人群中，一阵喧嚣之声。
<br/>　　但最终，没有任何一个人愿意放弃。
<br/>　　洞府遗迹，虽存在危险，但里头的宝贝，同样不少。
<br/>　　谁肯放弃呢？
<br/>　　半晌，冰宫长老点了点头，道，“各位天骄，可自行进入屏障之内了。”
<br/>　　“陪同而来的各势力武者，在原地等候。”
<br/>　　冰宫长老话音刚落。
<br/>　　一千五百名武者，已纷纷飞跃而入，生怕去晚了，被别人寻了宝贝。
<br/>　　萧逸一行人，也立刻入内。
<br/>　　五人穿过无形屏障。
<br/>　　哗哗两声。
<br/>　　无形屏障上，泛起一阵如水纹般的涟漪。
<br/>　　一阵光影涌动。
<br/>　　五人前脚刚踏入，后脚已是另一幅光景。
<br/>　　眼前，一座座宫殿，鳞次栉比，极为雄伟。
<br/>　　前方，是一巨大的广场。
<br/>　　整个洞府遗迹，占地方圆百里，置身于一片冰天雪地中。
<br/>　　不错，这里大雪皑皑，白茫茫一片。
<br/>　　此时，已经有不少武者踏着厚厚的积雪，快速前行。
<br/>　　打算越过这广场，进入里头的洞府。
<br/>　　然而，这些武者刚刚走到广场中央，数道身影忽然从雪地中跃出。
<br/>　　身影的速度，极快，一闪而过。
<br/>　　当它们的身影落下时，积雪上，已被一片腥血染红。
<br/>　　两具尸体，轰然倒下。
<br/>　　“是暴风雪狮。”萧逸皱了皱眉头，说道。
<br/>　　暴风雪狮，六级妖兽。
<br/>　　天生拥有操控风雪之力，极为强悍。
<br/>　　且一身皮毛皆为白色，隐藏于风雪中，伺机而动。
<br/>　　要么不动，一动，必是敌人身死。
<br/>　　原本心怀期待与喜悦的一众武者，瞬间被浇了一盆冷水。
<br/>　　他们马上反应过来，这里可是机遇与危机并存的洞府遗迹，大意不得。
<br/>　　一众武者，纷纷开始联手，随后再次前进。
<br/>　　萧逸皱着眉头，稍微感知了一番。
<br/>　　在这雪地之下，竟然有着不下十万之数的暴风雪狮。
<br/>　　恐怕单单这一关，就足以让得许多武者望而却步。
<br/>　　萧逸摇了摇头，道，“我们走吧。”
<br/>　　萧逸在前领路，白冰雪等人跟上。
<br/>　　果然，五人刚刚进入广场不多久，便有暴风雪狮从雪地中袭击而来。
<br/>　　这种袭击，对五人来讲，自然成不了什么威胁。
<br/>　　萧逸随手一剑便能将其毙命。
<br/>　　但奈何，暴风雪狮的数量，实在太多。
<br/>　　五人，每走几步，就会有数头暴风雪狮跃出。
<br/>　　且，周遭雪地，还不时卷来暴风雪。
<br/>　　暴风雪中，又不时诡异地冒出几头暴风雪狮，让人防不胜防。
<br/>　　萧逸手中寒霜剑，凭空而现。
<br/>　　随手打出一记覆海斩。
<br/>　　如潮剑音涌出，周遭百米所有暴风雪狮，顷刻被击杀一空。
<br/>　　萧逸大手一挥，将内丹和精血吸摄而出。
<br/>　　随后，五人一个闪身，瞬间跨越了这百米范围。
<br/>　　一记记覆海斩打出。
<br/>　　萧逸等人，横扫前行。
<br/>　　不过半晌，便穿越了广场，来到了洞府之前。
<br/>　　而这时，萧逸已收获了数千枚妖兽内丹。
<br/>　　整个洞府遗迹，宫殿无数。
<br/>　　萧逸随手推开一座宫殿，一行人走了进去。
<br/>　　萧逸感知了一番，道，“宫殿之内，有不少禁制和房间。”
<br/>　　“内头各有凶险与宝物，大家小心些。”
<br/>　　萧逸刚想触发一道禁制。
<br/>　　孰料，白冰雪忽然说道，“萧逸，要不我们分开行动吧。”
<br/>　　“一起行动，速度太慢，且所得宝物，又要平分，这样收获太少了。”
<br/>　　“后面不少人已经开始越过广场了。”
<br/>　　萧逸看了眼后面，果然，有不少人已经快要越过广场了。
<br/>　　那片广场的十万暴风雪狮虽然厉害，但终究不是密集地堆在一起。
<br/>　　武者们联手而行，直线穿越，虽有难度，但不会太大。
<br/>　　微微思索了一番，萧逸点了点头，道，“好吧。”
<br/>　　“你们三人各自行动，叶铭跟我。”
<br/>　　对于白冰雪三人的实力，萧逸还是有一定信心的。
<br/>　　早在冰缘大会第一轮结束时，白冰雪、钟无忧、凌羽三人，就已经是地元九重修为。
<br/>　　叶铭则是地元四重。
<br/>　　而那时，众人还各自拥有大量吞灵冰鱼，以及15头吞灵冰鲸，一头吞灵冰鲸王。
<br/>　　加上在不夜冰宫领取的奖励。
<br/>　　众人回到客栈闭关，并将这些东西悉数吸收殆尽后，早已修为大增。
<br/>　　白冰雪三人，修为已是天元二重巅峰，只差些许就能踏入天元三重。
<br/>　　叶铭，则是地元九重巅峰，差些许就能踏入天元境。
<br/>　　吞灵冰鲸和吞灵冰鲸王，对普通武者的修为增加，是非常巨大的。
<br/>　　要不然历届的冰缘大会第一轮，吞灵冰鲸王也不会成为最后的奖励，且是最让人梦寐以求之物了。
<br/>　　这时，钟无忧说道，“叶铭跟着我吧。”
<br/>　　“我与他一道在外历练许久，彼此有一定的默契。”
<br/>　　“也行。”萧逸点了点头。
<br/>　　众人分开，临走前，叶铭慎重地看着萧逸，道，“萧逸师弟，一切小心。”
<br/>　　“好。”萧逸笑笑，点了点头。
