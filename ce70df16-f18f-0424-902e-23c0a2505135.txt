　　“诸位圣人可是担忧这剩下的巫、妖二族，不如交给本祖来处理好了，全部杀了，如何？”
<br/>　　就在诸圣对剩下的巫妖二族之事议论纷纷之时，一声洪亮的声音传了过来。
<br/>　　“玄天道友？”
<br/>　　女娲见此，脸色变得难看之极，不由得沉声道：“你当真要赶尽杀绝？”
<br/>　　来人正是玄天，他看了六圣一眼，心知自己不可能真的如他所说，将巫妖二族全部杀了，他当即一笑，道：
<br/>　　“娘娘何须紧张，本祖也只是随便说说而已。”
<br/>　　他顿了一顿，接着笑道：“此来，是为了取回我人族之物。”说着，他抬手一摄，只见战场废墟之上，一道乌光闪烁，向他射来，正是帝俊的屠巫剑。
<br/>　　“此乃我人族亿万子民的鲜血魂魄所炼，今日归还人族，也是应当。”
<br/>　　众圣自然是没有意见，一柄屠巫剑，只是克制巫族，算得上是一件后天上品灵宝了，他们还不至于为了这么一件灵宝而恶了人族。
<br/>　　看着下方的巫妖二族，元始淡漠出声道：
<br/>　　“巫妖二族发起战乱，造成洪荒生灵涂炭，如今虽然大战已经结束，但仍然给洪荒带来了很大的影响。所以，两族想要如之前那般，却是不可能了。后土身化六大轮回，有大功德，福泽巫族，巫族却是不该灭亡绝。至于妖族，也是如此。两族修士，可自行选择，跟随何人，至于剩下的，那便全部将他们发配到北方之地吧。”
<br/>　　说完，他便闭上眼睛，不再说话，任由他们决定。
<br/>　　既然元始都开口，其他圣人虽然心里有些异议，但也不敢说出来。元始乃是大大师兄，一身实力深不可测，道法无边，神通无量，再加上他手中掌握着众多逆天至宝，其他圣人就算不同意也只有挨打的份，是以他们口上都赞同了他的话。
<br/>　　至于北方之地，因为上次补天，斩杀巨大鳌鱼四肢，造成怨气冲天，阴气不散，在那里，如今已经算的上是不毛之地了，巫妖二族被送到那里，他们也乐见其成。
<br/>　　下面的巫妖二族听了此话，巫族在后羿、玄冥、刑天、重山等人的带领之下，毫不犹豫的便转身离开了。至于妖族，倒是瞅着几位圣人，心中想着要选哪位圣人。
<br/>　　元始身为诸圣之首，法力最为高强，众妖第一时间将目光投向元始，但他们失望了。只见元始闭上双眼，静立虚空，一副生人勿近的模样，没有一丁点的收徒欲望。
<br/>　　再看太上，也是一样，一副淡然无为，不理任何人的样子。至于剩下的圣人，首先选择的自然是女娲，她为妖族圣人，自然不会亏待妖族。还有就是通天，通天主张有教无类，想来也不会拒绝。至于西方教，却是没有人愿意去，他们都知道那里一向贫瘠不堪，无甚造化。
<br/>　　如此一来，跟了女娲的妖族最多，然后就是通天，当然，也有白泽，鬼车等几位妖神，不愿归顺任何人，便带着剩余的妖族修士，向着北方而去。
<br/>　　剩下的妖族都是高手，通天收服了上万妖族，心中自是非常高兴，照单全收。而西方的接引与准提两位圣人，摆弄了半天，却是什么都没有收到，不由得脸色发黑。
<br/>　　见两族修士分配完后，众人便要离开。就在此时，在场六位圣人与人族玄天，脑海之中传来一道淡漠的声音：
<br/>　　“速来紫霄宫，吾有大事要宣！”
<br/>　　诸圣顿时脸色大变，相互对视一眼，便朝着三十三天威的混沌飞去。
<br/>　　玄天转身看了看下方的众人族强者，便对他们说道：
<br/>　　“你们先回人族，道祖有事要宣，我先去紫霄宫一趟。”随即，他转身向着混沌飞去。
<br/>　　“众位道友，老爷已经等候多时了。”
<br/>　　众人刚刚来到紫霄宫，便有昊天、瑶池两位童子将几人引了进去。
<br/>　　元始一马当先，带领众人走了进去，来到最前方的蒲团之上，坐了下来，其他人也依次坐下。
<br/>　　片刻，道祖鸿钧便突兀的出现在高台之上，没有丁点空间波动，也没有任何其他法则波动，这这么突然的出现在大殿之上，他脸色淡漠如冰，看不出丝毫的情绪。
<br/>　　元始内心深处一震，瞳孔微微一缩，看向鸿钧，仿佛看到了他身上有九道彩芒在闪烁，他顿时便知道了，鸿钧已经有混元大罗金仙九重天的境界了，而且还是九重天巅峰，快要进入混元圆满，十重天了。
<br/>　　其他圣人并不知道这些，但是看到鸿钧那依旧深不可测的身影，还有那淡漠无情的声音，只觉得天道无情，深不可测。
<br/>　　“见过老师！”
<br/>　　诸圣上前行礼，就连玄天也上前行了一礼，开口道：“见过道祖！”
<br/>　　鸿钧道祖淡淡点了点头，淡漠出声道：“此番巫妖大战，乃是开天第三劫，了却了开天所留的一桩大因果，却是可以暂缓洪荒安定。”
<br/>　　说着，他看向人族武祖玄天，淡漠道：“巫妖人三族大战，巫妖败亡，洪荒不存，故而，今后人族当为主角，洪荒大地也自人族而兴。”
<br/>　　“多谢道祖！”
<br/>　　玄天拱手道谢道。其他圣人仿佛早知如此，倒也没有什么意外，只不过心中多少还是有些羡慕人族的。
<br/>　　鸿钧不停，继续说道：“此番巫妖两族败亡，天庭倒塌，却是需要重新立天庭，恢复天地秩序，进而教化洪荒万灵。”
<br/>　　“贫道已身合天道，从此只取天道，不问世事，如今洪荒天庭无人掌管，周天星辰无人镇压，需要重立天庭，代天司法天下，镇压洪荒气运。至于何人执掌天庭，还需要尔等自行商议？”
<br/>　　说罢，鸿钧道祖高坐云台之上，闭目不语，仿佛已经神游太虚去了。
<br/>　　众圣听说自由商议天庭之主，顿时开始盘算各自的利益，天庭替天行道，权利甚大，直接关乎到各自今后的利益，每个人都需要好好盘算一番，为今后多做打算。
<br/>　　元始是知道鸿钧道祖的打算，天道讲究平衡，圣人也一样，他不可能任由一家做大，所以这天庭之主，无论如何也轮不到自己头上。他也就不在这方面多想，而只是在想如何从其中捞取更多利益。
<br/>　　他闭上双眼，静静思索，却是不知道，他这幅模样，给了其他众人一种他不想管理天庭的意思。
<br/>　　太上圣人却是因为他为人教教主，门下只有一位传入，也不想在为这天庭之主而争吵不休，所以闭目不语。
<br/>　　那么，剩下的便只有妖族女娲、通天、准提、接引四位圣人之间的争论了。
<br/>　　……
