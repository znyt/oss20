<br/>        苏青阳和雪自悟也是无话可说，然后就听到陈明道：“派遣血衣剑卫保护我燕山的弟子，收编所有妖军，我现在，要去看着紫霞仙子和妖族大君，以免妖族大君牵扯到其他的鬼仙势力大君！”

<br/>        “算了，还是去把妖族大君给宰了，以免夜长梦多！”

<br/>    
<br/>        张鸣道：“此次跟来的记名弟子不少，但是有我们这么多灵石的可不多，我们就召集上万的大妖，首先第一件事，就是收缴所有的灵兵，然后召集部分大妖，将这些大妖迁移到沙海之内，如同我们燕山一般，建立炼丹房和炼器房，唯有如此，才能保证炼丹房和炼器房的秘密不外泄，而其他的大妖，我们每人组建一支千人的血衣剑卫，在沙海之外驻扎，等到这些血衣剑卫的铠甲杀王剑炼制完毕，我们就出兵攻打其他妖王！”

<br/>        这么安排了下去，张鸣也是开始收服这些大妖，本来张鸣以为还要废一番口舌，没有想到他刚刚报出自己的身份，这些大妖就已经纳头就拜了，“我乃燕山山主，陈明的弟子，此来就是为了给你们指一条明路.......”

<br/>        “我等愿意归降！”

<br/>        “什么，他是燕山山主陈明的弟子！”

<br/>        “燕山山主这么厉害，他的弟子必定也是人中龙凤，你们没有听说吗，燕山大宗主，就是燕山山主的弟子，已经是斩了两位大妖王了！”

<br/>        “大君的记名弟子，那不是比妖王还要尊贵？”

<br/>        雪自悟道：“我终于明白宗师的修行路为什么跑偏了，怕不是就是为了教这么多的弟子，所以他什么都修.......”
