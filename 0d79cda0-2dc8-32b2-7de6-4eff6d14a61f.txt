　　锵…
<br/>　　萧逸一剑劈出，星光长河碾压而过。
<br/>　　前方北隐宗大长老的武道虚影冷笑一声，一掌轰出。
<br/>　　武道虚影，乃是圣皇境后期以上武者才拥有的手段。
<br/>　　由武道力量凝聚而成。
<br/>　　其实力等同本体的全力一击。
<br/>　　此时，武道虚影上，那只苍老的手掌轰出，却宛如一记武道长河重重轰来。
<br/>　　萧逸劈出的一剑，星光长河相随，霎时与武道长河碰撞。
<br/>　　嘭…
<br/>　　一声爆响。
<br/>　　星光长河霎时在武道长河下溃散殆尽。
<br/>　　“不好。”萧逸脸色一变。
<br/>　　二者的差距太大了。
<br/>　　武道大能的实力，绝对是武道皇者巅峰的数倍以上，乃至更强。
<br/>　　自己全力一剑，却不敌武道大能的一道武道虚影。
<br/>　　“噗。”萧逸再次一口腥血喷出。
<br/>　　那条巨大的武道长河，已然环绕他周遭，封锁了所有去路。
<br/>　　萧逸置身其中，仅仅是感受到其中分毫武道气息，已心头一阵惊骇。
<br/>　　对比这条武道长河，自己的武道力量，简直如同一条涓涓细流，不，甚至更弱。
<br/>　　“给我破。”萧逸暴喝一声。
<br/>　　顾不得体内翻滚的气血，一剑劈出。
<br/>　　若被困在这条武道长河内，会更加麻烦。
<br/>　　本就体内伤势不轻，再身陷囹圄，后果可想而知。
<br/>　　北隐宗大长老甚至可以在远方操控这条武道长河，再次重伤他。
<br/>　　毕竟这条武道长河是他的武道力量。
<br/>　　轰…
<br/>　　一声轰鸣。
<br/>　　萧逸一剑出，十数剑齐至。
<br/>　　八万星光汇聚下的星幻长河在一剑之内，轰鸣了十数次。
<br/>　　整条武道长河，霎时溃散，化作点点武道微光，消散天地。
<br/>　　萧逸瞬间破封而出，身影再次疾驰。
<br/>　　嗖…嗖…嗖…
<br/>　　几乎是萧逸破封而出，继续疾驰的瞬间，身后北隐宗大长老的身影再次追来。
<br/>　　果然，如他所料，武道虚影拦他分毫，北隐宗大长老便能追上而来。
<br/>　　“哼。”萧逸冷哼一声。
<br/>　　嘭…身上一阵气势爆发，速度一瞬间飙升，再次拉开了距离。
<br/>　　沸腾妖火的焚烧下，萧逸体内的元力以恐怖的速度不断消耗着。
<br/>　　嗖…一个呼吸间，转瞬消失在北隐宗大长老眼中视线。
<br/>　　“该死，怎么可能速度飙升这么多？”北隐宗大长老脸色一冷。
<br/>　　“这真的只是个圣王境六重的小子？”
<br/>　　北隐宗大长老脸上闪过一丝不可置信之色。
<br/>　　他亲自出手追杀，即便是一个成名多年的圣皇境强者也休想逃脱。
<br/>　　可现在，却在追杀一个扬名不久的年轻人中，屡屡失败。
<br/>　　……
<br/>　　数千里以外，萧逸额头上的汗水，愈发涌现。
<br/>　　但同时，他也微微松了口气。
<br/>　　这时便看出庞大气泉与沸腾妖火的强悍之处了。
<br/>　　庞大元力下的疾速燃烧，让他速度飙升数以十倍增。
<br/>　　武道大能手段虽惊人，可若追不上他，又有何意义？
<br/>　　只要拉开的距离足够大，他未尝没有办法逃脱。
<br/>　　北隐宗大长老的武道烙印他抹除不了。
<br/>　　可这不代表这些武道烙印会一直存在。
<br/>　　待得逃遁的距离足够远，武道烙印渐渐消散，届时，即便这北隐宗大长老是武道大能也休想再追踪到他。
<br/>　　十数分钟后。
<br/>　　轰…
<br/>　　疾速飞行的萧逸，脸色忽然再次一变。
<br/>　　前方，一股武道力量忽然凝聚，一个武道虚影，陡然出现。
<br/>　　“怎么可能。”萧逸脸色一变。
<br/>　　武道皇者虽有凝聚武道虚影的手段，能万里之外，一念间取人性命。
<br/>　　可这般遥隔数千里凝聚武道虚影，极耗武道力量。
<br/>　　正常的武道巅峰强者，能在一天内凝聚一两次就很不错了。
<br/>　　可现在，距离上次武道虚影出现‘拦路’，仅仅过去了十数分钟。
<br/>　　“该死。”萧逸咬了咬牙，一剑劈出。
<br/>　　武道虚影上，北隐宗大长老狞笑一声。
<br/>　　同样是一掌轰出，武道长河碾压而来。
<br/>　　星光长河，再次在其中被碾压溃散。
<br/>　　武道长河，亦再次封锁包裹了萧逸。
<br/>　　当然，这仅仅是数秒之事。
<br/>　　萧逸一剑破开，再次急速飞离。
<br/>　　身后，北隐宗大长老的身影再次出现。
<br/>　　“小子，你逃不掉的。”北隐宗大长老怒喝一声，凌空一掌轰出。
<br/>　　剧烈的掌风，将沿途所过一切轰成了齑粉。
<br/>　　掌风未至，萧逸已感受到其中巨大的压力。
<br/>　　萧逸未有半分停顿，仍旧疾驰。
<br/>　　嘭…一声爆响。
<br/>　　掌风重重地轰在了他背后。
<br/>　　“噗。”萧逸一口腥血吐出，未作停留，反倒借着掌力的冲力加速飞离。
<br/>　　不过一个呼吸间，萧逸的身影，再次消失在北隐宗大长老眼中。
<br/>　　“该死。”北隐宗大长老脸色开始难看了起来。
<br/>　　“元力爆发，以达速度飙升的效果？”
<br/>　　“能在一瞬间内飙升如此多的速度，甚至超过老夫，元力的消耗可是个天文数字。”
<br/>　　北隐宗大长老边全力追击，边暗暗思索着。
<br/>　　半晌，北隐宗大长老冷笑一声，“我倒要看看你有多少元力逃。”
<br/>　　……
<br/>　　数千里之外。
<br/>　　萧逸擦了擦嘴角鲜血。
<br/>　　又是十数分钟后。
<br/>　　前方，武道虚影再次凝聚。
<br/>　　“又来？”萧逸脸色突变。
<br/>　　毫无疑问，他再次被拦下。
<br/>　　堪堪破掉武道长河封锁，身后北隐宗大长老再次追袭而来。
<br/>　　嗖…
<br/>　　萧逸瞬间疾速飞离。
<br/>　　身后，又是凌空一记狂猛掌风，重重轰在了他背上。
<br/>　　萧逸借力远遁。
<br/>　　然而，疾速飙升的速度，却忽然变缓。
<br/>　　“嗯？“萧逸脸色一变。
<br/>　　“不好，体内元力快要耗尽了。”
<br/>　　能让他飙升的速度超过了北隐宗大长老，其元力的燃烧与消耗自然是极快极大的。
<br/>　　哪怕是他那庞大惊人的气泉，亦在短短数十分钟内，元力几近见底。
<br/>　　“哦？元力要耗尽了吗？”北隐宗大长老见状，冷笑一声。
<br/>　　他自是注意到萧逸忽然变缓的速度。
<br/>　　“小子，这次你没机会了。”北隐宗大长老冷喝一声。
<br/>　　嗖…身影一闪，瞬间来到萧逸身旁。
<br/>　　轰…一掌重重轰出。
<br/>　　北隐元劲，本就诡异异常。
<br/>　　如今再加北隐宗大长老这般层次的无力全力施出，其威力可想而知。
<br/>　　嘭…高空中，一声剧烈爆响。
<br/>　　萧逸的身影，重重被轰下。
<br/>　　嘭…
<br/>　　待得萧逸的身影落于地面，巨大的冲击力下，竟将方圆十数粒内一切树木花草轰成粉碎。
<br/>　　地面上，一个深不见底的坑洞，骇人之极。
<br/>　　“死了吗？”北隐宗大长老挑了挑眉头，不屑一笑。
<br/>　　恰在此时。
<br/>　　坑洞之内，一道身影疾速跃出。
<br/>　　其速度，竟是比之前更快几分。
<br/>　　不过片刻间，身影再次消失在他眼前。
<br/>　　“怎么可能，还有元力逃遁？”北隐宗大长老双眼一眯。
