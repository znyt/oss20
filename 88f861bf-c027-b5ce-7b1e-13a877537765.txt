　　“哗啦……”
<br/>　　云团降临，火焰出现。万劫魔尊的身上，仿佛有着数百个黑洞，在不断的吞噬着这些火劫之气。滚滚云团，化为数百道流云冲进万劫魔尊的体内。
<br/>　　劫难从弱到强，火劫也是逐渐增强，万劫魔尊并没有浪费太多精力去抵抗这些火劫的温度，如果自己的身躯承受不住之时，再出手也不迟，不过，单单这火劫，万劫魔尊不认为能够逼迫自己出手。
<br/>　　火焰炙热，上升到真火、天火、神火的层次，真火灼烧，一种种真火接连而来，包括太阳真火、紫薇天火、玄天劫焰等许多顶级天火。
<br/>　　可万劫魔尊虚空而立，丝毫不在乎，身上的功法在不停的运转，不管来多少真火，他都能很快吞噬，增强法力。
<br/>　　随着云团的完全消散。
<br/>　　“轰！”
<br/>　　万劫魔尊整个人却化为了火人，一团混沌色火光，正在万劫魔尊体内灼烧。
<br/>　　“这是……混沌火？”有人惊讶道。
<br/>　　“不错，这的确是混沌火，无法抵挡，只能承受，要比真火、天火高上一个等级，甚至能够威胁混元大罗金仙了。”有大神通者认识这种火。
<br/>　　“相传混沌火从低到高共分为混沌玄火，混沌天火，混沌仙火，混沌神火，混沌圣火，混沌道火。一般我们讲道的只有混沌玄火与混沌天火，其他的都非常稀少，很难见到。不过，就算混沌玄火与混沌天火就很厉害了，准圣高手，承受不住它十息时间，就算混元修士，都能伤害。”
<br/>　　“是吗，看来万劫魔尊要倒霉了。”
<br/>　　“那可不一定……”
<br/>　　又是三年过去了。
<br/>　　火劫过去了，万劫魔尊很快就恢复了正常。
<br/>　　此刻，九天之上的雷劫早已经遮蔽整个宇宙了，狂暴的雷电在空中肆虐，电闪雷舞，十方寂灭，整个宇宙金仙以下者都变得瑟瑟发抖。
<br/>　　关注这里的大能们，一个个真正肃容了，之前的风劫，火劫对于万劫魔尊而言，都有些轻松了，这让他们不得不承认，万劫魔尊的厉害，超出想象。
<br/>　　如今，只剩下了雷劫与心魔劫，不过，有可能心魔劫就会与雷劫一同降落，或者穿插在雷劫之中，以之前的形势来看，雷劫定会不简单，这是形成道体最重要的淬炼。
<br/>　　“不知道这雷劫会是什么雷劫，酝酿了六年的时间，肯定不简单！”
<br/>　　“是啊，不知道这万劫魔尊能否度过此劫？”
<br/>　　天地间的大能们，一个个都紧张看着，万劫魔尊的渡劫，定然不会简单，他们一个个都很期待。
<br/>　　“来了。”
<br/>　　不知道是谁这么说了一句，万劫魔尊抬头望天，面色平静，眼中闪烁着疯狂之意，魔气在他身上纵横。
<br/>　　轰隆隆！
<br/>　　一道道紫色亮光划破天际，撕裂天空般的降落下来，如同主宰九天的雷帝降临，亲手镇杀万劫魔尊。
<br/>　　无穷无尽的雷光，上来就是九九八十一重紫霄大劫，那是大道之光，化成了紫气压落下来，至高至大，浩瀚而无可相匹。
<br/>　　众人可以看见，九九大劫，每一重都如同一层天，紫气蒙蒙，压落星河，虚空崩断，宇宙粉碎。
<br/>　　这才刚刚开始而已，这片虚空所有大星辰就都炸了开来，成为粉碎，一个都没有剩下，这个景象让人颤栗，毛骨悚然！
<br/>　　没有一种大劫会这般，刚一上来就毁灭了这片虚空，最起码寻常修士不曾见过，就连圣人成圣之时，都没有雷劫，所以，恐怕就连圣人都不曾见过如此大劫。
<br/>　　“给本尊碎！”
<br/>　　万劫魔尊一声大吼，手中劫魔刀魔威凛凛，身上魔气纵横，刀身急速放大，化为一柄遮天的巨大魔刀，向着九天之上的雷层斩去。
<br/>　　“哗啦！”
<br/>　　只听一声哗啦，那八十一道紫霄神雷竟然被劈成两半，中间裂开一道巨大的口子。
<br/>　　“万道森罗，混洞吞天！”
<br/>　　还没等人反应过来，却又听万劫魔尊一声大吼，陡然在他头顶升起一方巨大的混沌洞墟，洞口打开，缓缓旋转，其上混沌之气弥漫，魔影闪现，庞大的吞噬之力，化为一张巨洞，将被魔刀劈成两半的神雷吞了进去，这天上地下一片真空，什么都没有剩下。
<br/>　　“嘶！”
<br/>　　这个景象太过骇然了，关注此地的各路大神，纷纷倒吸一口气，这得有多大的法力，才能让这漫天的雷海都消失不见。
<br/>　　要知道，这蒙蒙紫气将这片虚空都粉碎了，星河皆散，成了碎片，最后永不复存在。
<br/>　　一道劈出，斩碎雷海，一洞既出，吞纳天罚，完全是圣人的手段，超出了世人的理解！
<br/>　　与此同时，洪荒各地都不能平静了，这么大的劫难，开始一出现便惊动了各地，尤其是各大圣人都选择了观看。
<br/>　　“这么大的天劫，潜力无穷，若是能够渡过此劫，便能成为混元，而且还是不弱的混元。”
<br/>　　“万劫魔尊？还真有可能成功！”
<br/>　　“劫难，也是机缘，度过了，便更进一步，度不过，则万事皆休！”
<br/>　　“……”
<br/>　　“轰！”
<br/>　　大劫更加狂暴了，玄光灭世，浩瀚无疆，这片破碎的虚空成为了毁灭的源头，什么都被湮灭了，什么都不复存在。
<br/>　　雷海之浩大超越了世人的想象，不可理解，将这个地方化为了厄土，只要是生命进来，必将被劈碎，尸骨无存。
<br/>　　万劫魔尊面色凝重，进行抗衡，这才刚开始而已，大劫就如此狂暴，足以将准圣大圆满高手劈成灰烬，他纵然是再强大，也要谨慎对待。
<br/>　　之前被混洞吞噬的雷劫，被他收到魔界之中，化为养料，增加魔界的能量，化为魔界的天罚之力。
<br/>　　宇宙哀鸣，雷霆海洋滔天，六合八荒都是它的光，没有什么星辰闪烁，每一缕炽电都在交织与喷发。
<br/>　　宇宙最本源的规则构筑成的痕迹，劈开了一切，这是开天，太初的气息汹涌，而相对的灭世神力也在浩荡。
<br/>　　嗡！
<br/>　　这种大劫不止，另外的神纹又降临了，天穹之上数十种雷罚齐至，滚滚而行，万劫魔尊张口大啸，手中魔刀挥舞，魔道浩瀚，身体之中，《万劫魔典》也在快速运转起来，吞吐天地劫气，吸收周边的雷劫。
<br/>　　这里成了洪荒之中最恐怖的地方，什么都看不见，炽光照耀古今，天地皆抖，冥冥之中的主宰像是到达了此处，威严无比，不容挑衅，要镇杀万劫魔尊。
<br/>　　“破！”
