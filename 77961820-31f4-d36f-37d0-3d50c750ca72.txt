　　“哼，昨天你见到的那些，只是刚死不久的行僵，身体还算保持完好的，要是碰见更高级的丧尸，看你怎么能吃下饭去！”
<br/>　　恨铁不成钢的数落了刘哔一句，九叔放下了筷子，倒也没将此事放在心上，毕竟是第一次见到僵尸，有这样的反应也算不上稀奇。
<br/>　　“对了，你和文才一会准备准备，和我上街一趟，我要和任老爷谈点事情。”站起身来，九叔交代了一句，转身向房间内走去。
<br/>　　一听到师傅要带他外出，文才脸上露出开心的笑容，因为这段时间要帮四目师叔看尸，他大部分的时间只能待在义庄里，都快要无聊死了。
<br/>　　“师父，任老爷请您去做什么，我需要准备一些什么东西吗？”连饭都顾不得吃完，文才连忙站起身来，亦步亦趋的跟了上去。
<br/>　　刘哔坐在原地，身体微抖，体内的反胃感依旧像潮水般的不断涌来。
<br/>　　真是要死了，看着桌子上的简陋饭食，刘哔满脸的生无可恋，这样可怕的感受，他再也不想体验了。
<br/>　　可是一直到现在，他都不知道自己的身体是怎么一个情况，这种厌食的症状，不知道会不会一直持续下去，倘若一直如此的话，那以后自己用什么手段来摄取生存所必须的养分？
<br/>　　“丧尸，一种由人类异变而成的灵长目特殊单位，喜食肉，不沾五谷，性凶，富有进攻性，多生獠牙，利爪，赤目，力大无穷且恢复能力极强……”
<br/>　　正在刘哔低着头，一个人苦思冥索的时候，先前无论如何呼唤都不出来的系统，却主动开口了。
<br/>　　而随着系统给出的解释，刘哔也逐渐明白了过来，也就是说，自此之后，他只能靠吃一些肉类才可以存活，当然，生吃还是熟食，甚至坏没坏掉都对他没有影响了。
<br/>　　因为丧尸病毒的特殊性，根本不会有别的病菌可以潜入进他的身体，再加上丧尸身体内已经停止运转的各项机能，更是让他有了近乎于永恒的生命。
<br/>　　也就是说，只要刘哔能够保持摄入的能量可以维持身体的最低消耗，那么他就可以保持自己的身体永远不会衰老，也不会有任何疾病，当然，也会百毒不侵。
<br/>　　并且，和这个世界的僵尸不同，虽然两者都是由人类尸体变异而来的，但本质上，却有很大的区别。
<br/>　　若是按照武力进行划分，同阶僵尸的能力要比丧尸高出一些，毕竟两者的身体素质，就根本不在一个档次上面。
<br/>　　昨夜大厅的那场混战，刘哔曾和其中一个僵尸互相交手了一番，对方的身体，就如同一块钢板一样，他用尽浑身解数都没对那头僵尸造成多大损伤。
<br/>　　反而是对方将自己逼的险象环生，如果这样一直交手下去，输掉的一定会是刘哔，要知道这还只是一个最为低等的行僵而已。
<br/>　　但同样的，僵尸的限制非常之大，不说道术，就连一些糯米，桃木之类的普通物品，都会对其产生影响，道行低微的，甚至都不能在阳光之下自由行走。
<br/>　　丧尸就与其不同了，几乎没有什么东西能对它们造成克制，再加上那变态的恢复能力，想要将其击败也只能依靠蛮力进行绞杀。
<br/>　　如果非要说出几个丧尸的弱点，也就只能从智力方面下手了，但对于拥有完整意识的刘哔来说，这对于丧尸来说最大的限制都可以忽略不计了。
<br/>　　“嘿，师弟，想什么呢，笑的这么开心。”因为一直都在专注的倾听系统讲解，刘哔坐在饭桌前面，没有在第一时间察觉到身后悄悄潜行过来的文才。
<br/>　　心中悚然一惊，刘哔连忙闭上自己的嘴巴，因为害怕自己的獠牙暴露出来，他这段时间都是笑不露齿的。
<br/>　　天知道他一个钢铁直男，忍得有多么辛苦。
<br/>　　“赶紧收拾收拾准备出发了，师傅要带我们去喝外国茶。”文才笑呵呵的将手搭在刘哔肩膀上，似是想到了什么，补充了一句：“师弟，你的虎牙真可爱。”
<br/>　　看似随意的一句话语，却让刘哔微微眯起了眼睛，浅红色的眸子深处，透出一股极度危险的色彩，他那两颗獠牙，文才全都看到了。
<br/>　　或许是因为经过系统净化过的原因，刘哔身上的丧尸特征并不是十分明显，如果不仔细查看的话，他的獠牙确实和虎牙有几分相似，只不过要长上一些而已。
<br/>　　但文才的话中有几分真假，他也不敢妄下结论，在这一刻，他是有那么一点想要杀掉文才，以绝后患的。
<br/>　　毕竟，一只拥有人类思维和外表的活‘僵尸’，这个消息要是流传出去，将会给他带来无尽的麻烦。
<br/>　　但是当他真要准备下手的时候，却又有些犹豫不决了起来，这和杀不杀人没有什么关系，如果换成其他的一个人，刘哔早就已经动手了。
<br/>　　在末世生存了十年之久，他的双手也没有那么干净，以刘哔现在特殊的身体，想要暗中杀掉文才也是十分简单的事情。
<br/>　　麻烦的是，杀掉文才之后，所产生的蝴蝶效应。
<br/>　　虽然之前有看过原著，但终归是隔着一个屏幕，这个世界的很多东西，刘哔现在还不太了解，对他而言最大的优势，也无疑是熟知部分剧情。
<br/>　　可文才这个人物，是原著中十分重要的一个角色，如果冒然将他击杀，难免会对剧情发展造成影响，而他最大的优势，也就荡然无存了。
<br/>　　“恩，我去换件衣服，师兄等我一会，我马上就会出来。”犹豫了几下，刘哔还是放弃了心中的想法，手掌摊开，轻轻在文才后背上拍了拍。
<br/>　　他觉得对方应该不是在演戏，至少现在还没有发现他身上的异常，见过几个原著中的重要角色，刘哔发现，这个世界人物的性格，和原著差不了多少。
<br/>　　而依照文才胆小懦弱的性格，如果真的发现，甚至只是怀疑他变成了僵尸，那也不会主动和他靠的这么近，更不会将手臂搭在自己的肩膀上。
