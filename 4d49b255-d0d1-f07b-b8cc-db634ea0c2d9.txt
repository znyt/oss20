　　第497章
<br/>　　次日黎明，李莫愁平安折返。说平安折返，但也表示她一无所获。和程大雷所料得不差，她没能追上那杀人于无形的聂隐娘，如果真的追上了，究竟还能不能活着回来还不一定。
<br/>　　按照程大雷的吩咐，她特意来见程大雷，但其实也没见着面。程大雷被那聂隐娘吓破了胆，窝在地牢里不出来，李莫愁也是站在地牢外，隔着气窗与程大雷说话。所以，程大雷能看见她，但她看不见程大雷。
<br/>　　“程当家何不出来一见，或许让我进去。”
<br/>　　耳边响起一娇柔的声音，程大雷忍不住心中一荡，对这李莫愁的模样有几分好奇。李莫愁话里话外，有淡淡嘲讽的意思。
<br/>　　对于这样的高手，程大雷可不敢得罪。她的等级已在绝世，顶级人物都不一定忠诚自己，何况是绝世。那句话着她不开心了，她拂袖离去都是好的，最差的结果是和聂隐娘一起要杀自己。
<br/>　　“哈，实在是招惹了了不得的对头，倒是惹赤练仙子笑话了。”程大雷在地牢内踮起脚尖偷看李莫愁，也没忘把首领光环开启，先好好刷一波好感。
<br/>　　“大丈夫能屈能伸，不去在乎世间庸庸碌碌之辈人的看法，方是真英雄。”
<br/>　　“哈，果不愧是赤练仙子之名，所见自然不会和常人相同。”
<br/>　　程大雷隔着气窗偷看李莫愁，先看到一双玉足，随之视线慢慢向上，见到一亭亭玉立，模样娇人的美貌道姑。
<br/>　　程大雷吃了一惊，他倒是没觉得李莫愁是什么丑八怪，但对方的美貌还是出乎她的预料。毕竟有赤练仙子之名，心如蛇蝎，却是貌若仙子。
<br/>　　隔着气窗与她对视，程大雷偷看被抓了个现行，李莫愁眸底一寒，目光盯着程大雷。
<br/>　　程大雷倒是没有被抓个正着的窘迫，视线更坦荡起来，道：“常听人说起仙子是个魔头，却是没想到，原来魔头是个仙子。”
<br/>　　“程当家放尊重些。”李莫愁声音越寒。
<br/>　　程大雷似乎没有察觉李莫愁的怒意，哈哈笑了一声，道：“仙子且先在蛤蟆寨住下，等我解决掉眼前的麻烦，再去向仙子请罪。”
<br/>　　李莫愁冷哼一声，脚步声渐渐远去。
<br/>　　程大雷躺倒在稻草堆上，并不在意自己的话触怒李莫愁。
<br/>　　这赤练仙子的本事是很高的，但性情其实很简单。年纪轻轻不通时事时，被一个人渣忽悠了，然后性情越来越偏激。行走江湖杀得人越来越多，多了一个赤练仙子的恶名。可纵然她杀人如麻，江湖人也要在赤练后加上仙子二字，可见她的美貌到什么程度。大家都惧她的狠辣，自然也不会有什么追求者，不然也不会被姓公孙的人渣轻而易举忽悠。
<br/>　　说白了，无非是个缺爱的可怜人而已。
<br/>　　当然，现在李莫愁的身份背景肯定是不同的，不过程大雷估计性情不会有太大改变。等自己离开这地牢，想要收服她成为自己的手下应该不难。
<br/>　　不过，如何能解决掉聂隐娘带来的麻烦，依旧是程大雷一桩心病。
<br/>　　……
<br/>　　聂隐娘一路逃出了蛤蟆城，要做的第一件事就是找个地方洁身。纵然她从小就被当做刺客培养，不通人情世故，但女儿家总是有些爱洁之心。
<br/>　　她从城外一户人家中偷了一套衣物，临走前抛下几两散碎银子，然后找到一处偏僻的水源，迫不及待的将身上的衣物除去，身体浸入了水中。
<br/>　　平生杀过很多人，也并不是每次都能轻而易举的完成任务，有些情形也十分危险。但无论如何，这次都是最狼狈的时候，万没想到，程大雷竟会用这样龌龊的手段攻击自己，让自己落到难堪的境地。
<br/>　　她整个人浸入水中，手扣在肩膀上，有殷红的血散落在水中，与她洁白的肌肤相映成趣。
<br/>　　她的右肩被刺伤了，虽然伤势并不重，但依旧是难堪的经历。她闭目回忆着与程大雷相遇的经历，心中也不得不承认，这委实是个难缠的目标。
<br/>　　谨慎、多虑，为了活命不择手段。
<br/>　　师父说得果然不错，容这样一个人活在世上，不知还有多少善良无辜的人遭难。
<br/>　　杀一个恶人，有时候就能救无数人。此刻，聂隐娘更坚定了要杀程大雷的念头，不是为了雪耻，而是为了苍生。
<br/>　　在水中洗净了身子，她换上偷来的粗布麻衣，将原先的衣物就地埋了，又小心翼翼的伪装，如此避免被人发觉。
<br/>　　她再次返回蛤蟆城，寻找二次刺杀的机会。昨夜那个道姑也十分了得，以聂隐娘对江湖事的了解，对方当是赤练仙子李魔头。
<br/>　　这魔头也是难缠的对手，怎么会跑到蛤蟆寨，果然是蛇鼠一窝，恶人总是聚在一起。
<br/>　　其实程大雷之前还忌惮李莫愁是个杀人为乐的魔头，其实他完全是多虑了，因为在江湖上的名声，他一点也不比李莫愁好，都在聂隐娘的可杀名单上。
<br/>　　在聂隐娘心中，李莫愁手段残忍，又善于用毒。用毒……一听就卑鄙了，想要杀人，凭掌中利刃即可，为何要用毒，定然是功夫没学到家的缘故。
<br/>　　她重新返回蛤蟆城附近，寻找入城的机会。蛤蟆城加强了警戒，以前还只是盘查，现在是寸铁不许带入城。
<br/>　　当然，聂隐娘也没想着从城门进去，她只是在等待晚上。不过令她没有想到的是，晚上蛤蟆城城墙的防守更加严密，墙上站满了人，一时缝隙都没有。
<br/>　　有一个大胡子还不停训斥士兵，语气十分愤怒。
<br/>　　以聂隐娘的本事，想要突破入城其实并不是做不到。但这样一来，定然会暴露行踪，她的目的是杀掉程大雷，而不是这些普通的士兵。他们也不过是受制于程大雷的无辜之人而已，等自己杀了程大雷，他们也就逃离了魔窟。
<br/>　　这一夜，聂隐娘没寻到潜入城池的机会，她在城外找了一山洞安身。每天夜里都在城外收着，等蛤蟆城的境界松弛下来。
<br/>　　她曾有为杀一人，等待一年的经历，这一次，不过是刚刚开始。
<br/>(本章完)
