　　叶尘自然也察觉到了外面的动静，不由得嘴角微微一扬，自言自语到：
<br/>　　“没想到这烈日宗，反应挺快，倒也省的我去找他们了！”
<br/>　　说着，叶尘便要朝大殿外走去，不料却被云灵儿一把拦住，
<br/>　　“叶前辈！烈日宗来者不善，您还是赶快逃吧！”
<br/>　　叶尘微微一笑，
<br/>　　“区区一个小小的烈日宗，还不至于让我退步！”
<br/>　　“可是……”
<br/>　　云灵儿还想再劝，叶尘已经缓步朝大殿外走了出去。
<br/>　　而此时，大殿之外，
<br/>　　“哎呀！南宗主，是什么风把您和诸位长老吹来了？莫某有失远迎，还请赎罪！”
<br/>　　莫长林踏空而行，来到天空之上，冲着杀气冲冲的烈日宗众人，陪着笑脸道。
<br/>　　“莫长林！明日面前不说暗话，你们流光派弟子，杀了我的儿子，赶快把凶手交出来，否则我今日便踏平这流光宫！”
<br/>　　说话的是一名一身金袍的中年男子，赫然正是烈日宗宗主南日山。
<br/>　　“少宗主死了？此事和我们流光派有何关系？南宗主怕是有什么误会吧？”
<br/>　　莫长林继续矢口否认。
<br/>　　“哼！”
<br/>　　南日山顿时冷哼一声，
<br/>　　“误会？本宗主一路循着因果蛊的气息，一直追到你们流光城，你还敢说不是你流光派门下弟子所为？”
<br/>　　莫长林脸色微微一变，旋即又继续笑道：
<br/>　　“南宗主说笑了！想我流光城有十万人口，可并非全都是我流光派弟子，如果杀人凶手躲入流光城，我们又岂会察觉？”
<br/>　　就在两人争辩之时，叶尘此时从大殿中走了出来。
<br/>　　叶尘一出现，南日山身后，一名白发老者，两眼猛然一蹬，指着叶尘，大喝道：
<br/>　　“是他！因果蛊的气息就在此人身上，一定就是他杀了少宗主！”
<br/>　　哗！
<br/>　　众人的目光，顿时齐刷刷的转移到叶尘的身上。
<br/>　　南日山两眼微微一眯，在叶尘身上打量了片刻，眼眸中泛着浓浓的杀意，
<br/>　　“莫长林，这杀手凶手，就在你们流光宫之内，你现在还有什么话可说？”
<br/>　　“这……”
<br/>　　莫长林显然也没有想到，叶尘居然会自己跑出来，顿时有些不易辩解。
<br/>　　这时，
<br/>　　那楚星河忽然朗声开口道：
<br/>　　“启禀南宗主，此人的确是杀害少宗主的凶手，不过却并非我流光派弟子，我们故意将他引入城中，就是为了等南宗主前来报仇啊！”
<br/>　　楚星河这话一出口，无论是流光派还是烈日宗，众人全都一阵愕然，继而便明白了楚星河的意图。
<br/>　　他显然是看形势不对，准备舍弃叶尘，来平息烈日宗的怒火了。
<br/>　　不得不说，这一招，够狠！也够毒！
<br/>　　云灵儿听到这话，顿时气得两眼通红，
<br/>　　“楚星河！你知道你在说什么吗？叶前辈可是我们的救……”
<br/>　　还没等云灵儿说完，却又被楚星河一声暴喝，直接生生打断，
<br/>　　“灵儿师妹！你可要想清楚了，此人是杀害烈日宗少宗主的凶手，今天南宗主和烈日宗诸位长老在此，他肯定必死无疑，你可不要为了一个死人，连累了自己，还连累了整个宗门！”
<br/>　　“我……”
<br/>　　云灵儿还要辩解，莫长林又突然开口了，
<br/>　　“灵儿，此事不是你能决定的，莫要添乱！”
<br/>　　莫长林这么说，很显然已经默认了楚星河，过河拆桥的行为。
<br/>　　没办法，烈日宗的实力太过强大，在他们看来，叶尘今日肯定必死无疑，与其为了一个必死之人，去冒险得罪烈日宗，还不如尽快和他撇清关系。
<br/>　　“南宗主，您也看到了，此事并非我流光派弟子所为，还请您明鉴！”
<br/>　　流光派其他长老，顿时也纷纷开口辩解起来。
<br/>　　“你们……你们怎么可以……”
<br/>　　云灵儿气得不行，那本就十分饱满的胸脯，顿时一起一伏。
<br/>　　叶尘却一脸平淡，表情无悲无喜，不过眼眸深处，却透着一抹淡淡的失望。
<br/>　　这种事情，在弱肉强食的修真界，倒也十分常见，他并不怎么意外，只是他对流光派仅存的那一丝留恋，因为这件事，如今也荡然无存了。
<br/>　　原本，就算今日烈日宗不来，他将来也为流光派除掉这个隐患，甚至在他的提携之下，流光派定然能够发扬光大。
<br/>　　可惜，是流光派自己没有把握住这个机会，反倒觉得他是一个累赘，要将他一脚踢开。
<br/>　　若非这流光派中，还有一个云灵儿，以及那位曾经对他有恩的玄离真人，他此刻说不定，早就朝楚星河等人，一巴掌扇过去了！
<br/>　　“小子！你来说，我儿子是不是你杀的？你当真和流光派没有任何关系吗？”
<br/>　　南日山显然并不打算轻易放过流光派，转而望向叶尘，冷冷询问道。
<br/>　　叶尘却并没有鸟他，转而冲着一旁，满脸焦急之色的云灵儿，微微一笑，
<br/>　　“丫头，不必担心，就凭区区一个烈日宗，还伤不了我！”
<br/>　　说完这话，叶尘轻轻跃至空中，踏空而行，来到南日山等人的跟前，淡淡道：
<br/>　　“你的那个废物儿子，的确是我杀的，你们如果想报仇的话，那就一起上吧，我赶时间！”
<br/>　　轰！
<br/>　　叶尘这话一出口，烈日宗和流光派的众人，顿时全都听懵了，甚至都怀疑自己的耳朵出了问题。
<br/>　　“这家伙，难道疯了不成？”
<br/>　　“他难道不知道，自己所面对，可是天狼星域的霸主烈日宗啊！”
<br/>　　“虽说这次烈日宗并没有倾巢而出，却也来了足足两位元婴强者，和数十名金丹高手啊！”
<br/>　　“哼！我看他是不知死活！”
<br/>　　……
<br/>　　众人议论纷纷。
<br/>　　而南日山听到叶尘这番狂妄之语，顿时气得怒极反笑，
<br/>　　“好好好！小子！本宗主倒要看看，就凭你区区一个金丹境，拿什么和我烈日宗叫板！”
<br/>　　南日山说着，便要直接动手。
<br/>　　而这时，一名中年男子，率先从南日山的身后跳出，
<br/>　　“区区一个金丹境的毛头小子而已，何必劳烦宗主大人出手，看我灭了他！”
<br/>　　那中年男子大喝一声，掌中飞出一把黑色巨刀，朝叶尘冲杀了过来。
<br/>　　“蝼蚁一般！”
<br/>　　叶尘连眼皮子都没有抬一下，随手屈指一弹，
<br/>　　彭！
<br/>　　那中年男子的脑袋，瞬间便被一道剑气打爆！
<br/>　　“什么！”
<br/>　　众人顿时再次哗然。
