　　鬼姨一听，心头一喜，附近那些宾客也立刻跟着起哄。
<br/>　　“不要耽误了吉时，我们要先看拜堂，别的事先放在一边！”
<br/>　　很多人一起起哄，鬼姨面露难色，对叶少阳说道：“法术公会的弟子，不会赖账，本该这就让他磕头认错，但的确不能耽误了吉时，先让他们拜堂，再让晓宇给你认错，你看可好？”
<br/>　　说完又怕叶少阳反驳，转过脸，对陈晓宇呵斥道：“还不下去候着！”
<br/>　　陈晓宇跌跌撞撞地朝自己的座位走去。
<br/>　　“你还有脸坐在这，看看你狼狈的样子，你赶紧去找个地方洗刷一番，换个衣服再来，待会周周整整地给叶道长认错！”
<br/>　　“是！”陈晓宇低着头一路跑出了山门。
<br/>　　叶少阳心中明白，这都是鬼姨的计策，陈晓宇这一去肯定就不回来了，到时候就算自己逼问，鬼姨派人去找，但是找不到陈晓宇，那也没办法，等宾客都散去了，将来这件事也就不了了之。
<br/>　　不过，对叶少阳来说，当众羞辱了陈晓宇一番，自己也消气了，懒得再叫真，毕竟陈晓宇对他来说，只是一个小人物，还不值得他去认真对待，跑了也就跑了。
<br/>　　鬼姨走到张晓寒身边，呵斥他赶紧去拜堂。张晓寒狠狠瞪了叶少阳一眼，道：“你这会儿不走吧，最好不要走，等我拜堂之后，我倒见识一下你的手段。”
<br/>　　说完，在司仪的引领下，朝花轿走了过去。
<br/>　　奏乐声再度响起来。有人去点了鞭炮，现场再度恢复到了喜庆的气氛。
<br/>　　宾客们开始起哄，开着玩笑，故意把之前斗法的这件事抛在脑后。
<br/>　　梧桐已经放下了红盖头，局促不安地坐在轿子里。
<br/>　　张晓寒走过去，掀开帘子，两个算是送亲的小姑娘，凑到花轿前，在找张晓寒要红包。
<br/>　　张晓寒给了红包之后，立刻收到她们“早生贵子”之类的祝福。
<br/>　　张晓寒有些得意地看了叶少阳一眼，弯腰下去，想要去把梧桐抱起来。
<br/>　　叶少阳径直走了过去。
<br/>　　张晓寒对他这么做也有点纳闷，皱眉看着他，说道：“什么事？”
<br/>　　“你不是要跟我斗法吗？”
<br/>　　张晓寒惊了一下，失声笑道：“你居然这么急着送死，等着我拜堂之后，我满足你。”
<br/>　　叶少阳双手背在后面，说道：“何必要等到拜堂之后，不如我们来个约定，你我打一场，谁赢了，梧桐就跟谁走，你看怎么样？”
<br/>　　张晓寒石化。一时间，他还以为自己是听错了。
<br/>　　“叶少阳，你说什么！”
<br/>　　张晓寒转头看着那些吹唢呐的，大喝一声：“不要吹了！”
<br/>　　奏乐声戛然而止。
<br/>　　在场宾客们也都呆了，傻傻地看着站在喜棚前面的叶少阳和张晓寒。
<br/>　　张晓寒阴沉着脸，盯着叶少阳，说道：“叶少阳，你刚说什么！”
<br/>　　“我说，跟我打一场，谁赢了，谁带梧桐走。”叶少阳故意把双手拢在嘴边，冲他大声喊道。
<br/>　　举座皆惊。
<br/>　　“叶少阳，你放肆！”
<br/>　　“想抢别人老婆啊。简直混蛋！也不看看自己几斤几两！”
<br/>　　一石激起千层浪，那些依附着法术公会的法师，一起对叶少阳口诛笔伐起来。
<br/>　　四宝和吴嘉伟这时候已经吃饱了，正靠在椅子上喝茶，听见叶少阳这句话，一口茶差点没喷出来。
<br/>　　“卧槽，野山羊这是玩什么，怎么还抢婚上了！”四宝喃喃说道。
<br/>　　妙心叹了口气，道：“他可是够傻的，以为赢了陈晓宇，就可以挑战张晓寒了，这虽然差了一个牌位，但根本就是天壤之别。”
<br/>　　张晓寒突然大笑起来，弯下身子，对着里面的梧桐说道：“梧桐，你听到了吗，叶少阳要把你抢走呢……”说到这，陡然变色，望着叶少阳，一字一顿地说道：“你他么的当我张晓寒是什么人，在我大婚的喜宴上，要来抢我的媳妇？叶少阳，你活该要去死了！”
<br/>　　叶少阳摆了摆手，“现在，她还不是你媳妇。”
<br/>　　“其实我也不想这么做，但是……如果不打一场的话，你会让我带走她吗？”
<br/>　　张晓寒双手握成拳头，关节捏得咯吱响，但是过了一会，又松开了拳头，脸上再次现出不屑的微笑，道：“叶少阳，我很佩服你的勇气，但你这么做，是要付出代价的……”
<br/>　　张晓寒话刚落音，没料到的是，身后响起了一个声音：“我跟他走。”
<br/>　　梧桐！
<br/>　　张晓寒错愕地回头看去。
<br/>　　梧桐，自己从花轿里走下来，然后，伸手揭开了自己头上的红盖头。
<br/>　　“冷玉嫂子！！”吴嘉伟看见梧桐，一下跳了起来。
<br/>　　“冷玉啊，卧槽，你怎么成别人媳妇了……”四宝更是激动，直接跳上了桌子，指着张晓寒大骂，“你个王八蛋，居然敢跟少阳抢冷玉，兄弟们给我灭了他！”
<br/>　　“别激动，你妹的，认错人了你们！冷静！”
<br/>　　叶少阳就知道他们一定会把梧桐当成芮冷玉，冲着他们连连摆手。
<br/>　　“站住！！”
<br/>　　听见有人把张晓寒骂成是王八蛋，法术公会的弟子们坐不出了，一个个要冲上去弄他，被张晓寒大声呵斥住。
<br/>　　张晓寒一张脸阴沉的似乎能滴水，道：“今天毕竟是我大喜的日子，你们不要闹事，冤有头债有主，今天这件事，是你叶少阳引起的！”
<br/>　　叶少阳耸了耸肩道：“我也不想这样，是你没有搞清楚，人家姑娘其实根本就不喜欢你，你不能强迫人家嫁给你，你说是不是？”
<br/>　　张晓寒想起梧桐之前说的那句话，脸庞慢慢转向她，说道：“你刚才说什么？”
<br/>　　他的脸色，让梧桐不由自主地往后退了几步，不敢出声。
<br/>　　“你，不喜欢我？”
<br/>　　梧桐咬了咬牙，说道：“晓寒，你本来就知道的。”
<br/>　　“我不知道！”张晓寒怒道，“我有什么地方配不上你？”
<br/>　　梧桐道：“没有，你很优秀，是天选之子，法术界未来的第一强者，人也好看，但是……”
<br/>　　“那你有什么理由不喜欢我？”
<br/>　　叶少阳噗嗤一笑，道：“我一直觉得我挺自恋的，但是要比自恋啊，我尸体在水里漂了三年都不浮，就服你！”
<br/>(本章完)
