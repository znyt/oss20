　　广宗天师说道：“当然关你的事，狐王虽然死了，但也不是不可能复活！”
<br/>　　叶少阳浑身一颤，失声道：“不许骗我！”
<br/>　　“这么大的事情，我敢骗你？”
<br/>　　“可是她魂魄都分解了，这……”
<br/>　　“魂魄分解，也是被东皇钟吸收，并没有散去，完全可以聚拢元神。”广宗天师说完，冲下方的小青小白喊道：“你们两个，快提着狐王的尸体！去蒙山上，找冰晶镇压，可保尸身不腐！”
<br/>　　两人怔了一下，道：“图中世界？”
<br/>　　他们之前跟随广宗天师，在山河社稷图中住过好一阵子，一听到蒙山两个字，立刻知道说的是什么地方。
<br/>　　广宗天师对叶少阳道：“打开山河社稷图！”
<br/>　　叶少阳本能地往背包里一摸，空的，惊了一下，这才想起山河社稷图之前给了道风。于是急忙呼唤道风。
<br/>　　道风也在东皇钟下面，帮忙抵挡尸兵的进犯，听见了两人方才的对话，急忙对着叶少阳抖开了山河社稷图，叶少阳抓住一头。
<br/>　　“先进去一部分，留下几个先挡着这些僵尸！”
<br/>　　在广宗天师的吩咐下，小青小白带着小九的尸体钻进了山河社稷图。
<br/>　　“拦住他们！”女魃看见这一幕，虽然不知道是在干什么，但也担心他们鼓捣出什么幺蛾子来，结果尸族大军虽然人多，但一时半会也无法穿过防线。
<br/>　　女魃情急之中，再次祭出三尸之气，这次却不是面对叶少阳等人，而是直接对着自己手下一干尸兵施展出去。
<br/>　　三尸之气所经之处，将一路上的僵尸尽数绞杀，奔着东皇钟而来。
<br/>　　这一幕，让叶少阳等人也是看得目瞪口呆，她，为什么要对自己的手下下手？
<br/>　　女魃很快就给了他们答案：所有被三尸之气杀死的僵尸，在身体溶解之后，体内都流出了暗红色的液体，主要是血，加上尸水和尸油。随着死亡的僵尸越来也多，真正是血流成河。
<br/>　　血水在一股神秘力量的操控之下，仿佛一头洪水猛兽，以“之”字的形状朝着东皇钟卷了过来。
<br/>　　有几个空界的弟子还没完全反应过来，就首当其冲地被血浪击中，等血浪褪去，只剩下一具具的白骨尸首。
<br/>　　“这尸水有腐蚀性，快逃！”广宗天师大声喊道，“你们躲远一点，我有办法！”
<br/>　　阴间四神不理会，联手布置出了一道灵力结界，挡住了血浪。血浪却没有褪去，而是仿佛遇到了堤坝一般，汹涌地涨了起来。
<br/>　　四大神顿时感到了压力。
<br/>　　广宗天师大声喊道：“不行的，这是三尸化血术，是尸王的杀手锏，不光是尸血，这血水之中，还蕴含了那些被献祭的僵尸的修为，水涨船高，这里有数不清的僵尸，你们怎么抵挡！”
<br/>　　四人面面相觑，每杀一个僵尸，就多吸纳一份灵力……这里的僵尸本来就是精英，就这么吸纳下去，谁能抵挡？
<br/>　　转念一想也就明白了：如果不是为了完成这强大的法术，女魃也不可能甘愿牺牲这些精英。
<br/>　　“快点退出去，我让少阳作法！”
<br/>　　“作什么法，靠他一个人？”马面道出了几人心中的怀疑。
<br/>　　广宗天师冷冷道：“不是靠他，是靠东皇钟！”
<br/>　　转头寻找道风，发现他用五朝元气护身，踏浪而行，去击杀女魃了。
<br/>　　李浩然紧随其后，目标不知道是道风还是女魃，对他来说，这两个都是仇人。
<br/>　　“这几个笨蛋，罢了，反正他们也死不了！诸位阴神，你们火速退开，到钟上来！”
<br/>　　阴间四神转头看着东皇钟，犹豫了一下，最后还是飞身上去，黑无常甩了甩袖子，将自己手下一干鬼差全收在了袖子里。
<br/>　　捉鬼联盟的其余成员，也都上来了。至于那些空界的强者，不敢独自与女魃这强大的法术为敌，纷纷退出了战场，在附近找了地方落脚观战。
<br/>　　“师父……”林三生看了广宗天师一眼，有些犹豫。
<br/>　　广宗天师点点头，转头对叶少阳道：“狐王应该传授你操控东皇钟的法术了吧，这东皇钟刚刚现世，灵力正盛，你用咒语操控它，敲响即可。”
<br/>　　叶少阳怔怔地看着他：“你怎么知道这么多？”
<br/>　　广宗天师笑道：“你忘了，是谁让你来这里的，是谁告诉你东皇钟在这的？”
<br/>　　叶少阳心中悸动，道：“燕赤霞呢？”
<br/>　　“你现在问这么干什么！”广宗天师喝道，“你想救活狐王，你自己就先得活着再说，快一些，再晚来不及了！”
<br/>　　叶少阳俯身看去，在失去了众人的抵挡之后，尸血已经蔓延上来，绕着东皇钟旋转，东皇钟自有一层灵力结界，隔绝着血水，但是浪头不断击打着结界，随时都有突破进来的危险。
<br/>　　为了小九……
<br/>　　叶少阳深吸了一口气，转头看着远处被后卿托在手中的芮冷玉，也为了冷玉……
<br/>　　盘膝坐下，用小九在神识中传授自己的法门，感知着身下的东皇钟的存在。
<br/>　　他与东皇钟之间，有小九的魂力从中间作为纽带，已经完成了祭炼，在用神识感知之后，自己仿佛与东皇钟融合成了一体。
<br/>　　所有人都在睁大眼睛看着他，眼中带着期待和怀疑，没有人知道东皇钟的攻击方式，还有它传说中的威力，究竟能达到什么样的地步。
<br/>　　“不动金钟，碧落悬空，妖皇降临，天下归元……”
<br/>　　叶少阳猛然睁开眼睛，双手虚握，在丹田处捏成半圆，口中发出一声石破天惊的吼叫：“疾！！”
<br/>　　东皇钟屹立没动，但内部却像是有什么东西敲响了，站在东皇钟上的一干人，立刻感受到了东皇钟的晃动，在这晃动之中，有一股神秘的力量，以波浪一般的形态蔓延出去。
