　　萧逸云随后说道：“其实我这些天，也在求证这件事，府君大人之前反对我参与这件事，现在也睁只眼闭只眼了，所以我才有机会来跟你走这一趟。”
<br/>　　叶少阳听他这么说，似乎崔府君也对自己是暗中支持，只不过不方便出面，不过他也就是自己想想，不敢拿出来说。
<br/>　　连萧逸云都信了，道渊真人和毛小方当然也不怀疑了，两人傻傻地站着，还在一脸懵逼中。
<br/>　　“叶天师……”毛小方忍不住问道，“你在百年之后，也是一代宗师？”
<br/>　　叶少阳想了一下，虽然自己不喜欢在熟人面前吹牛比，不过也有必要跟他们交代一下自己的经历，当下说道：“我是先天灵体，五岁被师父带上茅山，十三岁入内门，十六岁晋升天师，下山之后，学会先天八卦，和大周天吐纳心法，一路上结识了很多朋友，也收了很多门人，后来我师父……我师父羽化，我继承茅山掌教，在阴司开设阴阳司，受大帝御赐牌匾：人间天师。现在的我，是灵仙牌位，不是我吹牛比，我在人间法术界，也算是一号人物。”
<br/>　　听他说完，三个人都彻底怔住了。
<br/>　　萧逸云说道：“你说你十六岁晋升天师牌位，这不可能！何况你说你是灵仙牌位，你不过二十出头……”
<br/>　　至于叶少阳说的其他的那些经历，什么大帝钦赐牌匾之类的，听上去更加不可能。
<br/>　　叶少阳道：“是的，你没听错。我在你们跟前，不会撒谎，我现在看上去实力渣，是因为我丹田被毁，现在要重铸丹田，目前只有之前一两成的法力。”
<br/>　　“我相信你！”毛小方说道，看了看萧逸云和道渊真人，说道，“我跟他一路来的，一路上相处，虽然时间不常，不过我相信我的直觉，他绝对不是奸诈之人。”
<br/>　　叶少阳对他投去感动的一瞥。
<br/>　　道渊真人托着下巴，说道：“照这么说，你是法术界晋升天师的最年轻的人了？十六岁，比而今的张晓寒还要小一岁……”
<br/>　　叶少阳道：“也不是，我师兄道风，十五岁就游历天下，在二代弟子中打遍天下无敌手，他比我修行还要快，比我厉害的多。”
<br/>　　三人再次震惊。
<br/>　　萧逸云震惊道：“怎么可能会有这样的人？”
<br/>　　“是真的，我师兄现在斩却两尸，修成了五朝元气三花聚顶，还拿到了三……”叶少阳心神一紧，急忙打住，把“三清鬼符”几个字咽了回去，因为道风三清鬼符中的其中一枚，还是从龙虎山抢的……这件事可不能提前跟道渊真人说。
<br/>　　“总之，我师兄真的很厉害，单挑甚至打败了空界十二金仙中的黎山老母。在人间肯定是无敌的，在那个时候，也只有道渊老祖你在人间能跟他一较高低。”
<br/>　　三人听了之后，震惊地说不出话来。
<br/>　　道渊真人听他提到自己，喃喃道：“我在百年之后，有那么强？”
<br/>　　叶少阳也不往下说了。
<br/>　　毛小方忍耐了半天，说道：“那……我呢？”
<br/>　　叶少阳犹豫了一下，说道：“说出来也不怕知道，九十多年后，已经没有你了……”
<br/>　　毛小方爽朗地大笑，“正常，正常，我现在都三十了，哪里还能活九十多年。”
<br/>　　叶少阳道：“你在法术界，似乎没人知道，不过你在老百姓中间，却非常非常有名，有一部电视剧，叫《僵尸道长》，主人公是个法术高强的道士，说的就是你。”
<br/>　　既然话说到这里，叶少阳也不怕告诉他这些了，反正这种事对他的人生轨迹也不会产生什么影响。
<br/>　　“电……电什么？”毛小方不懂。
<br/>　　“这个……电影你知道吗？”
<br/>　　“哦，听说过，上海那边有电影院，说是可以在上面看到有人表演，就像唱戏一样？”
<br/>　　“额，大致是吧。电视就跟电影差不多，就是一个机器……嗯，我也形容不好，就像是一个匣子吧，在那个时代，家家户户都有，可以看各种节目……节目就跟唱戏一样，反正那时候不听戏了，有各种各样的节目。”在描述这件事的时候，叶少阳才发现自己的语言功底有多么多差，废了半天劲来形容，才让他勉强大致了解了电视和电视剧是个什么东西。
<br/>　　毛小方对于为什么会有关于他的电视剧拍出来这一点，也是十分的不解，叶少阳当然也提供不出什么解释。
<br/>　　“好了，不要说别的了，我们来讨论一下，叶天师，你有什么打算？”道渊真人打断他们的对话，问道。
<br/>　　一句话把叶少阳又拉回了现实，叹了口气，说道：“我真的不知道怎么办，我当时找萧郎君，他不鸟我，我只好来找你，我就是想找你一起商量的。”
<br/>　　道渊真人沉吟道：“就算我们相信你，又能怎么样呢？我们也没有办法逆转时空，徐福在哪里，我们也不知道。”
<br/>　　三人讨论半天，最后觉得，如果徐福也在这个时空，那或许也会跟法术界产生联系，可以动用法术界的力量来寻找他，或许有可能找的到。另外，萧逸云也答应回去之后，继续说服崔府君，看能否通过生死簿或者别的什么的力量，搜寻到徐福的存在——只要他在这个世界，就一定会留下痕迹。
<br/>　　然后大伙再慢慢琢磨，有没有别的办法，可以找到徐福。
<br/>　　虽然前途还不明朗，不过叶少阳心中却满满都是安慰，至少，自己在这个世界不再是一个人，至少有人知道了自己的身世，有一种压力得到释放的感觉，更不用说还多了三个盟友。
<br/>　　萧逸云没有任务在身，不能在人间呆太久，于是叶少阳画了血符交给他，凭着这个，萧逸云可以随时找到他在人间的位置，前来找他。
<br/>　　萧逸云先走一步，暂回阴司，临走之前，也向叶少阳询问起自己百年后的情况。
<br/>　　“是有一个关于你的大事，但是我不知道该不该告诉你……”叶少阳犹豫着，不知道该不该说他跟橙子成为一对的事，决定还是想一想，下次见面再决定要不要说。
