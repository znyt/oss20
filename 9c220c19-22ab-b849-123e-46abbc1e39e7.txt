　　叶尘眉头微微一挑，他又岂会看不出对方的意图。
<br/>　　这个张明德，他显然是打算，先将叶尘引入北寒城，然后再慢慢对付。
<br/>　　“好！我们现在就动身！”
<br/>　　叶尘大手一挥，直接答应了下来。
<br/>　　张明德连忙点头称是，心里却暗暗得意，
<br/>　　‘等到了北寒城，见了城主大人，一切可就由不得你了！’
<br/>　　……
<br/>　　在几人的引领之下，不到半个小时，便来到一座城池脚下，城门上写着三个大字：北寒城！
<br/>　　一行人进入城中，沿着宽阔的街道，往前走了十余里，来到一处巍峨的宫殿之前。
<br/>　　“前辈，我们这就去将情况禀明城主大人，您请再次稍后！”
<br/>　　张明德说完这话，冲着其他六人，迅速离开了，将叶尘一个人，留在了大殿前的广场之上。
<br/>　　叶尘却没有着急动手，依旧老神在在的站在那里，嘴角露出一抹冷笑。
<br/>　　以他如今的修为，只有化神天君境界的强者，也具备和他一战的能力。
<br/>　　而放眼整个天玄星域，化神天君觉得是凤毛麟角的存在。
<br/>　　而且，他刚刚动用神识，已经将整个北寒城探查了一遍，整体实力也就比流光派略胜一筹，在整个天玄星，应该属于末流势力，否则也不会定居在如此偏远的地方。
<br/>　　片刻之后，
<br/>　　哗啦啦！
<br/>　　伴随着一阵躁动声，足足有数百人，从四面八方飞奔而来，将叶尘团团围在了中央。
<br/>　　“父亲，就是这个人，他非要逼我们，在三天之内，抵达天玄城，否则就要杀了我们！哼！”
<br/>　　说话的，赫然是之前那名少女，不过此时却一脸得意之色。
<br/>　　而少女旁边，则是一名威严之色的中年男子，气息浑厚，显然修为不弱。
<br/>　　那中年男子上去一步，在叶尘身上打量了一番之后，摇了摇头，冷笑道：
<br/>　　“区区金丹境而已，也敢如此猖狂？念在你之前救了小女和我门弟子，本城主就不和你计较了，速速离去吧！”
<br/>　　原来，
<br/>　　这中年男子，便是北寒城的城主，北山泰！
<br/>　　而那名少女，则是北山泰的掌上明珠，北山薇儿。
<br/>　　叶尘瞥了一眼北山泰，淡淡道：
<br/>　　“要我离开也行，遵守你们之前的承诺，把九翅紫电鹫借我一用，我立刻就走！”
<br/>　　这时，张明德站了出来，一脸嘲弄之色道：
<br/>　　“小子！九翅紫电鹫乃是我北寒城的镇城圣兽，岂能随便借给你一个外人？如今既然已经来到我北寒城，可就由不得你了！”
<br/>　　叶尘两眼微微一眯，目光缓缓转移到张明德的身上，
<br/>　　“这么说来，你之前是在欺骗我了？”
<br/>　　张明德冷冷一笑，
<br/>　　“是有如何？我们城主大人宽宏大量，不和你一般见识，你若是识相的话，还是赶紧滚出北寒城吧！哈哈哈！”
<br/>　　说到最后，张明德得意的哈哈大笑起来。
<br/>　　叶尘脸色平淡，无悲无喜，淡淡道：
<br/>　　“欺我着，死！”
<br/>　　说完这话的同时，叶尘随手一指弹出，
<br/>　　嗖！
<br/>　　一道凌厉的紫色剑芒，朝张明德爆射而去！
<br/>　　“嗯？”
<br/>　　那北寒城城主北山泰，顿时两眼一瞪，
<br/>　　“敢在本城主的面前动手，我看你是想找……什么！”
<br/>　　北山泰一掌拍出，本以为可以将叶尘的攻击拦下，却不料那道紫色剑芒，飞至一半，突然诡异的消失在虚空之中！
<br/>　　“噗！！”
<br/>　　伴随着一道鲜血飞溅而出，那张明德的脑袋，已经直接飞了起来！
<br/>　　“啊！！”
<br/>　　周围众人显然也都没有想到，北山泰亲自出手，竟然都没有拦下对方的攻击，反而让他轻而易举的，就给杀掉了张明德！
<br/>　　“可恶的小子！竟然敢杀我爱徒，今日你休想活着离开北寒城！”
<br/>　　北山泰大怒之下，右手一翻，手中已经多出一把金色长枪，
<br/>　　轰！
<br/>　　北山泰一枪刺出，那金色长枪，陡然爆发出一道数百米的气芒，猛然朝叶尘的胸口狠狠刺去，快如迅雷闪电。
<br/>　　而叶尘似乎躲闪不及，竟然直挺挺的站在那里，没有丝毫躲避。
<br/>　　轰隆！
<br/>　　长枪刺落在叶尘的胸膛之上，后者身上顿时爆发出一道耀眼的金光。
<br/>　　接下来的一幕，却令所有人都为之震撼，
<br/>　　只见，北山泰的全力一枪，刺在叶尘的身上，犹如挠痒痒一边，竟然连他的护体罡气都没有破开！
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，身上金芒大盛，
<br/>　　蹬蹬蹬！
<br/>　　北山泰顿时感觉到一股，强悍无匹的力量，自手上的长枪传来，顿时将他震的连连倒退，足足退了十几步，方才勉强站稳身形，体内气虚翻滚，险些吐出一口血来。
<br/>　　“这——这怎么可能！”
<br/>　　北山泰两眼瞪得老大，简直无法相信会是这么一个结果，
<br/>　　“你——你到底是什么人？明明只有金丹境初期的修为，怎么可能挡得住我的天雷碧炎枪一击？”
<br/>　　要知道，他好歹也是元婴境的实力，对方竟然能以肉身，硬抗他的全力一击，对方的实力，得强大到何等程度？
<br/>　　叶尘随手拍了拍胸口的衣衫，淡淡道：
<br/>　　“我是谁并不重要！交出九翅紫电鹫，可饶尔等不死，否则，休怪我要血洗你们北寒城！”
<br/>　　叶尘说出这话的同时，身上的气息，毫无保留的释放了出来。
<br/>　　轰隆！
<br/>　　周围众人只感觉到，一股惊人的杀戮气息，自眼前这个神秘少年的释放了出来，压得他们险些透不过气来，甚至一些修为稍弱的，已经忍不住跪俯在地上，瑟瑟发抖。
<br/>　　北山泰的脸色，同样变得难看无比，
<br/>　　眼前这个神秘少年的实力，远远超出了他的预想，就凭刚刚的表现，他纵然动用整个北寒城的实力，勉强能够击败此人，那也必将付出极为惨重的代价！
<br/>　　说不定，对方真的具备血洗北寒城的实力……
<br/>　　而相比之下，一个徒弟的性命，和一头九翅紫电鹫，也就算不上什么了。
<br/>　　“别别！前辈，有话好好说！九翅紫电鹫，我借给您便是了！”
<br/>　　北山泰在心里衡量了一番之后，果断选择了认怂。
<br/>　　“父亲！他刚刚杀了张师兄，您怎么能……”
<br/>　　一旁的北山薇儿，显然心有不甘。
<br/>　　“闭嘴！”
<br/>　　北山泰直接一声暴喝，将她的话生生打断，转而向叶尘赔笑道：
<br/>　　“小女口无遮拦，前辈勿怪！”
<br/>　　叶尘嘴角微微一扬，
<br/>　　“无妨！我此去天玄城，不熟悉路线，就由令爱随我一同前往吧！”
<br/>　　说完这话的同时，叶尘虚手一抓，
<br/>　　“啊！”
<br/>　　北山薇儿顿时一声娇呼，下一刻，已经自动飞到叶尘跟前，被他一把按住了肩头。
<br/>　　众人顿时再次脸色大变。
