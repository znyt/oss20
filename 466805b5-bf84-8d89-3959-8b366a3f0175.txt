　　第207章
<br/>　　空气仿佛突然安静下来，周围荡漾着一种诡异的气氛，只有纱衣女子的琵琶声还铮铮响着。
<br/>　　“好啦，小桃，你先下去吧。”青衫男子摆摆手。
<br/>　　“是。”纱衣女子恭恭敬敬退下。
<br/>　　屋子里还是没人说话，飞天蛤蟆张张嘴，却感觉有某种莫名的力量压着他无法张口。
<br/>　　良久，白袍男子咧咧嘴：“有点尴尬，对不对？”
<br/>　　徐灵儿张张嘴巴：“大当家，真的是你么？”
<br/>　　程大雷摊开双臂，赤脚从石棺里跃出来。
<br/>　　“货真价实，如假包换。”
<br/>　　徐神机三人凑过去，这个捏捏程大雷的胳膊，那个抓抓程大雷的头发。
<br/>　　“是真的诶。”
<br/>　　“活的，活的，还会动呢。”
<br/>　　“是不是被鬼附身了。”
<br/>　　飞天蛤蟆大惊：“你们住手，怎可对大哥如此无礼，快快住手。”
<br/>　　随后他惊讶的发现，程大雷站在三人中间，虽然满目尴尬，却没有半点动怒的意思也无。
<br/>　　难道他们认识？
<br/>　　“好啦，好啦。”程大雷挣开三人：“捏够了没有，是活的，不要再摸了。”
<br/>　　此刻程大雷忽然发现，徐神机三人已眼眶烦湿，徐灵儿轻声抽泣：“大当家，我们找得你……好苦。”
<br/>　　程大雷伸出手揉揉她的脑袋，却发现一年不见，她出落的更加亭亭玉立。徐神机和秦蛮的容貌也有改变，二人风尘仆仆，都瘦了几分。
<br/>　　“我懂，我懂，你们在找我，我何尝不在找你们。”
<br/>　　程大雷挥挥手：“小飞，你下去取几坛好酒来，今日我兄弟到了，大家不醉不归。”
<br/>　　人群鱼贯而入，座椅板凳立刻准备好，上面摆满酒菜。
<br/>　　徐神机看着这一幕愣了愣，道：“大当家，您怎么变成这个样子了？”
<br/>　　“哈，这说来就话长了……”
<br/>　　说来的确话长。
<br/>　　一年前，程大雷坠下悬崖，虽然仗着系统轻功，勉强没死，不过也足足养了一月的伤，方才痊愈。之后，也不知道该往那里去，便想既然身在这个世界走一遭，自然要见一见那天下第一城的长安城。
<br/>　　于是一路颠沛，来到长安，他也不敢在人前露面，如他这样的人，都是躲在地下城内。地下城本有一位老大，程大雷这初来乍到，自然有下马威，杀威棒之类的东东。
<br/>　　那人冲程大雷说：年轻人呀，你还年轻，日子还很长，要懂规矩，身上值钱的东西就拿出来吧。以后给我做事，叫我一声大哥，以后你就是我的小弟。
<br/>　　程大雷毕竟是手上沾过十万鲜血的主儿，虽然如今落魄了，但杀威犹在，他那里受得了这个。当场一脚就把对方腿骨踹断了。
<br/>　　大闹一场，地下城十几个猛人和程大雷打，程大雷以暴制暴，一一放倒。然后他就取而代之，成了这地下城的『大哥』。
<br/>　　之后的事情就简单了，程大雷还是有些管理经验的，恩威并施，地下城有人出事，他也会管，不是一味的压榨人，比如飞天蛤蟆曾经被人捉了，也是程大雷亲自救出来的。如此，仅仅半年时间，这座地下城便欣欣向荣，这里每人都唤程大雷一声大哥，倒也是心悦诚服。
<br/>　　“其他人的情况怎样？”程大雷问。
<br/>　　“高飞虎高飞豹兄弟还在江南，依旧占山为王，至于其他人，我们这一年来颠沛流离，也没再和他们遇到过。”徐神机。
<br/>　　“如今兵荒马乱，那些失散的兄弟也不知道如今是什么情况。”程大雷轻轻叹口气。
<br/>　　“这次武科举当有很多兄弟到，说不得我们会遇到。”徐神机。
<br/>　　“我也想到了，所以在城里留了标记，弟兄们看见了标记，就能找到我们的。”
<br/>　　徐神机一楞：“什么标记，我怎么没看见？”
<br/>　　“不太可能吧，我撒出去很多人，每座城门外我都派人摆了茶摊，他们幡子上都画着蛤蟆，你们路过的时候没注意么？”
<br/>　　“呃……这个，军师当初怎么说来着？”秦蛮。
<br/>　　“这只小狗好胖喔。”徐灵儿一副生无可恋。
<br/>　　“呃……我觉得你们找不到我，的确是有原因的。”
<br/>　　徐灵儿看着周围，忽然道：“大当家，刚才那个女人，是……”
<br/>　　“一支桃啊，怎么你也认识？”
<br/>　　“啊！”徐灵儿一声惊呼，最后悠悠的看着程大雷：“大当家，你堕落了。”
<br/>　　程大雷一怔，发现三人都用鄙视的眼神看着自己。他无奈的摊开双手：“我们是清白的。”
<br/>　　三人同时摇摇头。
<br/>　　正这时候，一支桃缓缓行来，温婉笑道：“知道大哥的弟兄来了，奴冒昧过来敬杯酒。”
<br/>　　说着，将杯中酒一饮而尽，冲徐神机三人道了个万福，然后又冲程大雷道：“今日奴就不扰大哥的酒兴，改日大哥欢喜时，再唤奴来陪你，奴一直候着。”
<br/>　　三人看着一支桃款款摇曳的倩影，目光又挪到程大雷身上。
<br/>　　“真是青白的。”程大雷无辜道：“当初有大人物想要她身子，她被逼的走投无路，托关系求到我身上。一个弱女子，孤苦伶仃的，我也不能看着不管是不是。”
<br/>　　“关键还是个美人。”徐灵儿。
<br/>　　“嗯，这一点还是蛮关键的。”程大雷下意识点点头，忽然又道：“真是青白的。”
<br/>　　正这个时候，飞天蛤蟆抱着一坛酒过来，道：“大哥，大哥，我这是特意潜进王府取来的，十八年的杏花醉，天上地下唯此一坛。”
<br/>　　“这个人又是怎么会事？”徐神机。
<br/>　　“啊，他叫白飞元，自小父母双亡，是山林里的老猿养大的，十二岁才见到活人，生得脚长手长，于是就做了飞贼，有次被官兵抓了，是我救的他。我又告诉他，人设很重要……”
<br/>　　白飞元走后，一一又有人过来，都是地下城的居民，大多都受过程大雷恩情。
<br/>　　“飞天蛤蟆，一支桃。”徐灵儿微微念叨着：“还差一个明玉公主，这明玉公主该不会也和你有关系吧？”
<br/>　　“什么明玉公主，喔，你们说婉儿啊。”程大雷抿了口酒，忽然一楞：“你们都这个眼神看着我干嘛？”
<br/>
