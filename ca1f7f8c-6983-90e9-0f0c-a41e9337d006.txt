　　萧逸的上台，霎时让得周遭一阵哗然。
<br/>　　一来，气宗器宗两脉的比拼竟也有人敢插手？
<br/>　　二来，气宗第一天骄，竟是被一道剑气便震得节节后退？
<br/>　　而且，挥出剑气的，还只是个看起来比桌万宗还年轻些许的武者。
<br/>　　几乎是一瞬间，萧逸成了全场瞩目的焦点。
<br/>　　不过，萧逸并未在意这些目光，而是指尖在丁秋月手臂上一划，一道元力打入。
<br/>　　哗…
<br/>　　丁秋月原本被封的元力，顷刻恢复。
<br/>　　“谢谢大哥哥。”丁秋月甜甜一笑。
<br/>　　“走，下去吧。”萧逸淡淡一笑。
<br/>　　恰在此时，气宗席位上，一个老者飞跃而出。
<br/>　　老者，乃是气宗三长老，一身修为与实力在剑域之内人人尽知，乃是个不折不扣的顶尖武者。
<br/>　　老者一上台，立刻怒目相视。
<br/>　　苍老的目光中，却泛着不善与冰冷。
<br/>　　“哪来的野小子，扰了两宗比拼不说，还忽然出手伤我气宗天骄，现在拍拍屁股便想下台？”
<br/>　　“那你想如何？”萧逸停下脚步，转过身，淡漠地问道。
<br/>　　对于气宗的人，萧逸毫无好感。
<br/>　　气宗六剑侍数日前的围杀，已然萧逸将气宗列入了是敌非友的名单。
<br/>　　“很简单。”气宗三长老冷声道，“两宗比拼，乃是我剑域一等一的大事，就是我剑域剑者都不敢擅加干扰。”
<br/>　　“你一外来武者，却如此破坏规矩，按例，需受我剑域剑者一剑，轰下比武台。”
<br/>　　“这是其一。”气宗三长老渐渐面露威严之色，甚至带上了一抹俯视的态度。
<br/>　　“其二，你伤我气宗天骄，也需给我气宗一个交代。”
<br/>　　“老夫也不为难你，你受老夫一剑即可。”
<br/>　　气宗三长老话音落下。
<br/>　　周遭，再次一阵哗然。
<br/>　　“这气宗三长老，是出了名的得理不饶人之辈，这萧逸小贼此次怕是不死也要掉层皮。”
<br/>　　“还有，气宗三长老乃是武道皇者巅峰，等闲之辈，谁能受他一剑？”
<br/>　　“萧逸这小贼，此次要麻烦了。”
<br/>　　比武台上，丁秋月听闻周遭话语，担忧地看着萧逸，“大哥哥。”
<br/>　　“没事。”萧逸摇了摇头。
<br/>　　瞥了眼气宗三长老，萧逸直接道，“第一点好说。”
<br/>　　“至于第二点，呵。”萧逸不屑一笑，“你气宗天骄技不如人，即便受伤，又有何好说？”
<br/>　　“莫不是打了小的，你们这些老家伙便要出来了。”
<br/>　　“混账。”气宗三长老须发皆怒，重声道，“若非你忽然出手，打了万宗一个措手不及，万宗岂会不敌。”
<br/>　　“难不成我气宗天骄，会怕了你这个外来的野小子不成？”
<br/>　　“忽然出手？”萧逸不禁面露戏谑笑容。
<br/>　　“行吧，便如你所说。”萧逸懒得解释。
<br/>　　“第一点，我可以受你一剑，但能否轰我下台，便看你的本事了。”
<br/>　　“第二点，你要一个交代，也可以，便让你打一剑。”
<br/>　　“合共两剑，来吧。”
<br/>　　萧逸的声音，很淡，却也充斥着毫不在意的意味。
<br/>　　气宗三长老见状，霎时气得老脸通红，“好，很好。”
<br/>　　“老夫也听闻过你萧逸小贼的名声，果真如传言那般，狂妄至极。”
<br/>　　“好，老夫便看看你是否真有那个轻视老夫的资格。”
<br/>　　锵…
<br/>　　一把利剑，凭空出现于气宗三长老手中。
<br/>　　氤…
<br/>　　一抹白色光芒，游走于气宗三长老指脉之间。
<br/>　　显然，气宗三长老已调动剑力，融入自身剑气和元力之中。
<br/>　　“大哥哥。”丁秋月脸色一惊。
<br/>　　在她的印象中，萧逸这个大哥哥无疑实力不俗。
<br/>　　可气宗三长老这个剑域有名的强者，却更让她为之惊惧。
<br/>　　“没事。”萧逸踏前一步，将她护在了身后。
<br/>　　这时，气宗三长老仗剑而来。
<br/>　　手中之剑，已然剑气凝聚。
<br/>　　剑气，锋锐到极点，亦快到了极点。
<br/>　　萧逸不躲不闪，嘴角咧过一道轻蔑笑容。
<br/>　　“找死。”气宗三长老冷笑一声，“也罢，你这等小贼，留在世上也是祸害。”
<br/>　　气宗三长老话音落下的瞬间，剑锋恰好抵至萧逸胸膛。
<br/>　　不过，此时剑上的剑气，却未能损萧逸分毫。
<br/>　　萧逸身上，一股惊天剑意爆发。
<br/>　　道道星光剑气，护于身前。
<br/>　　锵…
<br/>　　星光剑气陡然爆发。
<br/>　　萧逸甚至没有融入剑力，仅仅是星光剑气本身的威力，已将气宗三长老的剑重重震开。
<br/>　　气宗三长老只觉手臂一疼，一股距离从剑上传回手中。
<br/>　　还未反应过来，已脸色潮红，体内一阵气血翻滚。
<br/>　　嘭…一声爆响。
<br/>　　气宗三长老直接被震退数十步。
<br/>　　“就这点本事？”萧逸不屑一笑。
<br/>　　“气宗三长老，还有一剑，来吧。”
<br/>　　“你…”气宗三长老还未来得及稳下体内翻滚气息，此刻闻言，险些气得一口腥血吐出。
<br/>　　萧逸那轻蔑而不屑的语气，无疑是对他的最大讥讽。
<br/>　　比武台外。
<br/>　　云渊长老与讳执事见状，蓦地一笑。
<br/>　　“我懂了，这小子，有够记仇的。”云渊长老好笑道。
<br/>　　“数日前气宗六剑侍围杀与他，这般以大欺小，偏偏这小子那时剑力不足，被欺得险些只能远遁。”
<br/>　　“现在这小子剑力足够了，以这小子记仇的性格，不找他们麻烦才怪。”
<br/>　　数日前气宗六剑侍的围杀之事，萧逸早便与二人说过，二人亦知晓。
<br/>　　讳执事笑笑，“盛事就要开始了，趁还有些时间，让他好生出口恶气也好。”
<br/>　　“否则，心头憋着一口气，如何全力参加之后的盛事。”
<br/>　　“也对。”云渊长老点了点头。
<br/>　　同一时间，两人对视了一眼，目光，看向了气宗席位。
<br/>　　气宗席位处，一个中年人注意到这两道目光，脸色微微一变。
<br/>　　同时，一道传音，传入中年人耳中。
<br/>　　“气宗宗主，小辈间的小打小闹，想必你这等人物，不会不顾身份出手吧。”
<br/>　　气宗席位处的中年人，正是气宗宗主。
<br/>　　原本比武台上三长老被震退，他有些坐不住；但此时感受到这两股冰冷的目光，只得按捺下要出手的动作。
<br/>　　两大巅峰剑修的警告，哪怕他这个气宗宗主，也不敢直接无视。
<br/>　　云渊长老与讳执事见状，淡淡一笑，收回了目光。
