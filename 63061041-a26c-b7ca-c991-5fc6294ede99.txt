　　此时，天墉城中央主殿中。
<br/>　　陵端与肇临跪在下方，涵素真人脸色阴沉的坐在掌门椅上，高低起伏的胸膛，可见他的心情是何等的糟糕。
<br/>　　原本，今日慕清霄回到天墉城，他心情挺不错的，却没想到，自己门下弟子，做出这等恶劣的事情。
<br/>　　其中一名，更是由他亲自教导，若是传出去，他还有何颜面可言。
<br/>　　若非被凝丹长老撞见，命两人前来请罪，说不定这两个家伙，往后会做出更加恶劣，更不可原谅之事。
<br/>　　简直不将戒律堂放在眼中啊！
<br/>　　想到此处，涵素真人气不打一处来，但想到陵端与肇临天赋都不错，门下除了芙蕖也没有其他弟子，暂时压下了心中的怒气。
<br/>　　最主要的是，他知道慕清霄是不想拨了他的面子。
<br/>　　要知道，陷害同门可是死罪，身为凝丹长老的慕清霄，完全有资格先斩后奏。
<br/>　　但后者却没有，反倒让陵端与肇临来自己面前请罪，可见后者确实为他考虑，心情变得不错起来。
<br/>　　“你们真可是有能耐啊，陷害同门，陵端，肇临你们两个可知罪！”
<br/>　　见涵素真人微怒，陵端连忙磕头，额头都磕出了血，脸色惨白，心中却是在祈祷，希望惩罚轻一点。
<br/>　　“弟......弟子知罪，弟子知罪，请师尊惩罚！”
<br/>　　“弟子知罪，请掌门真人惩罚！”
<br/>　　揉了揉眉心，涵素真人叹了口气，道：“念在你们是初犯，并且没有铸成大错，罚你们两人去后山禁闭一年。”
<br/>　　“是，弟子领罪。”
<br/>　　听到只是关一年的禁闭，陵端瞬间松了口气，眼中也露出一丝笑意。
<br/>　　“等等，顺便给我将道德考校经文抄写三百遍，希望你们能记住这次教训，若非凝丹长老网开一面，你们现在已经是死人。”
<br/>　　“是......是，师尊。”
<br/>　　身躯哆嗦了一下，陵端刚放松的心情，顿时又紧张起来，心中暗自叫苦，抄写三百遍，那不得抄到吐啊！
<br/>　　“嗯，去吧。”
<br/>　　...
<br/>　　...
<br/>　　天墉城西侧，一座宫殿后山。
<br/>　　慕清霄盘坐与悬崖上，睁开双眼，嘴中吐出一口浊气：“练虚九重，也是时候将事情了解了。”
<br/>　　“小兰，你前往外门，将名为欧阳少恭的新进弟子请过来，速去。”
<br/>　　正在宫殿中打坐修炼的小兰，听到慕清霄的声音后，立刻动身，心中虽困惑，却不敢迟疑。
<br/>　　约莫半刻钟后，小兰归来，身后跟着一名年轻俊美的男子，正是欧阳少恭。
<br/>　　“长老一直在后山修炼，你自行过去吧。”小兰吩咐一声，便转身离去。
<br/>　　“多谢师姐。”
<br/>　　说罢，便迈步向后山走去，心中则暗自猜测，凝丹长老找他有何目的，虽只见过两面，但他总觉得，凝丹长老充满了神秘。
<br/>　　顺着林荫小道来到后山，望着盘坐于悬崖上的白衣青年，欧阳少恭面色如常。
<br/>　　“欧阳少恭，见过凝丹长老。”
<br/>　　闻言，慕清霄站起身来，转头深深的看了他一眼，旋即抬头望天。
<br/>　　“你觉得天道如何？”
<br/>　　“呵，天道不公！”
<br/>　　欧阳少恭下意识开口，说这句话的同时，身上的气息陡然一变，回过神来，微微一笑，道：“弟子只是随口说说而已，长老莫怪。”
<br/>　　慕清霄微微一笑，欧阳少恭经历过太多的绝望，自然不可能觉得天道公平。
<br/>　　而且，无论世间，或者是天道，何尝公道过？
<br/>　　“你可知道，我找你来，所谓何事？”
<br/>　　“弟子不知。”
<br/>　　微微点头，慕清霄也不打算在拐弯抹角，道：“我需要你手中的两样东西。”
<br/>　　欧阳少恭眉头皱了皱，目光变得深邃起来，眼神眯了眯，似乎要将慕清霄看个通透。
<br/>　　“我身上，恐怕没有长老需要的东西。”
<br/>　　“有没有，不是你说了算，而是我说了算。”说罢，慕清霄接着道：“难道，你不想知道巽芳的消息吗？”
<br/>　　其实，思考了数年，慕清霄都没有找到任何制约长琴太子的方法，若非如此，他真不愿意将女人当作筹码。
<br/>　　因为，慕清霄知道长琴太子对巽芳的感情，但他也知道，这是唯一的办法。
<br/>　　从小生活在得天独厚的蓬莱国，十分善解人意，富有同情心。
<br/>　　百年前，少女时的巽芳偷跑到中原游玩，被欧阳少恭的某一世渡魂所救，两人回到蓬莱国过了一段神仙眷侣般的生活。
<br/>　　后蓬莱天灾降临，她虽容貌变老，但仍逃出蓬莱国，去往中原寻找欧阳少恭。
<br/>　　蓬莱国人的寿命比普通人长久许多，但是终究躲不过容颜老去。
<br/>　　当巽芳找到分离多年后的少恭时，自惭形秽，不愿相认，她希望少恭心中的自己永远都是青春貌美，于是化名“寂桐”守护在其身旁。
<br/>　　话语刚落，欧阳少恭瞳孔一缩，恐怖的气势一闪即逝，沉声道：“你到底是谁，你有关于巽芳的线索？”
<br/>　　“我是谁并不重要，重要的是，我们的交易能不能顺利进行下去。”慕清霄语气平淡，哪怕是曾经的长琴太子，也并非不能对付。
<br/>　　深吸了口气，压下心中的愤怒，欧阳少恭心平气和道：“你想从我身上获得什么东西？”
<br/>　　“我需要烛龙之鳞！”
<br/>　　闻言，欧阳少恭眯了眯眼，声音冰冷道：“希望你不要骗我，否则，你将会后悔来到这个世上。”
<br/>　　“我若想走，没人能拦着我，就算仙也不行。”
<br/>　　他可真没想装逼，若他想逃走，支付几万掠夺积分，返回地球就行了，难道欧阳少恭还能找到自己不成？
<br/>　　别说是欧阳少恭，哪怕天界的仙人，也绝对留不下他，更找不到自己。
<br/>　　微微摇头，慕清霄微微一笑，伸出手来道：“巽芳还活着，你觉得这个信息，值不值得你用烛龙之鳞换取？”
<br/>　　欧阳少恭身躯猛的一颤，眼中满是回忆，脑海中闪过一道曾经依偎在他怀中的倩影。
<br/>　　沉吟片刻，慢慢取下腰间的烛龙之鳞，递到了慕清霄的手上：“说吧，你还想要什么？天界的仙术，仙丹法宝？这些我都可以给你。”
<br/>　　闻言，慕清霄并未说话，而是打量起手中的烛龙之鳞。
<br/>　　欧阳少恭则站在他的身后，眼眸中含有一丝期待，也有一丝杀意。
