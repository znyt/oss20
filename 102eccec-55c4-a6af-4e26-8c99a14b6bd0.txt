　　　　几分钟过后，张小凡拨开人群走了过来，向慕清霄点了点头，发现田不易脸色有些难看，隐晦的瞥了眼杜必书几人，看到他们沮丧的模样，顿时知道了原因。
<br/>　　田不易慢慢转过身来，发现张小凡悠哉悠哉的走过来，身上没有一丝伤痕，眼中闪过一丝惊诧。
<br/>　　不仅是田不易，苏茹美目中也闪过一丝诧异，杜必书几人目光也纷纷落在张小凡身上，唯有慕清霄面色淡然。
<br/>　　张小凡走到田不易的面前，田不易看着这名平日里自己最忽视的弟子，望着他眼中的自信，神情一滞，心中突然升起一阵荒诞的想法。
<br/>　　回过神来，田不易望着张小凡清秀的脸蛋，道：“老七，比试结果如何？”
<br/>　　此时，擂台之上，田灵儿正与朝阳峰弟子，你来我往，仙气凛然，互相斗的不可开交。
<br/>　　张小凡深深的看了一眼田灵儿所在的擂台，随后看向面前的师傅，看到他肥胖脸颊上含有一丝期待，还有那么一丝若有若无的关怀。
<br/>　　他自信一笑，道：“幸不辱命，徒儿又胜了一场。”
<br/>　　话语刚落，大竹峰一众人等面面相觑，周围气氛相当诡异，特别是杜必书几人，眼睛瞪如铜铃，若是记得没错，张小凡这一场比试的对手，是风回峰的彭倡师兄，修为比起他们，只强不弱。
<br/>　　第一次，张小凡获得胜利，他们都认为是运气。
<br/>　　第二次又获得胜利，他们认为是走了****运，但第三次获胜，他们看向张小凡的目光，都变得有些复杂起来，若是说运气好，理由是不是太扯淡了些？
<br/>　　七脉会武上的弟子都是各脉精英中的精英，能有什么逆天气运，让这些精英弟子，三番两次败在张小凡手中。
<br/>　　此时，苏茹脸上挂起温柔的笑意，终于知晓，为何当初看向张小凡的时候，后者隐隐让她看不透，女人的直觉很敏锐，本以为是错觉，现在看来也不尽然。
<br/>　　......
<br/>　　......
<br/>　　通天峰客房中，大竹峰一脉弟子拥挤在其中，田不易与苏茹坐在中央的桌上，张小凡就站在身前，宋大仁几人就围着一圈，目光紧紧盯着后者。
<br/>　　田灵儿望着张小凡，张小凡也正好看向她，两人目光相交，彼此都能看眼中的温柔，田灵儿芳心羞涩，急忙低下臻首，纤纤玉指捏着衣角，完全是一副女儿家的羞态。
<br/>　　坐在一旁，察觉到田灵儿的羞态，顺着她的目光看去，苏茹微微一愣，回过神来，身为过来人的她，又怎会不知情，脸上挂上温柔的笑容。
<br/>　　“小凡，你的对手是风回峰的弟子，想必很辛苦吧？”
<br/>　　张小凡受宠若惊，感激之情仍是溢于言表，低声道：“让师父和师娘操心了。”
<br/>　　田不易喝了口茶水，抬起头来，冷哼一声道：“老七，你可真有能耐啊，瞒着我们这么久，那风回峰的彭倡可是玉清境四重境界......”
<br/>　　张小凡又是一惊，由于功法的缘故，此刻的他已经脱胎换骨，自然清楚田不易话语中的意思，道：“师父，我不像师姐他们......”
<br/>　　说着说着，他的声音是越来越小，只是身前诸位师兄的脸色都有些古怪，尤其是站在最前面的大师兄宋大仁，脸色更是苍白无比，不复平日里的生气，给人一副摇摇欲坠的模样。
<br/>　　苏茹叹了口气，摇头道：“必书，快给你大师兄搬张椅子坐下。”
<br/>　　杜必书连忙应了一声，从门口处拿了一张椅子放到宋大仁身旁，宋大仁本想拒绝，但身子摇了摇，体力明显有些不支，终究还是坐了下来，大口喘息着。
<br/>　　张小凡愣了愣，道：“大师兄，你这是怎么了？”
<br/>　　宋大仁苦笑一声，低下头却没有开口说话，倒是一旁的老四何大智道：“小师弟，现在七脉会武到了最后一轮，我们大竹峰只剩下你一人了。”
<br/>　　说到这里，他情不自禁的向四周看了一眼。
<br/>　　张小凡若有所思的点了点头，转头看向坐在床头的田灵儿，道：“师姐，那你也......”
<br/>　　田灵儿美眸一黯，低声道：“朝阳峰的师兄道行深厚，虽然我与他斗法上百招，可最后还是败下阵来。”
<br/>　　张小凡看到她神色间一片失望，心中一揪，但此刻却容不得他胡思乱想，最后目光落在慕清霄身上，他可不相信这个神秘的师弟会输。
<br/>　　何大智苦笑道：“小凡，你也别看了，慕师弟已经失去比试资格了。”
<br/>　　“......到底怎么回事？”张小凡皱了皱眉道。
<br/>　　“你也知道，前两场比试，慕师弟废了龙首峰两名弟子，在第三场比试的时候，将龙首峰的大师兄，齐昊也给废了，虽然掌门令慕师弟在七脉会武结束后，前往青云门祠堂面壁思过半月，但苍松师伯却不肯善罢甘休，最终连比试的资格也剥夺了去。”
<br/>　　闻言，张小凡苦笑一声，对此也见怪不怪，只是没想到，慕清霄下手这么狠，直接把齐昊给废了，替他排除了一名竞争对手：“那我明日的对手？”
<br/>　　田灵儿一脸崇拜，娇笑道：“小凡，本来最后一轮比试也是抽签决定对手，可由于慕师弟的缘故，你明天顺利晋级，必须等小竹峰的陆雪琪与龙首峰的林惊羽决出胜负，胜者便是你的对手。”
<br/>　　此时，田不易上上下下打量了张小凡一番，发现越发的看不透这位弟子，脸色一沉，道：“老七！”
<br/>　　张小凡心中一跳，听出田不易语气中带有一丝怒意，再看师父脸色极其的难看，便不由自主地有些畏惧，道：“是，师傅，有什么......”
<br/>　　话语还未落下，便被田不易截断，道：“你这一身道法修行，是从何而来？”
<br/>　　张小凡呆了呆，张了张嘴，一时间竟不知如何开口。
<br/>　　他向屋内所有人看去，只见平日里熟悉和蔼的师兄们，此时也保持了沉默，看向自己的目光中有疑惑之意。
