　　“蚩尤，可敢一战！”
<br/>　　公孙轩辕那霸气的声音在逐鹿长空响起，久久不散。
<br/>　　“哈哈……”
<br/>　　蚩尤怒极反笑，大声喝道：“你要战，那便战，你们谁先来？”
<br/>　　轩辕身穿皇极战甲，右手变掌，拍向旁边虚空。他手捏印诀，虚空紧接着便震动起来，瞬间便出现一道无形的乾坤八卦图，乾坤轮转，八卦易位，阴阳逆转，于那阴阳八卦之间，缓缓出现一柄金色剑柄，剑柄之上，一面书农耕畜养之术，一面书四海一统之策。
<br/>　　公孙轩辕手握剑柄，缓缓将剑抽出虚空，随着剑身露出，一柄金光闪闪的长剑出现在众人面前，剑身一面可日月星辰，一面可山川草木。
<br/>　　正是轩辕剑！
<br/>　　轩辕剑，圣道之剑，采首山之铜，日月之精，以天地之火，用乾坤之鼎所炼。
<br/>　　公孙轩辕单手握剑，剑锋所指，正是蚩尤方向。
<br/>　　“杀！”
<br/>　　“无知小儿，受死！”
<br/>　　被一向看不起的公孙轩辕挑衅，蚩尤大怒，大喝一声，便持手中虎魄魔刀飞杀上前。
<br/>　　公孙轩辕亦飞身而出，手中金色圣光闪烁不断，便祭出轩辕圣剑，剑锋所指，虚空为之一颤，锋利的剑气划破天地，斩了出去，斩向飞来的蚩尤。
<br/>　　“哼！”
<br/>　　蚩尤冷哼一声，手中那柄通体乌黑、血光缠绕的虎魄魔刀，直接迎着剑气，向着轩辕劈去，带着凌厉无比的血**光，斩破剑气，划破一切阻碍，杀向轩辕。
<br/>　　蚩尤魔威滔天，全身煞气逼人，本身就是顶尖大巫，修炼巫族神通，这一刀下来，若是平常大罗金仙，都要退避三舍。
<br/>　　公孙轩辕却毫不畏惧，剑光一闪，放出万丈金色圣光，应了上去。
<br/>　　“铛——”
<br/>　　剑锋相交，蚩尤倒退三步，公孙轩辕退了六步，第一次交锋，轩辕略逊一筹。
<br/>　　蚩尤冷笑一声，眼中的不屑，依旧是原样，不将让何人放在心里。他手中刀锋一转，卷起无尽刀气，充斥天地，血色刀气瞬间成为狂暴的风浪，向着公孙轩辕劈去，声势浩大，似乎想要两者天地都要劈碎。
<br/>　　“战！”
<br/>　　轩辕战意高昂，第一次落日下风没有丝毫气馁，比起第一次与蚩尤大战被打得落荒而逃，现在的情况，已经很好了。况且，这只是首次交锋，他还有好多手段没有施展呢。
<br/>　　话音刚落，轩辕手中圣剑不停，一件劈出，金光璀璨，劈开上天的乌云，金光落下，毁灭一切，金光之中蕴含着强烈的毁灭之光，直接将持有的血色刀气毁灭个干净。
<br/>　　剑光与刀气同时消散一空，公孙轩辕手中的轩辕剑不停，剑光一闪，带着强悍的杀戮之意，斩向蚩尤。
<br/>　　蚩尤又岂会惧战，他手中魔刀绽放出强烈血色刀光，与轩辕战在一起。
<br/>　　两人你来我往，一时竟然难以分出胜负。
<br/>　　战场之上，剑光森寒，刀气霸道，杀机与煞气纵横天地。
<br/>　　双方交战极为凶悍，巫族神通诡异霸道，而人族武道也同样霸道，再加上轩辕元神强大，压制没有元始的蚩尤，一时竟僵持住了。
<br/>　　见自己竟然拿不下一直看不起的轩辕小儿，蚩尤不由得有些恼羞成怒了，他突然大喝一声。
<br/>　　“巫体本相，喝！”
<br/>　　随着话音刚落，蚩尤变出巫族的强悍本体，身形开始无限涨大，顷刻间，就从一个九尺大汉，变成一个高大百万丈高大的巨人，他头如山岳，腰似峻岭，眼若闪电，口如血盆，牙如剑柱，面如牛首，头顶双角，仿佛能够刺破虚空，背生双翼，能够翱翔天际。
<br/>　　此乃巫族的神通——本体神通，每个巫族都会，这才是他们的本体，能够战力发挥到极致，甚至能增幅战斗力。平时的样子乃是他们变化成人的模样，不是真实的模样。
<br/>　　想当年巫族称霸洪荒，纵横天地之时，靠的就是巫族强悍无比的肉身神通，每个巫族都能变化出强大的法相，如帝江，如黄囊，赤如丹火，六足四翼，浑敦无面目。再如句芒，青若翠竹，鸟身人面，足乘两龙，等等。
<br/>　　蚩尤的肉身神通虽然比不上祖巫那般强悍，但也不弱，对上一般准圣亦可战斗。
<br/>　　轩辕一方众仙，见蚩尤如此威武，大多都变了脸色。
<br/>　　也只有广成子，孔宣等人面色不变，依旧从容淡定，仿佛早知如此。
<br/>　　曾经在洪荒辉煌一时的巫族，神通果然厉害。
<br/>　　然而，面对如此蚩尤，公孙轩辕依然不惧，你巫族有肉身神通，我人族难道就没有吗？人族的武道，一样有法相。
<br/>　　在蚩尤施展出神通的同时，公孙轩辕也不由分说地施展出他的天地法相。
<br/>　　只见交战之处，金光闪烁，一道耀眼无比的金光从轩辕身上亮起，紧接着，轩辕的身体也开始无限涨大，在他身后的虚空中，正缓缓出现一个高大百万丈的人体法相。
<br/>　　人族武道境界的天地法相各有不同，有风雨雷电等自然现象，也有山河草木、日月星辰等形象，像轩辕这样的人体法相就非常稀少。而武祖玄天便具有一道人体形象的武帝法相。
<br/>　　而公孙轩辕这法相，通体金光，身穿金色黄袍，头戴帝冠，手持金剑，如同一尊无上战帝。

<br/>　　“葬剑诀！葬天，葬地，葬众生！”

<br/>　　蚩尤面色冷峻，手中魔刀连挥，劈向那向他撞下来的神棺。
<br/>　　“魔神斩之斩天变！”
<br/>　　巨大刀影横空，带着强横的破灭意境，斩向虚空。
<br/>　　“轰隆！”
<br/>　　神棺破碎，同时那巨大的斩天刀影也破碎，两者变成锋利的刀气剑气相互交织纠缠，缓缓消散于虚空之中。

<br/>　　蚩尤双翅一振，飞向上方，手中魔刀以一种势不可挡的气势斩了下来，刀势中有一种毁天灭地的信念，一击之下，地都要裂开，更何况是人？
<br/>　　混乱剑光与霸道相撞，将周边的虚空都打出个巨大的窟窿，强悍的气势席卷周边一切，仿佛是要荡平天地一般。
<br/>　　与此同时，一道冷酷霸道的声音响起，回荡在天地间：
<br/>　　“绝灭天地变，方得乱中平。一剑横世间，乱世我来平！平——乱——诀！”
